## Теоретическая часть

### 1. Функция `print()`

Функция `print()` используется для вывода данных на экран.

**Основной синтаксис:**
```python
print(значение1, значение2, ..., sep=' ', end='\n')
```

**Параметры:**
- `sep` — разделитель между значениями (по умолчанию пробел)
- `end` — символ в конце вывода (по умолчанию перевод строки)

### 2. Функция `input()`

Функция `input()` используется для получения данных от пользователя.

**Синтаксис:**
```python
переменная = input(приглашение)
```

**Важно:** `input()` всегда возвращает строку (тип `str`).

### 3. Форматирование строк

**f-строки (Python 3.6+):**
```python
name = "Анна"
age = 20
print(f"Меня зовут {name}, мне {age} лет")
```

**Метод `.format()`:**
```python
print("Меня зовут {}, мне {} лет".format(name, age))
```

**Оператор `%`:**
```python
print("Меня зовут %s, мне %d лет" % (name, age))
```

### Задание 2: Работа с числовыми данными

Напишите программу для вычисления площади прямоугольника.

```python
# Задание 2: Вычисление площади прямоугольника
print("=== Вычисление площади прямоугольника ===")

# Ввод данных
length = float(input("Введите длину прямоугольника (см): "))
width = float(input("Введите ширину прямоугольника (см): "))

# Вычисление
area = length * width
perimeter = 2 * (length + width)

# Вывод результатов
print(f"\nРезультаты:")
print(f"Площадь: {area:.2f} см²")
print(f"Периметр: {perimeter:.2f} см")
```

## Дополнительные возможности

### 1. Использование escape-последовательностей

```python
# Специальные символы
print("Первая строка\nВторая строка")  # Перевод строки
print("Столбец1\tСтолбец2\tСтолбец3")  # Табуляция
print("Кавычки: \"текст в кавычках\"")  # Экранирование кавычек
print("Обратный слэш: \\")              # Обратный слэш
```

### 2. Выравнивание текста

```python
# Форматирование с выравниванием
name = "Python"
print(f"|{name:<10}|")  # Влево
print(f"|{name:>10}|")  # Вправо
print(f"|{name:^10}|")  # По центру

# Числа с фиксированной шириной
number = 42
print(f"Число: {number:05d}")  # 00042
```
