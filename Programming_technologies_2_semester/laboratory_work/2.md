## Теоретическая часть

**Разделение текста (Text Segmentation)** — это процесс разбиения текстовой строки на смысловые или структурные единицы: символы, токены, слова, предложения, абзацы.

### Основные признаки разделения

| Признак                  | Описание                               | Пример                                      |
|--------------------------|----------------------------------------|---------------------------------------------|
| По пробелам              | Разделение по пробельным символам      | `"hello world"` → `["hello", "world"]`      |
| По знакам препинания     | Разделение по `.`, `,`, `!`, `?` и др. | `"Привет, мир!"` → `["Привет", " мир", ""]` |
| По символу-разделителю   | Разделение по заданному символу        | `"a;b;c"` → `["a", "b", "c"]`               |
| По регулярному выражению | Гибкое разделение по шаблону           | любой сложный случай                        |
| По длине                 | Разделение на N-граммы или фрагменты   | `"абвгд"` → `["аб", "вг", "д"]`             |

### Задание 1. Разделение по пробелам

Разбить строку на слова, используя метод `split()`.

```python
text = "Компьютерная лингвистика изучает язык с помощью программ"

# Простое разделение по пробелу
words = text.split()
print("Слова:", words)
print("Количество слов:", len(words))
```


### Задание 2. Разделение по знакам препинания

Разбить текст на предложения по точке.

```python
text = "Язык — это система знаков. Лингвистика изучает язык. Программы помогают анализировать тексты."

sentences = text.split(".")
# Убираем пустые строки и лишние пробелы

print("Предложения:")
for i, sentence in enumerate(sentences, 1):
```

### Задание 3. Разделение по регулярному выражению

Использовать модуль `re` для разделения по нескольким разделителям одновременно.

```python
import re

text = "Слово1, слово2; слово3. Слово4! Слово5?"

# Разделение по любому знаку препинания и пробелу
tokens = re.split(r'[,;.!?\s]+', text)
tokens = [t for t in tokens if t]  # убираем пустые элементы

print("Токены:", tokens)
```

### Задание 4. Разделение на N-граммы

Разбить текст на биграммы (пары символов или слов).

```python
def get_ngrams(tokens, n):
    """Возвращает список n-грамм из списка токенов."""
    return [tuple(tokens[i:i+n]) for i in range(len(tokens) - n + 1)]

text = "я люблю изучать языки и программирование"
words = text.split()

bigrams = get_ngrams(words, 2)
trigrams = get_ngrams(words, 3)

print("Биграммы:")
for bg in bigrams:
    print(f"  {bg}")

print("\nТриграммы:")
for tg in trigrams:
    print(f"  {tg}")
```

**Ожидаемый вывод:**
```
Биграммы:
  ('я', 'люблю')
  ('люблю', 'изучать')
  ('изучать', 'языки')
  ('языки', 'и')
  ('и', 'программирование')

Триграммы:
  ('я', 'люблю', 'изучать')
  ('люблю', 'изучать', 'языки')
  ('изучать', 'языки', 'и')
  ('языки', 'и', 'программирование')
```

### Задание 5. Итоговое задание

Программа, которая:

1. Принимает на вход произвольный текст (минимум 5 предложений).
2. Разделяет текст на предложения.
3. Каждое предложение разделяет на слова.
4. Подсчитывает общее количество слов и уникальных слов.
5. Выводит результат в виде таблицы.

**Шаблон программы:**

```python
import re

def analyze_text(text):
    # 1. Разделение на предложения
    sentences = re.split(r'[.!?]+', text)
    sentences = [s.strip() for s in sentences if s.strip()]
    
    # 2. Разделение каждого предложения на слова
    all_words = []
    for sentence in sentences:
        words = re.findall(r'\b[а-яёА-ЯЁa-zA-Z]+\b', sentence)
        all_words.extend(words)
    
    # 3. Подсчёт статистики
    total_words = len(all_words)
    unique_words = len(set(w.lower() for w in all_words))
    
    # 4. Вывод результатов
    print(f"{'Показатель':<30} {'Значение':>10}")
    print("-" * 42)
    print(f"{'Количество предложений':<30} {len(sentences):>10}")
    print(f"{'Всего слов':<30} {total_words:>10}")
    print(f"{'Уникальных слов':<30} {unique_words:>10}")
    print(f"{'Средняя длина предложения':<30} {total_words/len(sentences):>10.1f}")

# --- Тестовый текст ---
text = """
Введите сюда ваш текст из пяти и более предложений.
Каждое предложение должно заканчиваться точкой, восклицательным или вопросительным знаком.
Программа автоматически разобьёт текст на части.
Вы увидите статистику по словам и предложениям.
Это полезно для задач компьютерной лингвистики!
"""

analyze_text(text)
```
