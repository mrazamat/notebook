## Теоретическая часть

### 1. Что такое помощник переводчика?

**Помощник переводчика** — это программа, которая автоматически переводит слова или фразы с одного языка на другой. В компьютерной лингвистике такие системы занимают важное место, так как они используются в:

- машинном переводе текстов;
- обработке многоязычных корпусов;
- создании учебных словарей;
- разработке чат-ботов и голосовых помощников.

---

### 2. Основные подходы к переводу

| Подход                     | Описание                                   | Пример                           |
|----------------------------|--------------------------------------------|----------------------------------|
| **Словарный перевод**      | Поиск слова в заранее составленном словаре | `dict["apple"] → "яблоко"`       |
| **Перевод через API**      | Обращение к внешнему сервису перевода      | `googletrans`, `deep-translator` |
| **Статистический перевод** | Перевод на основе обученных моделей        | Moses, Fairseq                   |
| **Нейросетевой перевод**   | Глубокое обучение (трансформеры)           | Google Translate, DeepL          |

> В этой лабораторной работе мы рассмотрим **первые два подхода** как наиболее доступные и простые для реализации.

---

### 3. Словарь в Python

Структура данных `dict` идеально подходит для хранения пар «исходное слово — перевод».

```python
# Простой словарь: английский → русский
my_dict = {
    "hello": "привет",
    "world": "мир",
    "cat":   "кошка",
    "dog":   "собака"
}

# Получение перевода
word = "cat"
translation = my_dict.get(word, "Перевод не найден")
print(translation)  # кошка
```

**Метод `.get(ключ, значение_по_умолчанию)`** позволяет безопасно получать значение из словаря: если ключ не найден, возвращается значение по умолчанию, а не ошибка.

---

### 4. Библиотека `deep-translator`

`deep-translator` — простая библиотека для машинного перевода, которая использует Google Translate, без необходимости регистрации API-ключа.

```python
from deep_translator import GoogleTranslator

# Создаём переводчик: с английского на русский
translator = GoogleTranslator(source='en', target='ru')

# Переводим текст
result = translator.translate("Hello, how are you?")
print(result)  # Привет, как дела?
```

**Поддерживаемые языки** задаются кодами по стандарту ISO 639-1:

| Код     | Язык                   |
|---------|------------------------|
| `en`    | Английский             |
| `ru`    | Русский                |
| `de`    | Немецкий               |
| `fr`    | Французский            |
| `uz`    | Узбекский              |
| `zh-CN` | Китайский (упрощённый) |

### 5. Сравнение подходов

| Критерий              | Словарный перевод     | `deep-translator`    |
|-----------------------|-----------------------|----------------------|
| **Скорость работы**   | Очень быстро          | Зависит от интернета |
| **Нужен интернет?**   | Нет                   | Да                   |

| **Гибкость**          | Только заданные слова | Любые слова и фразы  |
| **Сложность кода**    | Простой               | Простой              |

### Задание 3. Сохранение словаря в файл и загрузка из файла

```python
# Задание 3: Работа с файлом словаря

import json

# Наш словарь
dictionary = {
    "hello": "привет",
    "world": "мир",
    "python": "питон/Python",
    "language": "язык",
    "translate": "переводить"
}

# --- Сохранение в файл ---
with open("my_dictionary.json", "w", encoding="utf-8") as f:
    json.dump(dictionary, f, ensure_ascii=False, indent=2)

print("Словарь сохранён в файл my_dictionary.json")

# --- Загрузка из файла ---
with open("my_dictionary.json", "r", encoding="utf-8") as f:
    loaded_dict = json.load(f)

print("\nЗагруженный словарь:")
for eng, rus in loaded_dict.items():
    print(f"  {eng} → {rus}")

# --- Перевод с использованием загруженного словаря ---
word = "python"
print(f"\nПеревод слова '{word}': {loaded_dict.get(word, 'не найдено')}")
```

**Ожидаемый результат:**
```
Словарь сохранён в файл my_dictionary.json

Загруженный словарь:
  hello → привет
  world → мир
  python → питон/Python
  language → язык
  translate → переводить

Перевод слова 'python': питон/Python
```

> **Примечание:** Параметр `ensure_ascii=False` необходим для корректного сохранения кириллицы в файл.

### Задание 4. Перевод через `deep-translator`

```python
# Задание 4: Использование библиотеки deep-translator
# Установка: pip install deep-translator

from deep_translator import GoogleTranslator

def translate_text(text, source_lang='en', target_lang='ru'):
    """Переводит текст с помощью Google Translate."""
    try:
        translator = GoogleTranslator(source=source_lang, target=target_lang)
        result = translator.translate(text)
        return result
    except Exception as e:
        return f"Ошибка перевода: {e}"

# Тестовые фразы
phrases = [
    "Hello, world!",
    "I love programming.",
    "Computational linguistics is very interesting."
]

print("=== Перевод через deep-translator ===\n")
for phrase in phrases:
    translation = translate_text(phrase)
    print(f"  Оригинал:  {phrase}")
    print(f"  Перевод:   {translation}")
    print()
```

**Ожидаемый результат:**
```
=== Перевод через deep-translator ===

  Оригинал:  Hello, world!
  Перевод:   Привет, мир!

  Оригинал:  I love programming.
  Перевод:   Я люблю программирование.

  Оригинал:  Computational linguistics is very interesting.
  Перевод:   Вычислительная лингвистика очень интересна.
```
