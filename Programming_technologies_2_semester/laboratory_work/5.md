## Теоретическая часть

### Что такое словарь в Python?

**Словарь (dict)** — это встроенная структура данных в Python, которая хранит пары **ключ → значение**. В задачах перевода ключом служит слово на исходном языке, а значением — его перевод.

```python
словарь = {
    "ключ1": "значение1",
    "ключ2": "значение2"
}
```

В контексте переводчика:

```python
translator = {
    "apple": "яблоко",
    "book":  "книга",
    "cat":   "кошка"
}
```

### Основные операции со словарём

| Операция          | Синтаксис                   | Описание                         |
|-------------------|-----------------------------|----------------------------------|
| Добавить запись   | `d[ключ] = значение`        | Добавляет новую пару             |
| Получить значение | `d[ключ]`                   | Вернёт значение или ошибку       |
| Безопасный поиск  | `d.get(ключ, "не найдено")` | Вернёт значение или заглушку     |
| Удалить запись    | `del d[ключ]`               | Удаляет пару из словаря          |
| Проверить наличие | `ключ in d`                 | Возвращает `True`/`False`        |
| Все ключи         | `d.keys()`                  | Список всех ключей               |
| Все значения      | `d.values()`                | Список всех значений             |
| Все пары          | `d.items()`                 | Список кортежей (ключ, значение) |

### Сравнение методов поиска

| Метод                | Поведение при отсутствии ключа | Когда использовать                |
|----------------------|--------------------------------|-----------------------------------|
| `d["ключ"]`          | Вызывает `KeyError`            | Когда ключ гарантированно есть    |
| `d.get("ключ")`      | Возвращает `None`              | Когда ключ может отсутствовать    |
| `d.get("ключ", "?")` | Возвращает `"?"`               | Когда нужен текст-заглушка        |
| `"ключ" in d`        | Возвращает `False`             | Для явной проверки перед доступом |

## Демонстрационные примеры

### Пример 1 — Простой переводчик отдельных слов

```python
# Мини-словарь английских слов
dictionary = {
    "hello":   "привет",
    "world":   "мир",
    "apple":   "яблоко",
    "book":    "книга",
    "cat":     "кошка",
    "dog":     "собака",
    "house":   "дом",
    "student": "студент",
    "teacher": "учитель",
    "language": "язык"
}

def translate_word(word):
    """Переводит одно слово с английского на русский."""
    word = word.lower().strip()
    return dictionary.get(word, f"['{word}' — перевод не найден]")

# Тест
print(translate_word("apple"))    # яблоко
print(translate_word("book"))     # книга
print(translate_word("sky"))      # ['sky' — перевод не найден]
```

**Вывод:**
```
яблоко
книга
['sky' — перевод не найден]
```

### Пример 2 — Перевод предложения по словам

```python
dictionary = {
    "i":       "я",
    "love":    "люблю",
    "python":  "питон",
    "and":     "и",
    "study":   "учу",
    "every":   "каждый",
    "day":     "день"
}

def translate_sentence(sentence):
    """Переводит предложение слово за словом."""
    words = sentence.lower().split()
    translated = []
    for word in words:
        translated.append(dictionary.get(word, f"[{word}]"))
    return " ".join(translated)

# Тест
print(translate_sentence("I love Python"))
print(translate_sentence("I study every day"))
```

**Вывод:**
```
я люблю питон
я учу каждый день
```

### Пример 3 — Интерактивный мини-переводчик

```python
dictionary = {
    "hello":    "привет",
    "bye":      "пока",
    "thank":    "спасибо",
    "yes":      "да",
    "no":       "нет",
    "name":     "имя",
    "water":    "вода",
    "food":     "еда"
}

def interactive_translator():
    """Переводчик с вводом слов пользователем."""
    print("=== Мини-переводчик (EN → RU) ===")
    print("Введите 'стоп' для выхода\n")

    while True:
        word = input("Введите слово: ").strip()
        if word.lower() == "стоп":
            print("До свидания!")
            break
        result = dictionary.get(word.lower(), "Слово не найдено в словаре")
        print(f"  → {result}\n")

interactive_translator()
```

**Пример работы:**
```
=== Мини-переводчик (EN → RU) ===
Введите 'стоп' для выхода

Введите слово: hello
  → привет

Введите слово: sky
  → Слово не найдено в словаре

Введите слово: стоп
До свидания!
```

### Пример 4 — Расширяемый словарь (добавление новых слов)

```python
dictionary = {
    "sun": "солнце",
    "moon": "луна"
}

def add_word(eng, rus):
    """Добавляет новое слово в словарь."""
    dictionary[eng.lower()] = rus
    print(f"Добавлено: {eng} → {rus}")

def show_dictionary():
    """Выводит весь словарь."""
    print("\n--- Текущий словарь ---")
    for eng, rus in sorted(dictionary.items()):
        print(f"  {eng:15} → {rus}")

# Пример использования
add_word("star", "звезда")
add_word("cloud", "облако")
show_dictionary()
```

**Вывод:**
```
Добавлено: star → звезда
Добавлено: cloud → облако

--- Текущий словарь ---
  cloud           → облако
  moon            → луна
  star            → звезда
  sun             → солнце
```
