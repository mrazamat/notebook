## Теоретическая часть

### Что такое «выделение цифр из текста»?

В задачах компьютерной лингвистики часто требуется извлечь из текста числовую информацию: даты, цены, номера телефонов, возраст, координаты и т.д. Такая задача называется **выделением числовых сущностей** (числовая токенизация).

### Способы выделения цифр в Python

| Способ               | Метод                  | Когда использовать |
|----------------------|------------------------|--------------------|
| Проверка символа     | `char.isdigit()`       | Отдельные цифры    |
| Разбиение строки     | `split()` + фильтрация | Простые случаи     |
| Регулярные выражения | `re.findall()`         | Сложные шаблоны    |
| Comprehension        | Списковое включение    | Быстрая фильтрация |

---

### Метод 1: `isdigit()` — проверка отдельных символов

Метод `isdigit()` возвращает `True`, если символ является цифрой.

```python
text = "Мне 23 года, я живу на 5 этаже."

digits = [char for char in text if char.isdigit()]
print(digits)
# ['2', '3', '5']
```

> `isdigit()` возвращает **отдельные цифры**, а не целые числа.

---

### Метод 2: `split()` + `isdigit()` — выделение числовых токенов

```python
text = "В 2024 году вышло 15 новых моделей ИИ."

tokens = text.split()
numbers = [token for token in tokens if token.isdigit()]
print(numbers)
# ['2024', '15']
```

> Не сработает, если число стоит рядом с пунктуацией: `"15,"` → не число.

---

### Метод 3: `re.findall()` — регулярные выражения

Модуль `re` позволяет искать **любые числа**, в том числе дробные и с разными разделителями.

```python
import re

text = "Температура: -3.5°C, давление 760 мм рт.ст., влажность 87%."

# Целые числа
integers = re.findall(r'\d+', text)
print("Целые:", integers)
# ['3', '5', '760', '87']

# Целые и дробные числа
floats = re.findall(r'-?\d+\.?\d*', text)
print("Числа:", floats)
# ['-3.5', '760', '87']
```

---

### Таблица регулярных выражений для чисел

| Шаблон        | Описание                          | Пример совпадения |
|---------------|-----------------------------------|-------------------|
| `\d`          | Одна цифра                        | `5`               |
| `\d+`         | Одно или более цифр (целое число) | `123`             |
| `\d{4}`       | Ровно 4 цифры                     | `2024`            |
| `-?\d+`       | Целое со знаком минус             | `-42`, `99`       |
| `\d+\.\d+`    | Дробное число                     | `3.14`            |
| `-?\d+\.?\d*` | Целое или дробное со знаком       | `-3.5`, `760`     |
| `\b\d+\b`     | Целое число (граница слова)       | `15` (не `15,`)   |

### Задание 2: Извлечение чисел с помощью `re.findall()`

```python
import re

text = "Рейс 234 вылетает в 08:45, прибывает в 13:10, стоимость 4500.99 руб."

all_numbers = re.findall(r'\d+', text)
print("Все числа:", all_numbers)

# Найти время в формате ЧЧ:ММ
times = re.findall(r'\d{2}:\d{2}', text)
print("Время:", times)

# Найти стоимость (целое или дробное)
price = re.findall(r'\d+\.\d+', text)
print("Цена:", price)
```

text = "Оценки студентов: 5, 4, 3, 2, 5, 4, 3, 5, 1, 4."
