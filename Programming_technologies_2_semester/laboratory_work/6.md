## Теоретическая часть

### Что такое файловый ввод-вывод?

В лингвистических задачах часто приходится работать с большими текстами: корпусами, словарями, результатами анализа. Хранить всё это в переменных невозможно — нужно уметь **сохранять данные в файл** и **читать их обратно**.

Python предоставляет встроенную функцию `open()` для работы с файлами.

---

### Функция `open()`

```python
open(filename, mode, encoding)
```

| Параметр   | Описание                              |
|------------|---------------------------------------|
| `filename` | Имя файла (путь к файлу)              |
| `mode`     | Режим открытия файла (см. ниже)       |
| `encoding` | Кодировка (рекомендуется `"utf-8"`)   |

---

### Режимы открытия файла

| Режим  | Название            | Описание                                                 |
|--------|---------------------|----------------------------------------------------------|
| `"r"`  | Read (чтение)       | Только чтение. Файл должен существовать                  |
| `"w"`  | Write (запись)      | Запись. Создаёт новый файл или **перезаписывает** старый |
| `"a"`  | Append (добавление) | Добавляет текст в конец файла                            |
| `"r+"` | Чтение + запись     | Открывает существующий файл для чтения и записи          |

---

### Способы чтения файла

| Метод          | Описание                                          |
|----------------|---------------------------------------------------|
| `read()`       | Читает весь файл как одну строку                  |
| `readline()`   | Читает одну строку                                |
| `readlines()`  | Читает все строки и возвращает список строк       |

---

### Конструкция `with`

Рекомендуется открывать файлы через `with` — это автоматически закрывает файл после работы:

```python
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()
# Файл закрыт автоматически
```

### Задание 1. Запись текста в файл

Запустите и изучите следующий код:

```python

text = "Компьютерная лингвистика — наука на стыке языкознания и информатики.\n"
text += "Она изучает методы автоматической обработки естественного языка.\n"
```

### Задание 3. Чтение файла построчно

```python
# Чтение строк в список
with open("linguistics.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()

print(f"Количество строк: {len(lines)}")

for i, line in enumerate(lines, start=1):
    print(f"Строка {i}: {line.strip()}")
```

**Обратите внимание:** метод `.strip()` убирает символ переноса строки `\n` в конце.

### Задание 5. Сравнение режимов `"w"` и `"a"`

```python
# Режим "w" — перезаписывает файл
with open("test_w.txt", "w", encoding="utf-8") as f:
    f.write("Первый запуск\n")

with open("test_w.txt", "w", encoding="utf-8") as f:
    f.write("Второй запуск\n")  # Первая строка будет удалена!

with open("test_w.txt", "r", encoding="utf-8") as f:
    print("Режим 'w':", f.read())

# Режим "a" — добавляет к существующему содержимому
with open("test_a.txt", "a", encoding="utf-8") as f:
    f.write("Первый запуск\n")

with open("test_a.txt", "a", encoding="utf-8") as f:
    f.write("Второй запуск\n")  # Обе строки сохранятся!

with open("test_a.txt", "r", encoding="utf-8") as f:
    print("Режим 'a':", f.read())
```
