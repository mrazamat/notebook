## Теоретическая часть

### Что такое токенизация?

**Токенизация** — это процесс разбиения текста на отдельные значимые единицы, называемые **токенами**. Токеном может быть слово, знак препинания, число или любой другой элемент текста.

Например, предложение:

```
"Кот сидит на окне."
```

после токенизации превращается в список:

```
["Кот", "сидит", "на", "окне", "."]
```

Токенизация является **первым и обязательным шагом** в большинстве задач обработки естественного языка (NLP):
- машинный перевод
- анализ тональности текста
- поиск информации
- автоматическое реферирование

---

### Методы токенизации в Python

#### Метод 1: `str.split()`

Самый простой способ — разбить строку по пробелам с помощью встроенного метода `split()`.

```python
text = "Кот сидит на окне"
tokens = text.split()
print(tokens)
# ['Кот', 'сидит', 'на', 'окне']
```

Можно указать **разделитель** явно:

```python
text = "яблоко,груша,слива"
tokens = text.split(",")
print(tokens)
# ['яблоко', 'груша', 'слива']
```

**Ограничение:** `split()` не удаляет знаки препинания — `"окне."` останется одним токеном вместе с точкой.

---

#### Метод 2: Регулярные выражения (`re`)

Модуль `re` из стандартной библиотеки Python позволяет гибко описывать шаблоны для выделения токенов.

```python
import re

text = "Кот сидит на окне. Птица поёт в саду!"
tokens = re.findall(r'\b[а-яёА-ЯЁ]+\b', text)
print(tokens)
# ['Кот', 'сидит', 'на', 'окне', 'Птица', 'поёт', 'в', 'саду']
```

**Полезные шаблоны:**

| Шаблон        | Описание                        |
|---------------|---------------------------------|
| `\w+`         | Любое слово (буквы, цифры, `_`) |
| `[а-яёА-ЯЁ]+` | Только русские буквы            |
| `[a-zA-Z]+`   | Только латинские буквы          |
| `\d+`         | Только цифры                    |
| `\b`          | Граница слова                   |

---

#### Метод 3: Библиотека `nltk`

`nltk` (Natural Language Toolkit) — популярная библиотека для NLP, содержащая готовые токенизаторы.

```python
import nltk
nltk.download('punkt')

from nltk.tokenize import word_tokenize

text = "Кот сидит на окне. Птица поёт в саду!"
tokens = word_tokenize(text, language='russian')
print(tokens)
# ['Кот', 'сидит', 'на', 'окне', '.', 'Птица', 'поёт', 'в', 'саду', '!']
```

`word_tokenize` корректно обрабатывает знаки препинания, выделяя их как отдельные токены.

### Задание 2: Токенизация с помощью регулярных выражений

Напишите программу, которая:
1. Использует предложенный текст
2. Извлекает **только русские слова** с помощью `re.findall()`
3. Выводит токены без знаков препинания

```python
import re

text = "Добрый день! Как вы себя чувствуете? Надеюсь, хорошо."

# Извлекаем только русские слова
tokens = re.findall(r'[а-яёА-ЯЁ]+', text)

print("Токены без знаков препинания:")
print(tokens)
print("Количество слов:", len(tokens))
```

**Ожидаемый результат:**
```
Токены без знаков препинания:
['Добрый', 'день', 'Как', 'вы', 'себя', 'чувствуете', 'Надеюсь', 'хорошо']
Количество слов: 8
```
