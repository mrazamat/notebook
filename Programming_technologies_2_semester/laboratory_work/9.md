## Теоретическая часть

### Что такое статистика текста?

**Статистика текста** — это набор числовых характеристик, описывающих структуру и содержание текстового документа. В компьютерной лингвистике статистический анализ используется для:

- оценки сложности и читаемости текста;
- сравнения авторских стилей;
- предварительной обработки данных в NLP-задачах;
- анализа корпусов текстов.

---

### Основные метрики

| Метрика                            | Описание                     | Способ вычисления в Python                |
|------------------------------------|------------------------------|-------------------------------------------|
| Количество символов                | Все символы, включая пробелы | `len(text)`                               |
| Количество символов (без пробелов) | Только непробельные символы  | `len(text.replace(" ", ""))`              |
| Количество слов                    | Разбиение по пробелам        | `len(text.split())`                       |
| Количество предложений             | Разбиение по знакам `.!?`    | `re.split(r'[.!?]+', text)`               |
| Количество уникальных слов         | Уникальные токены            | `len(set(words))`                         |
| Средняя длина слова                | Среднее число букв в слове   | `sum(len(w) for w in words) / len(words)` |
| Самое частое слово                 | Слово с наибольшей частотой  | `Counter(words).most_common(1)`           |

---

### Модуль `collections.Counter`

`Counter` — это специальный класс из стандартной библиотеки Python, который автоматически считает количество вхождений элементов в список.

```python
from collections import Counter

words = ["кот", "пёс", "кот", "птица", "кот", "пёс"]
counter = Counter(words)

print(counter)
# Counter({'кот': 3, 'пёс': 2, 'птица': 1})

print(counter.most_common(2))
# [('кот', 3), ('пёс', 2)]
```

### Нормализация текста

Перед подсчётом статистики текст обычно **нормализуют**:

1. Приводят к нижнему регистру — `text.lower()`
2. Удаляют знаки пунктуации — с помощью `re.sub()`
3. Разбивают на слова — `text.split()`

Это позволяет корректно считать слова: «Кот», «кот» и «КОТ» будут считаться одним словом.

```python
import re

text = "Привет, Мир! Это — Python."
text_clean = re.sub(r'[^\w\s]', '', text.lower())
print(text_clean)
# привет мир это  python
```

## Практическая часть

### Пример 1 — Базовая статистика

```python
text = """
Компьютерная лингвистика изучает язык с помощью вычислительных методов.
Она используется в машинном переводе, поиске информации и анализе текста.
Программирование помогает автоматизировать анализ больших объёмов текста.
"""

# Подсчёт символов
print("Символов (всего):", len(text))
print("Символов (без пробелов):", len(text.replace(" ", "").replace("\n", "")))

# Подсчёт слов
words = text.split()
print("Слов:", len(words))

# Подсчёт предложений
import re
sentences = [s.strip() for s in re.split(r'[.!?]+', text) if s.strip()]
print("Предложений:", len(sentences))
```

**Вывод:**
```
Символов (всего): 222
Символов (без пробелов): 173
Слов: 29
Предложений: 3
```

### Пример 2 — Частота слов

```python
import re
from collections import Counter

text = """
Питон — это язык программирования. Язык питон прост и удобен.
Программирование на питоне популярно среди лингвистов.
"""

# Нормализация
words = re.sub(r'[^\w\s]', '', text.lower()).split()

# Подсчёт частоты
freq = Counter(words)

print("Частота слов:")
for word, count in freq.most_common():
    print(f"  {word}: {count}")

print("\nСамое частое слово:", freq.most_common(1)[0])
```

**Вывод:**
```
Частота слов:
  питон: 2
  язык: 2
  программирование: 2
  это: 1
  ...

Самое частое слово: ('питон', 2)
```

### Пример 3 — Средняя длина слова и уникальность

```python
import re
from collections import Counter

text = "Лингвистика изучает языки. Языки бывают разные и сложные."

words = re.sub(r'[^\w\s]', '', text.lower()).split()

# Средняя длина слова
avg_len = sum(len(w) for w in words) / len(words)
print(f"Средняя длина слова: {avg_len:.2f} символов")

# Уникальные слова
unique = set(words)
print(f"Уникальных слов: {len(unique)} из {len(words)}")
print(f"Коэффициент уникальности: {len(unique)/len(words):.2f}")
```

**Вывод:**
```
Средняя длина слова: 6.78 символов
Уникальных слов: 8 из 9
Коэффициент уникальности: 0.89
```

### Пример 4 — Полная функция анализа текста

```python
import re
from collections import Counter

def analyze_text(text):
    """Возвращает полную статистику текста."""
    
    # Нормализация
    words_raw = text.split()
    words = re.sub(r'[^\w\s]', '', text.lower()).split()
    sentences = [s.strip() for s in re.split(r'[.!?]+', text) if s.strip()]
    
    freq = Counter(words)
    
    stats = {
        "Символов (всего)": len(text),
        "Символов (без пробелов)": len(re.sub(r'\s', '', text)),
        "Слов": len(words_raw),
        "Предложений": len(sentences),
        "Уникальных слов": len(set(words)),
        "Средняя длина слова": round(sum(len(w) for w in words) / len(words), 2),
        "Самое частое слово": freq.most_common(1)[0] if words else None,
    }
    
    return stats

# Тестирование
my_text = """
Лингвистика — наука о языке. Она изучает структуру языка и его использование.
Компьютерная лингвистика применяет методы программирования для анализа языка.
Язык — главный инструмент общения людей.
"""

result = analyze_text(my_text)
print("=== Статистика текста ===")
for key, value in result.items():
    print(f"{key}: {value}")
```

**Вывод:**
```
=== Статистика текста ===
Символов (всего): 210
Символов (без пробелов): 173
Слов: 30
Предложений: 4
Уникальных слов: 24
Средняя длина слова: 7.13
Самое частое слово: ('языка', 3)
```

## Сравнительная таблица методов подсчёта слов

| Метод                      | Пример                | Учитывает пунктуацию | Регистр   |
|----------------------------|-----------------------|----------------------|-----------|
| `text.split()`             | `"Кот."` → `["Кот."]` | Нет (слипается)      | Сохраняет |
| `re.findall(r'\w+', text)` | `"Кот."` → `["Кот"]`  | Да (отделяет)        | Сохраняет |
| `re.sub()` + `split()`     | `"кот."` → `["кот"]`  | Да                   | Нижний    |

> **Рекомендуется** использовать `re.sub(r'[^\w\s]', '', text.lower()).split()` для корректной нормализации.
