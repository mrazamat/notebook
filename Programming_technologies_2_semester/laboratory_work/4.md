## Теоретическая часть

### Что такое предложение?

**Предложение** — это минимальная единица речи, выражающая законченную мысль. В письменном тексте предложение завершается знаком конца предложения:

| Знак  | Описание             | Пример                   |
|-------|----------------------|--------------------------|
| `.`   | Точка                | *Солнце светит ярко.*    |
| `!`   | Восклицательный знак | *Какой прекрасный день!* |
| `?`   | Вопросительный знак  | *Как тебя зовут?*        |
| `...` | Многоточие           | *Он задумался...*        |

### Сложности при подсчёте предложений

Автоматический подсчёт предложений — нетривиальная задача, потому что:

- Точка используется не только в конце предложения, но и в сокращениях (*д-р*, *т.е.*, *и др.*), в числах (*3.14*), в инициалах (*И. И. Иванов*).
- После многоточия может не начинаться новое предложение.
- Текст может содержать знаки препинания внутри кавычек или скобок.

### Подходы к подсчёту предложений

**Подход 1 — Метод `split()` (простой)**  
Разбиваем текст по символам-терминаторам: `.`, `!`, `?`. Быстро и просто, но не учитывает сокращения и аббревиатуры.

**Подход 2 — Регулярные выражения (`re`)**  
Используем шаблоны, чтобы точнее определить конец предложения. Позволяет задать правила: предложение заканчивается знаком препинания, за которым следует пробел и заглавная буква.

**Подход 3 — Библиотека `nltk`**  
`nltk.sent_tokenize()` — готовый инструмент, специально обученный на задаче разбиения текста на предложения. Учитывает аббревиатуры и сложные случаи.

## Практическая часть

### Задание 1. Подсчёт предложений методом `split()`

```python
text = "Привет! Как дела? Я учусь программировать. Это интересно!"

# Разбиваем по знакам конца предложения
import re

sentences = re.split(r'[.!?]+', text)

# Убираем пустые строки, которые могут появиться после разбиения
sentences = [s.strip() for s in sentences if s.strip()]

print(f"Количество предложений: {len(sentences)}")
print("Предложения:")
for i, s in enumerate(sentences, 1):
    print(f"  {i}. {s}")
```

**Ожидаемый вывод:**
```
Количество предложений: 4
Предложения:
  1. Привет
  2. Как дела
  3. Я учусь программировать
  4. Это интересно
```

### Задание 2. Подсчёт предложений с помощью регулярных выражений

```python
import re

text = """Д-р Иванов пришёл на работу в 9 утра. Он поздоровался с коллегами.
Погода была хорошей! Все были рады началу дня.
Ты уже сделал домашнее задание? Нет, ещё не успел..."""

# Шаблон: конец предложения — знак препинания, за которым идёт
# пробел/перевод строки и заглавная буква (или конец строки)
pattern = r'[.!?…]+(?=\s+[А-ЯЁA-Z]|\s*$)'

# Находим все концы предложений
endings = re.findall(pattern, text)
sentence_count = len(endings)

print(f"Количество предложений: {sentence_count}")
```

**Ожидаемый вывод:**
```
Количество предложений: 6
```

### Задание 3. Подсчёт предложений с помощью `nltk`

```python
import nltk

# Скачиваем необходимые данные (выполняется один раз)
nltk.download('punkt')
nltk.download('punkt_tab')

text = """Д-р Петров написал статью. Статья была опубликована в журнале.
Редакция была довольна! Читатели оценили работу высоко.
Будет ли продолжение? Автор пообещал написать ещё."""

# Разбиваем текст на предложения
sentences = nltk.sent_tokenize(text, language='russian')

print(f"Количество предложений: {len(sentences)}")
print("\nПредложения:")
for i, s in enumerate(sentences, 1):
    print(f"  {i}. {s}")
```

**Ожидаемый вывод:**
```
Количество предложений: 6

Предложения:
  1. Д-р Петров написал статью.
  2. Статья была опубликована в журнале.
  3. Редакция была довольна!
  4. Читатели оценили работу высоко.
  5. Будет ли продолжение?
  6. Автор пообещал написать ещё.
```

### Задание 4. Сравнение методов на одном тексте

```python
import re
import nltk

nltk.download('punkt', quiet=True)
nltk.download('punkt_tab', quiet=True)

text = """Проф. Смирнов провёл лекцию по лингвистике. Студенты слушали внимательно!
Один из них спросил: "Что такое токенизация?" Преподаватель подробно объяснил тему.
Занятие закончилось в 14:30. Все разошлись по домам..."""

# Метод 1: простое разбиение
method1 = [s.strip() for s in re.split(r'[.!?]+', text) if s.strip()]
count1 = len(method1)

# Метод 2: регулярные выражения
endings = re.findall(r'[.!?…]+(?=\s+[А-ЯЁA-Z]|\s*$)', text)
count2 = len(endings)

# Метод 3: nltk
sentences_nltk = nltk.sent_tokenize(text, language='russian')
count3 = len(sentences_nltk)

print(f"{'Метод':<35} {'Результат':>10}")
print("-" * 47)
print(f"{'Метод 1 (split по знакам)':<35} {count1:>10}")
print(f"{'Метод 2 (регулярные выражения)':<35} {count2:>10}")
print(f"{'Метод 3 (nltk)':<35} {count3:>10}")
```
