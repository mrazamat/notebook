**Тема:** Работа с текстовыми файлами в Python

## Теоретическая часть

### Зачем работать с файлами?

В компьютерной лингвистике большая часть данных хранится в текстовых файлах: корпуса текстов, словари, результаты анализа. Python предоставляет удобные встроенные инструменты для чтения и записи таких файлов.

### Открытие файла: функция `open()`

Основная функция для работы с файлами — `open()`. Её синтаксис:

```python
file = open("filename.txt", mode, encoding="utf-8")
```

Параметр `encoding="utf-8"` особенно важен при работе с текстами на русском, казахском и других языках с кириллицей.

### Режимы открытия файла

| Режим  | Описание           | Файл существует? | Файл не существует? |
|--------|--------------------|------------------|---------------------|
| `"r"`  | Чтение             | Открывает        | Ошибка              |
| `"w"`  | Запись (с нуля)    | Перезаписывает   | Создаёт новый       |
| `"a"`  | Добавление в конец | Дописывает       | Создаёт новый       |
| `"r+"` | Чтение и запись    | Открывает        | Ошибка              |

### Два способа работы с файлами

**Способ 1 — Явное закрытие (старый стиль):**

```python
file = open("text.txt", "r", encoding="utf-8")
content = file.read()
file.close()  # Важно не забыть закрыть файл!
```

**Способ 2 — Контекстный менеджер `with` (рекомендуется):**

```python
with open("text.txt", "r", encoding="utf-8") as file:
    content = file.read()
# Файл закрывается автоматически после блока with
```

Конструкция `with` надёжнее: файл закрывается автоматически даже в случае ошибки.

### Методы чтения

| Метод             | Что возвращает                 | Когда использовать        |
|-------------------|--------------------------------|---------------------------|
| `read()`          | Всё содержимое как одну строку | Небольшие файлы           |
| `readlines()`     | Список строк                   | Пообработка каждой строки |
| `readline()`      | Одну строку                    | Построчное чтение         |
| Итерация по файлу | По одной строке                | Большие файлы             |

### Методы записи

| Метод              | Описание                |
|--------------------|-------------------------|
| `write(string)`    | Записывает одну строку  |
| `writelines(list)` | Записывает список строк |

> **Важно:** `write()` не добавляет перенос строки автоматически. Нужно явно указывать `"\n"`.

## Практические примеры

### Пример 1. Запись текста в файл

```python
text = "Лингвистика — наука о языке.\nОна изучает структуру и функции языков мира."

with open("linguistics.txt", "w", encoding="utf-8") as file:
    file.write(text)

print("Файл успешно создан.")
```

### Пример 2. Чтение всего файла

```python
with open("linguistics.txt", "r", encoding="utf-8") as file:
    content = file.read()

print(content)
```

**Вывод:**
```
Лингвистика — наука о языке.
Она изучает структуру и функции языков мира.
```

### Пример 3. Чтение по строкам с помощью `readlines()`

```python
with open("linguistics.txt", "r", encoding="utf-8") as file:
    lines = file.readlines()

for i, line in enumerate(lines, start=1):
    print(f"Строка {i}: {line.strip()}")
```

**Вывод:**
```
Строка 1: Лингвистика — наука о языке.
Строка 2: Она изучает структуру и функции языков мира.
```

> Метод `.strip()` убирает лишние пробелы и символ переноса строки `\n` с концов строки.

### Пример 4. Подсчёт слов в файле

```python
with open("linguistics.txt", "r", encoding="utf-8") as file:
    content = file.read()

words = content.split()
print(f"Количество слов в файле: {len(words)}")
```

**Вывод:**
```
Количество слов в файле: 12
```

### Пример 5. Добавление текста в существующий файл

```python
addition = "\nКомпьютерная лингвистика объединяет язык и технологии."

with open("linguistics.txt", "a", encoding="utf-8") as file:
    file.write(addition)

print("Текст добавлен.")
```

### Пример 6. Запись списка слов в файл

```python
words = ["морфология", "синтаксис", "семантика", "фонетика", "прагматика"]

with open("linguistics_sections.txt", "w", encoding="utf-8") as file:
    for word in words:
        file.write(word + "\n")

print("Список записан в файл.")
```

### Пример 7. Чтение файла и подсчёт символов

```python
with open("linguistics.txt", "r", encoding="utf-8") as file:
    text = file.read()

chars_no_spaces = len(text.replace(" ", "").replace("\n", ""))
print(f"Символов (без пробелов): {chars_no_spaces}")
print(f"Символов (с пробелами): {len(text)}")
```

### Пример 8. Поиск слова в файле

```python
query = "лингвистика"

with open("linguistics.txt", "r", encoding="utf-8") as file:
    lines = file.readlines()

for number, line in enumerate(lines, start=1):
    if query.lower() in line.lower():
        print(f"Найдено в строке {number}: {line.strip()}")
```

### Пример 9. Копирование текстового файла

```python
with open("linguistics.txt", "r", encoding="utf-8") as source:
    content = source.read()

with open("linguistics_copy.txt", "w", encoding="utf-8") as copy:
    copy.write(content)

print("Файл скопирован.")
```

### Пример 10. Запись результата анализа в файл

```python
text = "Язык — это система знаков. Знаки передают смысл. Смысл важен для общения."

words = text.split()
sentences = text.split(".")
unique_words = set(word.strip(".,!?").lower() for word in words)

result = f"""Анализ текста
==============
Слов всего: {len(words)}
Предложений: {len([s for s in sentences if s.strip()])}
Уникальных слов: {len(unique_words)}
"""

with open("analysis_result.txt", "w", encoding="utf-8") as file:
    file.write(result)

print("Результат сохранён в файл.")
```
