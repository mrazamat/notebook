## Теоретическая часть

### Что такое очистка текста?

В компьютерной лингвистике **очистка текста** (text cleaning / text preprocessing) — это этап подготовки данных перед дальнейшим анализом: лемматизацией, классификацией, машинным переводом и т.д. Сырые тексты, взятые из интернета, документов или пользовательского ввода, как правило, содержат различные виды «мусора».

### Типичные проблемы в «грязных» текстах

| Проблема                  | Пример (до)                   | Результат (после) |
|---------------------------|-------------------------------|-------------------|
| Лишние пробелы            | `"слово  другое"`             | `"слово другое"`  |
| Знаки пунктуации (лишние) | `"привет!!!"`                 | `"привет"`        |
| Цифры внутри текста       | `"текст123"`                  | `"текст"`         |
| Специальные символы       | `"@#$слово"`                  | `"слово"`         |
| Смешанный регистр         | `"Привет МИР"`                | `"привет мир"`    |
| HTML-теги                 | `"<b>жирный</b>"`             | `"жирный"`        |
| Ссылки (URL)              | `"https://example.com текст"` | `"текст"`         |
| Пустые строки             | `"\n\n\nТекст"`               | `"Текст"`         |

### Основные инструменты Python для очистки текста

**Строковые методы:**

- `str.strip()` — удаляет пробельные символы по краям строки.
- `str.lower()` / `str.upper()` — приведение к нижнему/верхнему регистру.
- `str.replace(old, new)` — замена подстроки.
- `str.split()` / `str.join()` — разбивка и склейка строк.

**Модуль `re` (регулярные выражения):**

- `re.sub(pattern, repl, string)` — замена по шаблону.
- `re.findall(pattern, string)` — поиск всех совпадений.
- `re.compile(pattern)` — компиляция шаблона для многократного использования.

### Таблица: строковые методы vs регулярные выражения

| Задача                      | Строковый метод          | Регулярное выражение                |
|-----------------------------|--------------------------|-------------------------------------|
| Удалить крайние пробелы     | `text.strip()`           | `re.sub(r'^\s+                      |\s+$', '', text)` |
| Привести к нижнему регистру | `text.lower()`           | —                                   |
| Убрать все цифры            | —                        | `re.sub(r'\d+', '', text)`          |
| Убрать HTML-теги            | —                        | `re.sub(r'<[^>]+>', '', text)`      |
| Убрать URL                  | —                        | `re.sub(r'https?://\S+', '', text)` |
| Убрать повторы пробелов     | `' '.join(text.split())` | `re.sub(r'\s+', ' ', text)`         |

### Упражнение 3. Сборка универсального очистителя

Объедините отдельные функции в единый конвейер (pipeline). Порядок шагов важен.

```python
import re

def clean_text(text, steps=None):
    """
    Применяет выбранные шаги очистки к тексту.
    steps: список строк с названиями шагов.
           Если None — применяются все шаги.
    """
    available_steps = {
        'lowercase':       lambda t: t.lower(),
        'remove_urls':     lambda t: re.sub(r'https?://\S+|www\.\S+', '', t),
        'remove_html':     lambda t: re.sub(r'<[^>]+>', '', t),
        'remove_digits':   lambda t: re.sub(r'\d+', '', t),
        'remove_special':  lambda t: re.sub(r'[^а-яёa-z\s]', '', t, flags=re.IGNORECASE),
        'remove_spaces':   lambda t: re.sub(r'\s+', ' ', t).strip(),
    }

    if steps is None:
        steps = list(available_steps.keys())

    for step in steps:
        if step in available_steps:
            text = available_steps[step](text)
    return text

# Тест
dirty_text = "  <h1>Новости</h1> Сегодня, 10 мая 2024г., вышла новая версия!!! Подробности: https://news.example.com  "

cleaned = clean_text(dirty_text)
print(cleaned)
# Ожидаемый результат: "новости сегодня мая г вышла новая версия подробности"
```

### Упражнение 5. Чтение текста из файла и сохранение результата

```python
import re

def clean_text(text):
    text = text.lower()
    text = re.sub(r'<[^>]+>', '', text)
    text = re.sub(r'https?://\S+', '', text)
    text = re.sub(r'[^\w\s]', '', text)
    text = re.sub(r'\d+', '', text)
    text = re.sub(r'\s+', ' ', text).strip()
    return text

def process_file(input_path, output_path):
    with open(input_path, 'r', encoding='utf-8') as f:
        raw_text = f.read()

    cleaned = clean_text(raw_text)

    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(cleaned)

    print(f"Файл обработан. Исходный размер: {len(raw_text)} символов.")
    print(f"Очищенный размер: {len(cleaned)} символов.")
    print(f"Сохранено в: {output_path}")

# Пример использования:
# process_file("dirty_text.txt", "clean_text.txt")
```
