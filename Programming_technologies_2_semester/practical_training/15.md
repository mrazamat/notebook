## Теоретическая часть

### Что такое частотный анализ слов?

Частотный анализ слов — это базовая задача компьютерной лингвистики, позволяющая определить, насколько часто каждое слово встречается в тексте. Он используется в поисковых системах, системах машинного перевода, анализе тональности текста и создании языковых моделей.

### Основные инструменты Python для подсчёта слов

| Инструмент               | Описание                          | Пример применения   |
|--------------------------|-----------------------------------|---------------------|
| `str.split()`            | Разбивает строку на список слов   | Простая токенизация |
| `str.lower()`            | Приводит текст к нижнему регистру | Нормализация        |
| `str.strip(punctuation)` | Удаляет знаки препинания          | Очистка токенов     |
| `dict`                   | Стандартный словарь Python        | Ручной подсчёт      |
| `collections.Counter`    | Специализированный счётчик        | Быстрый подсчёт     |
| `sorted()`               | Сортировка по ключу или значению  | Ранжирование слов   |

### Базовый алгоритм счётчика

```
1. Получить текст (ввод пользователя или файл)
2. Привести текст к нижнему регистру
3. Разбить текст на токены (слова)
4. Очистить каждый токен от знаков препинания
5. Подсчитать вхождения каждого слова
6. Отсортировать и вывести результат
```

## Практические упражнения

### Упражнение 1. Простой счётчик с использованием словаря

Реализуем счётчик вручную, без дополнительных библиотек.

```python
import string

text = "Кошка сидит на окне. Кошка любит солнце. Солнце светит в окно."

# Приводим к нижнему регистру
text = text.lower()

# Разбиваем на слова
words = text.split()

# Считаем вхождения
word_count = {}
for word in words:
    # Удаляем знаки препинания
    clean_word = word.strip(string.punctuation)
    if clean_word:
        word_count[clean_word] = word_count.get(clean_word, 0) + 1

# Выводим результат
print("Частота слов:")
for word, count in word_count.items():
    print(f"  {word}: {count}")
```

**Ожидаемый вывод:**
```
Частота слов:
  кошка: 2
  сидит: 1
  на: 1
  окне: 1
  любит: 1
  солнце: 2
  светит: 1
  в: 1
  окно: 1
```

### Упражнение 2. Счётчик с использованием `collections.Counter`

`Counter` автоматизирует подсчёт и предоставляет удобные методы, например `most_common()`.

```python
import string
from collections import Counter

text = """
Лингвистика изучает язык. Язык — это система знаков.
Знаки языка используются для общения. Общение — основа лингвистики.
"""

# Нормализация и токенизация
words = text.lower().split()
clean_words = [word.strip(string.punctuation) for word in words if word.strip(string.punctuation)]

# Подсчёт с помощью Counter
counter = Counter(clean_words)

# Топ-5 самых частых слов
print("Топ-5 самых частых слов:")
for word, count in counter.most_common(5):
    print(f"  {word}: {count}")
```

**Ожидаемый вывод:**
```
Топ-5 самых частых слов:
  лингвистики: 2
  язык: 2
  языка: 1
  знаки: 1
  общение: 1
```

### Упражнение 3. Сортировка результатов

Сортируем слова по убыванию частоты и выводим в виде таблицы.

```python
import string
from collections import Counter

text = """
Python — популярный язык программирования. Python используется в лингвистике,
науке о данных и веб-разработке. Программирование на Python доступно
и интересно для начинающих.
"""

words = text.lower().split()
clean_words = [w.strip(string.punctuation) for w in words if w.strip(string.punctuation)]
counter = Counter(clean_words)

# Сортировка: сначала по частоте (убывание), затем по алфавиту
sorted_words = sorted(counter.items(), key=lambda item: (-item[1], item[0]))

print(f"{'Слово':<20} {'Частота':>8}")
print("-" * 30)
for word, count in sorted_words:
    print(f"{word:<20} {count:>8}")
```

**Ожидаемый вывод:**
```
Слово                Частота
------------------------------
python                      3
программирование            2
в                           2
и                           2
лингвистике                 1
...
```

### Упражнение 4. Чтение текста из файла и запись результата

В реальных задачах NLP тексты хранятся в файлах. Научимся читать текст из файла и сохранять результат.

```python
import string
from collections import Counter

def count_words_from_file(input_path, output_path, top_n=10):
    """
    Читает текст из файла, подсчитывает частоту слов
    и сохраняет топ-N результатов в другой файл.
    """
    # Чтение входного файла
    with open(input_path, "r", encoding="utf-8") as f:
        text = f.read()

    # Обработка текста
    words = text.lower().split()
    clean_words = [w.strip(string.punctuation) for w in words if w.strip(string.punctuation)]
    counter = Counter(clean_words)

    # Запись результатов
    with open(output_path, "w", encoding="utf-8") as f:
        f.write(f"Всего уникальных слов: {len(counter)}\n")
        f.write(f"Всего слов: {sum(counter.values())}\n\n")
        f.write(f"Топ-{top_n} самых частых слов:\n")
        f.write(f"{'Слово':<20} {'Частота':>8}\n")
        f.write("-" * 30 + "\n")
        for word, count in counter.most_common(top_n):
            f.write(f"{word:<20} {count:>8}\n")

    print(f"Результат сохранён в: {output_path}")
```

### Упражнение 5. Полноценный интерактивный счётчик

Соберём все предыдущие упражнения в единую программу с выбором режима ввода.

```python
import string
from collections import Counter

def clean_tokens(raw_words):
    """Очищает список токенов от знаков препинания."""
    return [w.strip(string.punctuation) for w in raw_words if w.strip(string.punctuation)]

def display_results(counter, top_n=10):
    """Выводит результаты частотного анализа."""
    total_words = sum(counter.values())
    unique_words = len(counter)

    print(f"\nВсего слов: {total_words}")
    print(f"Уникальных слов: {unique_words}")
    print(f"\nТоп-{top_n} самых частых слов:")
    print(f"{'#':<4} {'Слово':<20} {'Частота':>8} {'%':>8}")
    print("-" * 44)

    for rank, (word, count) in enumerate(counter.most_common(top_n), start=1):
        percent = count / total_words * 100
        print(f"{rank:<4} {word:<20} {count:>8} {percent:>7.1f}%")

def main():
    print("=== Счётчик частоты слов ===\n")
    print("Выберите источник текста:")
    print("  1 — Ввод с клавиатуры")
    print("  2 — Чтение из файла")
    choice = input("\nВаш выбор (1/2): ").strip()

    if choice == "1":
        print("Введите текст (нажмите Enter дважды для завершения):")
        lines = []
        while True:
            line = input()
            if line == "":
                break
            lines.append(line)
        text = " ".join(lines)

    elif choice == "2":
        file_path = input("Введите путь к файлу: ").strip()
        try:
            with open(file_path, "r", encoding="utf-8") as f:
                text = f.read()
        except FileNotFoundError:
            print("Ошибка: файл не найден.")
            return
    else:
        print("Неверный выбор.")
        return

    if not text.strip():
        print("Текст пустой. Завершение программы.")
        return

    # Обработка
    words = text.lower().split()
    clean_words = clean_tokens(words)
    counter = Counter(clean_words)

    top_n = int(input("Сколько топ-слов вывести? (по умолчанию 10): ").strip() or 10)
    display_results(counter, top_n)

main()
```
