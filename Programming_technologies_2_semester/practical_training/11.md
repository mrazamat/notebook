## Теоретическая часть

### Что такое модуль?

Модуль в Python — это обычный файл с расширением `.py`, который содержит функции, переменные и классы. Модули позволяют разбивать программу на логические части и повторно использовать код в разных проектах.

В лингвистических задачах модули особенно полезны: можно один раз написать функции для токенизации, подсчёта частот или нормализации текста — и затем подключать их в любом проекте.

### Создание модуля

Чтобы создать модуль, достаточно сохранить файл `.py` с нужными функциями:

**Файл `text_tools.py`:**
```python
# Модуль для работы с текстом

def tokenize(text):
    """Разбивает текст на слова."""
    return text.lower().split()

def count_words(text):
    """Подсчитывает количество слов в тексте."""
    return len(tokenize(text))

def unique_words(text):
    """Возвращает множество уникальных слов."""
    return set(tokenize(text))
```

### Импорт модуля

После создания файла его можно подключить в другом скрипте. Существует несколько способов импорта:

| Способ импорта | Синтаксис | Пример вызова |
|---|---|---|
| Весь модуль | `import text_tools` | `text_tools.tokenize(текст)` |
| Конкретная функция | `from text_tools import tokenize` | `tokenize(текст)` |
| Несколько функций | `from text_tools import tokenize, count_words` | `tokenize(текст)` |
| С псевдонимом | `import text_tools as tt` | `tt.tokenize(текст)` |

### Переменная `__name__`

Каждый модуль имеет встроенную переменную `__name__`. Если файл запускается напрямую, она равна `"__main__"`. Если файл импортируется как модуль — она равна имени файла. Это позволяет писать тестовый код внутри модуля:

```python
def tokenize(text):
    return text.lower().split()

if __name__ == "__main__":
    # Этот блок выполнится только при прямом запуске
    print(tokenize("Привет мир"))
```

## Практические примеры

### Пример 1. Создание и импорт простого модуля

**Шаг 1.** Создайте файл `string_utils.py`:

```python
# string_utils.py

def reverse_text(text):
    """Переворачивает строку."""
    return text[::-1]

def capitalize_words(text):
    """Делает первую букву каждого слова заглавной."""
    return text.title()

def remove_spaces(text):
    """Удаляет лишние пробелы."""
    return " ".join(text.split())
```

**Шаг 2.** Создайте файл `main.py` в той же папке и импортируйте модуль:

```python
# main.py
import string_utils

text = "  привет   мир  "

print(string_utils.reverse_text("питон"))           # нотип
print(string_utils.capitalize_words("привет мир"))  # Привет Мир
print(string_utils.remove_spaces(text))             # привет мир
```

### Пример 2. Импорт отдельных функций

```python
# main.py
from string_utils import reverse_text, capitalize_words

print(reverse_text("лингвистика"))                       # акитсивгнил
print(capitalize_words("компьютерная лингвистика"))      # Компьютерная Лингвистика
```

### Пример 3. Лингвистический модуль с проверкой `__name__`

**Файл `freq_analysis.py`:**

```python
# freq_analysis.py — модуль частотного анализа

def get_char_frequency(text):
    """Возвращает словарь с частотой каждого символа."""
    freq = {}
    for char in text.lower():
        if char.isalpha():
            freq[char] = freq.get(char, 0) + 1
    return freq

def most_common_char(text):
    """Возвращает самый частый символ в тексте."""
    freq = get_char_frequency(text)
    if not freq:
        return None
    return max(freq, key=freq.get)

def word_frequency(text):
    """Возвращает словарь с частотой каждого слова."""
    freq = {}
    for word in text.lower().split():
        freq[word] = freq.get(word, 0) + 1
    return freq

if __name__ == "__main__":
    # Тест модуля при прямом запуске
    sample = "лингвистика изучает язык и языковые явления"
    print("Частота символов:", get_char_frequency(sample))
    print("Самый частый символ:", most_common_char(sample))
    print("Частота слов:", word_frequency(sample))
```

**Файл `main.py`:**

```python
# main.py
from freq_analysis import most_common_char, word_frequency

text = "язык и речь являются основой общения язык важен"

print("Самый частый символ:", most_common_char(text))
print("Частота слов:", word_frequency(text))
```

**Вывод:**
```
Самый частый символ: а
Частота слов: {'язык': 2, 'и': 1, 'речь': 1, 'являются': 1, 'основой': 1, 'общения': 1, 'важен': 1}
```
