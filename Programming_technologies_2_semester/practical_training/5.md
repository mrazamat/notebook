## 1. Теоретическая часть

### 1.1 Что такое «длина текста»?

В компьютерной лингвистике «длина текста» — понятие многозначное. В зависимости от задачи длину можно измерять по-разному:

| Единица измерения        | Описание                                               | Пример                    |
|--------------------------|--------------------------------------------------------|---------------------------|
| **Символы**              | Все символы строки, включая пробелы и знаки препинания | `len("Привет!")` → 7      |
| **Символы без пробелов** | Только буквы и знаки, без пробелов                     | `"Привет мир"` → 9        |
| **Слова**                | Количество слов в тексте                               | `"Я люблю Python"` → 3    |
| **Предложения**          | Количество предложений                                 | Разбивка по `.`, `!`, `?` |
| **Строки**               | Количество строк в многострочном тексте                | Разбивка по `\n`          |

### 1.2 Функция `len()`

Самый простой способ измерить длину строки в Python — функция `len()`.

```python
text = "Компьютерная лингвистика"
print(len(text))  # 24
```

Функция `len()` считает **все символы**, включая пробелы.

### 1.3 Подсчёт слов с помощью `split()`

Метод `split()` разбивает строку на список слов по пробелам. Длина этого списка — количество слов.

```python
text = "Я изучаю технологии программирования"
words = text.split()
print(len(words))  # 4
```

### 1.4 Подсчёт предложений

Чтобы подсчитать предложения, можно считать количество точек, восклицательных и вопросительных знаков.

```python
import re

text = "Привет! Как дела? Я учусь программировать."
sentences = re.split(r'[.!?]+', text)
# Убираем пустые строки после разбиения
sentences = [s for s in sentences if s.strip()]
print(len(sentences))  # 3
```

### 1.5 Длина текста без пробелов и пунктуации

Иногда нужно подсчитать только «значимые» символы — буквы.

```python
text = "Привет, мир!"
only_letters = [c for c in text if c.isalpha()]
print(len(only_letters))  # 10
```

### Упражнение 4. Функция анализа текста

Изучите и запустите следующую функцию:

```python
def analyze_text(text):
    """Возвращает основные метрики длины текста."""
    num_chars = len(text)
    num_chars_no_spaces = len(text.replace(" ", ""))
    num_words = len(text.split())
    num_letters = sum(1 for c in text if c.isalpha())

    print("=== Анализ текста ===")
    print(f"Всего символов:             {num_chars}")
    print(f"Символов без пробелов:      {num_chars_no_spaces}")
    print(f"Только букв:                {num_letters}")
    print(f"Количество слов:            {num_words}")
    if num_words > 0:
        avg = round(num_chars_no_spaces / num_words, 2)
        print(f"Средняя длина слова:        {avg} символов")

# Пример использования
my_text = "Лингвистика изучает языки и их структуру."
analyze_text(my_text)
```

### Упражнение 5. Подсчёт предложений

```python
import re

def count_sentences(text):
    """Подсчитывает количество предложений в тексте."""
    parts = re.split(r'[.!?]+', text)
    sentences = [s.strip() for s in parts if s.strip()]
    return len(sentences)

text = "Сегодня хорошая погода. Мы будем программировать! Как вам задание?"
print("Количество предложений:", count_sentences(text))
```
