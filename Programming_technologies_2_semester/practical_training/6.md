## Теоретическая часть

### Что такое частота букв?

**Частота буквы** — это показатель того, как часто данная буква встречается в тексте. Частота может быть выражена:

- в **абсолютном** виде — количество раз, сколько буква встретилась в тексте;
- в **относительном** виде — доля (процент) от общего числа букв.

Формула относительной частоты:

```
частота(буква) = количество_вхождений / общее_количество_букв
```

### Зачем это нужно в компьютерной лингвистике?

Анализ частоты букв применяется в:

- **криптографии** — для взлома шифров замены;
- **определении языка текста** — каждый язык имеет характерный частотный профиль;
- **сжатии данных** — алгоритм Хаффмана использует частотный анализ;
- **орфографических корректорах** — оценка вероятности слов.

### Частотный профиль русского языка

В русском языке наиболее частые буквы: **о, е, а, и, н, т, с, р, в, л**.

### Инструменты Python

| Инструмент            | Описание                                | Пример                                  |
|-----------------------|-----------------------------------------|-----------------------------------------|
| `str.lower()`         | Приводит строку к нижнему регистру      | `"Привет".lower()` → `"привет"`         |
| `dict`                | Словарь для хранения пар ключ-значение  | `{"а": 5, "б": 2}`                      |
| `collections.Counter` | Автоматически считает частоты элементов | `Counter("аббба")` → `{"б": 3, "а": 2}` |
| `sorted()`            | Сортировка списка или словаря           | `sorted(d.items())`                     |
| `str.isalpha()`       | Проверяет, является ли символ буквой    | `"а".isalpha()` → `True`                |

## Практические примеры

### Пример 1. Подсчёт букв вручную через словарь

```python
text = "Лингвистика — наука о языке"
text = text.lower()  # приводим к нижнему регистру

frequency = {}

for char in text:
    if char.isalpha():  # учитываем только буквы
        if char in frequency:
            frequency[char] += 1
        else:
            frequency[char] = 1

print(frequency)
```

**Вывод:**
```
{'л': 1, 'и': 3, 'н': 2, 'г': 1, 'в': 1, 'с': 1, 'т': 1, 'к': 2, 'а': 3, 'у': 1, 'о': 1, 'я': 1, 'з': 1, 'е': 1}
```

### Пример 2. Подсчёт с помощью `collections.Counter`

```python
from collections import Counter

text = "Лингвистика — наука о языке"
text = text.lower()

# Фильтруем только буквы
letters = [char for char in text if char.isalpha()]

frequency = Counter(letters)

# Выводим 5 самых частых букв
print(frequency.most_common(5))
```

**Вывод:**
```
[('и', 3), ('а', 3), ('н', 2), ('к', 2), ('л', 1)]
```

### Пример 3. Вычисление относительной частоты

```python
from collections import Counter

text = "Компьютерная лингвистика изучает язык"
text = text.lower()

letters = [char for char in text if char.isalpha()]
total = len(letters)

frequency = Counter(letters)

print(f"Всего букв: {total}")
print()
print(f"{'Буква':<10} {'Кол-во':<10} {'Частота, %'}")
print("-" * 35)

for letter, count in sorted(frequency.items()):
    relative = (count / total) * 100
    print(f"{letter:<10} {count:<10} {relative:.2f}%")
```

**Вывод:**
```
Всего букв: 31

Буква      Кол-во     Частота, %
-----------------------------------
а          3          9.68%
в          1          3.23%
г          1          3.23%
...
```

### Пример 4. Сравнение двух текстов

```python
from collections import Counter

def get_frequency(text):
    text = text.lower()
    letters = [c for c in text if c.isalpha()]
    total = len(letters)
    counter = Counter(letters)
    return {k: round(v / total * 100, 2) for k, v in counter.items()}

text1 = "Мама мыла раму"
text2 = "Папа читал книгу"

freq1 = get_frequency(text1)
freq2 = get_frequency(text2)

# Все уникальные буквы из обоих текстов
all_letters = sorted(set(freq1) | set(freq2))

print(f"{'Буква':<10} {'Текст 1, %':<15} {'Текст 2, %'}")
print("-" * 40)
for letter in all_letters:
    f1 = freq1.get(letter, 0)
    f2 = freq2.get(letter, 0)
    print(f"{letter:<10} {f1:<15} {f2}")
```
