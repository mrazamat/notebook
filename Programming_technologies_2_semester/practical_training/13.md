# Практический урок: Удаление стоп-слов из текста

**Предмет:** Технологии программирования  
**Специальность:** Компьютерная лингвистика  
**Тема:** Удаление стоп-слов из текста с помощью Python

## Теоретическая часть

### Что такое стоп-слова?

**Стоп-слова** (stop words) — это часто встречающиеся слова в языке, которые несут мало смысловой нагрузки и, как правило, удаляются при обработке текста. Примерами стоп-слов в русском языке являются: *и, в, на, с, по, для, это, как, что, он, она* и т. д.

В задачах компьютерной лингвистики удаление стоп-слов применяется при:
- частотном анализе текста,
- построении поисковых индексов,
- классификации текстов,
- извлечении ключевых слов.

### Подходы к удалению стоп-слов

| Подход           | Описание                                    | Инструмент                 |
|------------------|---------------------------------------------|----------------------------|
| Ручной список    | Программист задаёт список стоп-слов вручную | Встроенные средства Python |
| Библиотека NLTK  | Готовые списки стоп-слов для многих языков  | `nltk.corpus.stopwords`    |
| Библиотека spaCy | Встроенные стоп-слова в языковой модели     | `nlp.Defaults.stop_words`  |

### Общий алгоритм удаления стоп-слов

1. Получить исходный текст.
2. Разбить текст на токены (слова).
3. Привести токены к нижнему регистру.
4. Проверить каждый токен: входит ли он в список стоп-слов?
5. Сохранить только те токены, которых нет в списке.
6. Собрать результат обратно в строку (при необходимости).

## Практические примеры

### Пример 1. Удаление стоп-слов вручную (без библиотек)

```python
text = "Это простой пример текста и он содержит некоторые стоп слова"

stop_words = {"это", "и", "он", "в", "на", "с", "по", "для", "как", "что",
              "не", "но", "а", "или", "из", "от", "до", "за", "при", "о"}

tokens = text.lower().split()
filtered_tokens = [word for word in tokens if word not in stop_words]

result = " ".join(filtered_tokens)
print("Исходный текст:", text)
print("После фильтрации:", result)
```

**Вывод:**
```
Исходный текст: Это простой пример текста и он содержит некоторые стоп слова
После фильтрации: простой пример текста содержит некоторые стоп слова
```

### Пример 2. Использование NLTK для русского языка

```python
import nltk
nltk.download("stopwords")

from nltk.corpus import stopwords

text = "Это простой пример и он показывает как работает фильтрация слов в тексте"

russian_stop_words = set(stopwords.words("russian"))
tokens = text.lower().split()
filtered_tokens = [word for word in tokens if word not in russian_stop_words]

result = " ".join(filtered_tokens)
print("Количество слов до:", len(tokens))
print("Количество слов после:", len(filtered_tokens))
print("Отфильтрованный текст:", result)
```

### Пример 3. Добавление собственных стоп-слов к списку NLTK

```python
import nltk
from nltk.corpus import stopwords

text = "Сегодня мы изучаем Python для обработки текстов в лингвистике"

base_stop_words = set(stopwords.words("russian"))
custom_stop_words = {"сегодня", "мы", "для"}

all_stop_words = base_stop_words.union(custom_stop_words)

tokens = text.lower().split()
filtered_tokens = [word for word in tokens if word not in all_stop_words]

print("Удалённые слова:", [w for w in tokens if w in all_stop_words])
print("Результат:", " ".join(filtered_tokens))
```

### Пример 4. Удаление стоп-слов с сохранением знаков препинания (через re)

```python
import re
import nltk
from nltk.corpus import stopwords

text = "Это интересная задача, и она требует внимания к деталям."

stop_words = set(stopwords.words("russian"))
tokens = re.findall(r"\b\w+\b", text.lower())
filtered_tokens = [word for word in tokens if word not in stop_words]

print("Токены без стоп-слов:", filtered_tokens)
```

### Пример 5. Подсчёт частоты слов после удаления стоп-слов

```python
from collections import Counter
import nltk
from nltk.corpus import stopwords

text = """Лингвистика изучает язык и структуру языка.
Компьютерная лингвистика применяет методы программирования для анализа языка."""

stop_words = set(stopwords.words("russian"))
tokens = text.lower().split()
filtered_tokens = [word for word in tokens if word not in stop_words]

freq = Counter(filtered_tokens)
print("Топ-5 слов:", freq.most_common(5))
```
