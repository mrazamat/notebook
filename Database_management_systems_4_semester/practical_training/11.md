## Теоретический материал

### Классификация функций в MySQL

В MySQL функции делятся на несколько категорий в зависимости от типа обрабатываемых данных и характера вычислений.

**Скалярные функции** принимают одно значение и возвращают одно значение. Они применяются к каждой строке независимо.

**Агрегатные функции** обрабатывают набор строк и возвращают одно результирующее значение. Применяются совместно с `GROUP BY` или ко всей таблице целиком.

### 1. Агрегатные функции

| Функция                  | Описание                       | Пример                    |
|--------------------------|--------------------------------|---------------------------|
| `COUNT(*)`               | Количество строк               | `COUNT(*)`                |
| `COUNT(column)`          | Количество не-NULL значений    | `COUNT(lemma)`            |
| `COUNT(DISTINCT column)` | Количество уникальных значений | `COUNT(DISTINCT pos_tag)` |
| `SUM(column)`            | Сумма значений                 | `SUM(frequency)`          |
| `AVG(column)`            | Среднее арифметическое         | `AVG(word_length)`        |
| `MIN(column)`            | Минимальное значение           | `MIN(frequency)`          |
| `MAX(column)`            | Максимальное значение          | `MAX(frequency)`          |
| `GROUP_CONCAT(column)`   | Конкатенация значений группы   | `GROUP_CONCAT(word_form)` |

> **Важно:** Агрегатные функции игнорируют `NULL`-значения, за исключением `COUNT(*)`.

### 2. Строковые функции

| Функция                    | Описание                         | Пример                              |
|----------------------------|----------------------------------|-------------------------------------|
| `LENGTH(str)`              | Длина строки в байтах            | `LENGTH(lemma)`                     |
| `CHAR_LENGTH(str)`         | Длина строки в символах          | `CHAR_LENGTH(lemma)`                |
| `UPPER(str)`               | Перевод в верхний регистр        | `UPPER(lemma)`                      |
| `LOWER(str)`               | Перевод в нижний регистр         | `LOWER(lemma)`                      |
| `SUBSTRING(str, pos, len)` | Извлечение подстроки             | `SUBSTRING(word_form, 1, 3)`        |
| `LEFT(str, n)`             | Первые n символов                | `LEFT(lemma, 4)`                    |
| `RIGHT(str, n)`            | Последние n символов             | `RIGHT(lemma, 3)`                   |
| `CONCAT(s1, s2, ...)`      | Объединение строк                | `CONCAT(lemma, ' (', pos_tag, ')')` |
| `CONCAT_WS(sep, s1, s2)`   | Объединение с разделителем       | `CONCAT_WS(', ', lemma, pos_tag)`   |
| `TRIM(str)`                | Удаление пробелов с обоих концов | `TRIM(lemma)`                       |
| `LTRIM(str)`               | Удаление пробелов слева          | `LTRIM(word_form)`                  |
| `RTRIM(str)`               | Удаление пробелов справа         | `RTRIM(word_form)`                  |
| `REPLACE(str, from, to)`   | Замена подстроки                 | `REPLACE(lemma, 'е', 'ё')`          |
| `INSTR(str, substr)`       | Позиция подстроки                | `INSTR(lemma, 'ник')`               |
| `LOCATE(substr, str)`      | Позиция подстроки (вариант)      | `LOCATE('ать', lemma)`              |
| `LPAD(str, len, pad)`      | Дополнение слева                 | `LPAD(frequency, 6, '0')`           |
| `RPAD(str, len, pad)`      | Дополнение справа                | `RPAD(lemma, 20, '.')`              |
| `REVERSE(str)`             | Переворот строки                 | `REVERSE(lemma)`                    |

### 3. Числовые функции

| Функция                  | Описание               | Пример                     |
|--------------------------|------------------------|----------------------------|
| `ROUND(x, d)`            | Округление до d знаков | `ROUND(AVG(frequency), 2)` |
| `CEIL(x)` / `CEILING(x)` | Округление вверх       | `CEIL(frequency / 10.0)`   |
| `FLOOR(x)`               | Округление вниз        | `FLOOR(frequency / 10.0)`  |
| `ABS(x)`                 | Модуль числа           | `ABS(diff)`                |
| `MOD(x, y)`              | Остаток от деления     | `MOD(word_id, 2)`          |
| `POWER(x, y)`            | Возведение в степень   | `POWER(frequency, 2)`      |
| `SQRT(x)`                | Квадратный корень      | `SQRT(frequency)`          |
| `TRUNCATE(x, d)`         | Усечение до d знаков   | `TRUNCATE(avg_freq, 1)`    |

### 4. Функции даты и времени

| Функция                           | Описание             | Пример                                  |
|-----------------------------------|----------------------|-----------------------------------------|
| `NOW()`                           | Текущая дата и время | `NOW()`                                 |
| `CURDATE()`                       | Текущая дата         | `CURDATE()`                             |
| `YEAR(date)`                      | Год                  | `YEAR(created_at)`                      |
| `MONTH(date)`                     | Месяц (число)        | `MONTH(created_at)`                     |
| `DAY(date)`                       | День                 | `DAY(created_at)`                       |
| `MONTHNAME(date)`                 | Название месяца      | `MONTHNAME(created_at)`                 |
| `DAYNAME(date)`                   | Название дня недели  | `DAYNAME(created_at)`                   |
| `DATEDIFF(d1, d2)`                | Разница дат в днях   | `DATEDIFF(NOW(), created_at)`           |
| `DATE_FORMAT(date, fmt)`          | Форматирование даты  | `DATE_FORMAT(created_at, '%d.%m.%Y')`   |
| `DATE_ADD(date, INTERVAL n unit)` | Добавление интервала | `DATE_ADD(created_at, INTERVAL 1 YEAR)` |

### 5. Условные функции

| Функция                         | Описание                             |
|---------------------------------|--------------------------------------|
| `IF(cond, val_true, val_false)` | Условное выражение                   |
| `IFNULL(val, default)`          | Замена NULL на значение по умолчанию |
| `NULLIF(a, b)`                  | Возвращает NULL если a = b           |
| `COALESCE(v1, v2, ...)`         | Первое не-NULL значение              |

## Схема базы данных

Для выполнения заданий используется база данных лингвистического корпуса.

```sql
-- Авторы литературных произведений
CREATE TABLE authors (
    author_id   INT PRIMARY KEY AUTO_INCREMENT,
    full_name   VARCHAR(100) NOT NULL,
    birth_year  INT,
    death_year  INT
);

-- Тексты корпуса
CREATE TABLE texts (
    text_id     INT PRIMARY KEY AUTO_INCREMENT,
    title       VARCHAR(200) NOT NULL,
    author_id   INT,
    genre       VARCHAR(50),
    created_at  DATE,
    word_count  INT,
    FOREIGN KEY (author_id) REFERENCES authors(author_id)
);

-- Части речи
CREATE TABLE pos_tags (
    pos_id      INT PRIMARY KEY AUTO_INCREMENT,
    tag         VARCHAR(10) NOT NULL,
    description VARCHAR(100)
);

-- Леммы (словарные формы)
CREATE TABLE lemmas (
    lemma_id    INT PRIMARY KEY AUTO_INCREMENT,
    lemma       VARCHAR(100) NOT NULL,
    pos_id      INT,
    syllable_count INT,
    FOREIGN KEY (pos_id) REFERENCES pos_tags(pos_id)
);

-- Словоформы в корпусе с частотами
CREATE TABLE word_forms (
    form_id     INT PRIMARY KEY AUTO_INCREMENT,
    word_form   VARCHAR(100) NOT NULL,
    lemma_id    INT,
    text_id     INT,
    frequency   INT DEFAULT 0,
    FOREIGN KEY (lemma_id) REFERENCES lemmas(lemma_id),
    FOREIGN KEY (text_id) REFERENCES texts(text_id)
);
```

## Практические задания

### Блок 1. Агрегатные функции

**Задание 1.1.** Подсчитайте общее количество словоформ, зарегистрированных в корпусе.

```sql
SELECT COUNT(*) AS total_word_forms
FROM word_forms;
```

**Задание 1.2.** Найдите суммарное, среднее, минимальное и максимальное значение частоты среди всех словоформ. Среднее округлите до двух знаков после запятой.

```sql
SELECT
    SUM(frequency)            AS total_frequency,
    ROUND(AVG(frequency), 2)  AS avg_frequency,
    MIN(frequency)            AS min_frequency,
    MAX(frequency)            AS max_frequency
FROM word_forms;
```

**Задание 1.3.** Определите количество уникальных лемм, встречающихся в словоформах корпуса.

```sql
SELECT COUNT(DISTINCT lemma_id) AS unique_lemmas
FROM word_forms;
```

**Задание 1.4.** Для каждой части речи выведите: тег, описание, количество лемм. Отсортируйте по убыванию количества.

```sql
SELECT
    p.tag,
    p.description,
    COUNT(l.lemma_id) AS lemma_count
FROM pos_tags p
LEFT JOIN lemmas l ON p.pos_id = l.pos_id
GROUP BY p.pos_id, p.tag, p.description
ORDER BY lemma_count DESC;
```

**Задание 1.5.** Для каждого текста выведите название, суммарную частоту всех словоформ и количество различных лемм. Отфильтруйте тексты, у которых суммарная частота превышает 3000.

```sql
SELECT
    t.title,
    SUM(wf.frequency)          AS total_freq,
    COUNT(DISTINCT wf.lemma_id) AS unique_lemma_count
FROM texts t
JOIN word_forms wf ON t.text_id = wf.text_id
GROUP BY t.text_id, t.title
HAVING SUM(wf.frequency) > 3000
ORDER BY total_freq DESC;
```

**Задание 1.6.** Используя `GROUP_CONCAT`, выведите для каждой части речи (тег) все леммы, принадлежащие ей, через запятую. Отсортируйте леммы внутри группы по алфавиту.

```sql
SELECT
    p.tag,
    GROUP_CONCAT(l.lemma ORDER BY l.lemma SEPARATOR ', ') AS lemmas_list
FROM pos_tags p
JOIN lemmas l ON p.pos_id = l.pos_id
GROUP BY p.pos_id, p.tag;
```

### Блок 2. Строковые функции

**Задание 2.1.** Выведите все леммы вместе с их длиной в символах. Отсортируйте по убыванию длины, затем по алфавиту.


**Задание 2.8.** Для каждого автора выведите: полное имя, количество символов в имени, имя в нижнем регистре и имя с переставленными символами (`REVERSE`).

### Блок 3. Числовые функции

**Задание 3.1.** Для каждого текста вычислите среднюю частоту его словоформ. Выведите: название текста, среднее с округлением до целого (`ROUND`), округление вверх (`CEIL`) и вниз (`FLOOR`).

```sql
SELECT
    t.title,
    ROUND(AVG(wf.frequency), 0) AS avg_round,
    CEIL(AVG(wf.frequency))     AS avg_ceil,
    FLOOR(AVG(wf.frequency))    AS avg_floor
FROM texts t
JOIN word_forms wf ON t.text_id = wf.text_id
GROUP BY t.text_id, t.title;
```

**Задание 3.2.** Вычислите для каждой леммы «индекс сложности»: произведение длины леммы на количество слогов. Выведите лемму, длину, количество слогов и индекс. Отсортируйте по убыванию индекса.

```sql
SELECT
    lemma,
    CHAR_LENGTH(lemma)                           AS char_len,
    syllable_count,
    CHAR_LENGTH(lemma) * syllable_count          AS complexity_index
FROM lemmas
ORDER BY complexity_index DESC;
```

**Задание 3.3.** Разделите все словоформы на «высокочастотные» (частота чётная) и «остальные» с помощью `MOD`. Выведите словоформу, частоту и результат `MOD(frequency, 2)`.

```sql
SELECT
    word_form,
    frequency,
    MOD(frequency, 2)                               AS is_odd,
    IF(MOD(frequency, 2) = 0, 'чётная', 'нечётная') AS parity_label
FROM word_forms
ORDER BY frequency DESC;
```

**Задание 3.4.** Для каждой словоформы вычислите «нормализованную частоту» как `frequency / word_count * 1000000` (частота на миллион слов), округлённую до 4 знаков. Выведите словоформу, текст, исходную частоту и нормализованную частоту.

```sql
SELECT
    wf.word_form,
    t.title,
    wf.frequency,
    ROUND(wf.frequency / t.word_count * 1000000, 4) AS freq_per_million
FROM word_forms wf
JOIN texts t ON wf.text_id = t.text_id
ORDER BY freq_per_million DESC
LIMIT 15;
```

### Блок 4. Функции даты и времени

**Задание 4.1.** Для каждого текста выведите: название, год создания, название месяца и день недели создания (на основе поля `created_at`).

```sql
SELECT
    title,
```


**Задание 4.3.** Выведите для каждого текста его дату создания в формате `ДД.ММ.ГГГГ`.

```sql
SELECT
```

```sql

    MAX(l.lemma)                             AS longest_lemma
```

**Задание 6.2.** Найдите для каждого текста «топ-лемму» — лемму с наибольшей суммарной частотой словоформ. Используйте подзапрос для нахождения максимума.

```sql

    t.title,
    l.lemma,
    SUM(wf.frequency) AS lemma_total_freq
FROM texts t
JOIN word_forms wf  ON t.text_id  = wf.text_id
JOIN lemmas l       ON wf.lemma_id = l.lemma_id
GROUP BY t.text_id, t.title, l.lemma_id, l.lemma
HAVING SUM(wf.frequency) = (
    SELECT MAX(sub_freq)
    FROM (
        SELECT SUM(wf2.frequency) AS sub_freq
        FROM word_forms wf2
        WHERE wf2.text_id = t.text_id
        GROUP BY wf2.lemma_id
    ) sub
)
ORDER BY t.title;
```

**Задание 6.3.** Постройте «частотный профиль» по частям речи для каждого текста: название текста, тег части речи, суммарная частота словоформ этой ЧР в данном тексте, доля от общей частоты текста в процентах (округлить до 1 знака). Используйте оконные функции или подзапросы.

```sql
SELECT
    t.title,
    p.tag,
    SUM(wf.frequency)                           AS pos_freq,
    ROUND(
        SUM(wf.frequency) * 100.0
        / SUM(SUM(wf.frequency)) OVER (PARTITION BY t.text_id),
        1
    )                                           AS pos_share_pct
FROM texts t
JOIN word_forms wf ON t.text_id  = wf.text_id
JOIN lemmas l      ON wf.lemma_id = l.lemma_id
JOIN pos_tags p    ON l.pos_id    = p.pos_id
GROUP BY t.text_id, t.title, p.pos_id, p.tag
ORDER BY t.title, pos_freq DESC;
```

**Задание 6.4.** Для каждой леммы вычислите «индекс распространённости»: количество текстов, в которых она встречается, делённое на общее количество текстов в корпусе, умноженное на 100. Выведите лемму, ЧР, индекс (округлить до 2 знаков) и метку: `'ядерная'` (> 50), `'периферийная'` (10–50), `'редкая'` (< 10).

```sql
SELECT
    l.lemma,
    p.tag,
    ROUND(
        COUNT(DISTINCT wf.text_id) * 100.0 / (SELECT COUNT(*) FROM texts),
        2
    ) AS spread_index,
    CASE
        WHEN COUNT(DISTINCT wf.text_id) * 100.0 / (SELECT COUNT(*) FROM texts) > 50
            THEN 'ядерная'
        WHEN COUNT(DISTINCT wf.text_id) * 100.0 / (SELECT COUNT(*) FROM texts) >= 10
            THEN 'периферийная'
        ELSE 'редкая'
    END AS coverage_label
FROM lemmas l
JOIN pos_tags p    ON l.pos_id    = p.pos_id
JOIN word_forms wf ON l.lemma_id  = wf.lemma_id
GROUP BY l.lemma_id, l.lemma, p.tag
ORDER BY spread_index DESC;
```
