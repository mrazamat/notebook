## 1. Теоретический справочник

### 1.1 Функции округления и отсечения

| Функция            | Синтаксис        | Описание               | Пример               | Результат |
|--------------------|------------------|------------------------|----------------------|-----------|
| `ROUND`            | `ROUND(x, d)`    | Округление до d знаков | `ROUND(3.567, 2)`    | `3.57`    |
| `FLOOR`            | `FLOOR(x)`       | Округление вниз        | `FLOOR(3.9)`         | `3`       |
| `CEIL` / `CEILING` | `CEIL(x)`        | Округление вверх       | `CEIL(3.1)`          | `4`       |
| `TRUNCATE`         | `TRUNCATE(x, d)` | Отсечение до d знаков  | `TRUNCATE(3.567, 2)` | `3.56`    |

> **Разница `ROUND` и `TRUNCATE`:** `ROUND(3.567, 2)` → `3.57` (округляет), `TRUNCATE(3.567, 2)` → `3.56` (просто отрезает без округления).

### 1.2 Математические функции

| Функция         | Синтаксис               | Описание                | Пример          | Результат  |
|-----------------|-------------------------|-------------------------|-----------------|------------|
| `ABS`           | `ABS(x)`                | Абсолютное значение     | `ABS(-15)`      | `15`       |
| `MOD`           | `MOD(x, y)` или `x % y` | Остаток от деления      | `MOD(10, 3)`    | `1`        |
| `POWER` / `POW` | `POWER(x, y)`           | Возведение в степень    | `POWER(2, 8)`   | `256`      |
| `SQRT`          | `SQRT(x)`               | Квадратный корень       | `SQRT(144)`     | `12`       |
| `EXP`           | `EXP(x)`                | Экспонента e^x          | `EXP(1)`        | `2.718...` |
| `LN`            | `LN(x)`                 | Натуральный логарифм    | `LN(2.718)`     | `≈1`       |
| `LOG`           | `LOG(b, x)`             | Логарифм по основанию b | `LOG(10, 1000)` | `3`        |
| `LOG2`          | `LOG2(x)`               | Логарифм по основанию 2 | `LOG2(8)`       | `3`        |
| `LOG10`         | `LOG10(x)`              | Десятичный логарифм     | `LOG10(100)`    | `2`        |

### 1.3 Функции знака и случайности

| Функция | Синтаксис               | Описание                | Пример      | Результат     |
|---------|-------------------------|-------------------------|-------------|---------------|
| `SIGN`  | `SIGN(x)`               | Знак числа: -1, 0 или 1 | `SIGN(-42)` | `-1`          |
| `RAND`  | `RAND()` / `RAND(seed)` | Случайное число [0, 1)  | `RAND()`    | `0.748...`    |
| `PI`    | `PI()`                  | Число π                 | `PI()`      | `3.141592...` |

### 1.4 Формулы, применяемые в корпусной лингвистике

В корпусной лингвистике числовые функции используются для вычисления метрик:

| Метрика                     | Формула SQL                        | Описание                    |
|-----------------------------|------------------------------------|-----------------------------|
| Относительная частота (ipm) | `(freq * 1000000.0) / corpus_size` | Вхождений на миллион слов   |
| Доля в %                    | `ROUND(freq * 100.0 / total, 2)`   | Процент от общего объёма    |
| TF (term frequency)         | `freq * 1.0 / doc_length`          | Частота термина в документе |
| TF-IDF (упрощённо)          | `tf * LOG(N / df)`                 | Вес термина в документе     |
| Стандартное отклонение      | `SQRT(variance)`                   | Разброс значений            |

## 2. Схема базы данных

Перед выполнением заданий создайте таблицы и заполните их данными.

```sql
-- Тексты корпуса
CREATE TABLE corpus_texts (
    text_id     INT PRIMARY KEY AUTO_INCREMENT,
    title       VARCHAR(200),
    author      VARCHAR(100),
    genre       VARCHAR(50),
    word_count  INT,
    char_count  INT,
    year_written INT
);

-- Леммы (словарные формы)
CREATE TABLE lemmas (
    lemma_id    INT PRIMARY KEY AUTO_INCREMENT,
    lemma       VARCHAR(100),
    pos_tag     VARCHAR(20),   -- NOUN, VERB, ADJ, ADV, ...
    is_stopword TINYINT DEFAULT 0
);

-- Частотный словарь: сколько раз лемма встречается в каждом тексте
CREATE TABLE word_frequencies (
    freq_id     INT PRIMARY KEY AUTO_INCREMENT,
    text_id     INT,
    lemma_id    INT,
    frequency   INT,
    FOREIGN KEY (text_id)  REFERENCES corpus_texts(text_id),
    FOREIGN KEY (lemma_id) REFERENCES lemmas(lemma_id)
);

-- Морфемный анализ лемм
CREATE TABLE morpheme_stats (
    stat_id         INT PRIMARY KEY AUTO_INCREMENT,
    lemma_id        INT,
    morpheme_count  INT,
    syllable_count  INT,
    avg_morpheme_len DECIMAL(5,2),
    FOREIGN KEY (lemma_id) REFERENCES lemmas(lemma_id)
);
```

## 3. Демонстрационные примеры

### Пример 1. Базовые числовые функции

```sql
-- Различные операции округления с одним значением
SELECT
    3.14159                          AS original,
    ROUND(3.14159, 2)                AS rounded_2,
    FLOOR(3.14159)                   AS floored,
    CEIL(3.14159)                    AS ceiled,
    TRUNCATE(3.14159, 2)             AS truncated_2,
    ABS(-3.14159)                    AS absolute;
```

**Результат:**

| original | rounded_2 | floored | ceiled | truncated_2 | absolute |
|----------|-----------|---------|--------|-------------|----------|
| 3.14159  | 3.14      | 3       | 4      | 3.14        | 3.14159  |

### Пример 2. Относительная частота слов (ipm)

Частота на миллион слов (instances per million) — стандартная метрика в корпусной лингвистике.

```sql
SELECT
    l.lemma,
    l.pos_tag,
    wf.frequency                                    AS abs_freq,
    ct.word_count,
    ROUND(wf.frequency * 1000000.0 / ct.word_count, 2) AS ipm
FROM word_frequencies wf
JOIN lemmas       l  ON wf.lemma_id = l.lemma_id
JOIN corpus_texts ct ON wf.text_id  = ct.text_id
WHERE ct.title = 'Война и мир'
ORDER BY ipm DESC;
```

### Пример 3. Процентная доля леммы в тексте

```sql
SELECT
    ct.title,
    l.lemma,
    wf.frequency,
    ROUND(wf.frequency * 100.0 / ct.word_count, 4) AS share_pct
FROM word_frequencies wf
JOIN lemmas       l  ON wf.lemma_id = l.lemma_id
JOIN corpus_texts ct ON wf.text_id  = ct.text_id
WHERE l.lemma = 'человек'
ORDER BY share_pct DESC;
```

### Пример 4. Логарифмическая шкала частот

В лингвистике часто применяют логарифмирование для сглаживания распределения Ципфа.

```sql
SELECT
    l.lemma,
    SUM(wf.frequency)              AS total_freq,
    ROUND(LOG10(SUM(wf.frequency)), 4) AS log10_freq,
    ROUND(LN(SUM(wf.frequency)),   4) AS ln_freq
FROM word_frequencies wf
JOIN lemmas l ON wf.lemma_id = l.lemma_id
GROUP BY l.lemma_id, l.lemma
ORDER BY total_freq DESC;
```

### Пример 5. FLOOR для группировки по диапазонам

```sql
-- Разбить тексты на классы по объёму: 0-99к, 100-199к, 200-299к, ...
SELECT
    FLOOR(word_count / 100000) * 100000   AS size_class_from,
    FLOOR(word_count / 100000) * 100000
        + 99999                           AS size_class_to,
    COUNT(*)                              AS text_count,
    GROUP_CONCAT(title ORDER BY word_count) AS titles
FROM corpus_texts
GROUP BY FLOOR(word_count / 100000)
ORDER BY size_class_from;
```

### Пример 6. MOD — чётные и нечётные lemma_id

```sql
SELECT
    lemma_id,
    lemma,
    CASE MOD(lemma_id, 2)
        WHEN 0 THEN 'чётный'
        ELSE       'нечётный'
    END AS parity
FROM lemmas
ORDER BY lemma_id;
```
