## Теоретическая часть
### 1. Оператор GROUP BY

Оператор `GROUP BY` используется для объединения строк с одинаковыми значениями в указанных столбцах в сводные строки. Как правило, он применяется совместно с агрегатными функциями.

**Синтаксис:**

```sql
SELECT столбец1, агрегатная_функция(столбец2)
FROM таблица
WHERE условие
GROUP BY столбец1
ORDER BY столбец1;
```

> **Важно:** Все столбцы в `SELECT`, которые не являются агрегатными функциями, **обязаны** присутствовать в `GROUP BY`.



### 2. Агрегатные функции

| Функция   | Описание                 | Пример             |
|-----------|--------------------------|--------------------|
| `COUNT()` | Подсчёт количества строк | `COUNT(*)`         |
| `SUM()`   | Сумма значений           | `SUM(частота)`     |
| `AVG()`   | Среднее значение         | `AVG(длина_слова)` |
| `MIN()`   | Минимальное значение     | `MIN(частота)`     |
| `MAX()`   | Максимальное значение    | `MAX(длина_слова)` |



### 3. Оператор HAVING

`HAVING` используется для фильтрации **групп** (аналог `WHERE`, но для агрегированных данных).

**Отличие `WHERE` от `HAVING`:**

```sql
-- WHERE фильтрует строки ДО группировки
SELECT часть_речи, COUNT(*) AS кол_во
FROM слова
WHERE язык = 'русский'
GROUP BY часть_речи;

-- HAVING фильтрует группы ПОСЛЕ группировки
SELECT часть_речи, COUNT(*) AS кол_во
FROM слова
GROUP BY часть_речи
HAVING COUNT(*) > 100;
```



### 4. Порядок выполнения SQL-запроса

```
FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY → LIMIT
```

### Задание 1. Базовая группировка

**Задача:** Подсчитать количество слов по частям речи в русском языке.

```sql
SELECT
    pos                  AS часть_речи,
    COUNT(*)             AS количество_слов,
    AVG(frequency)       AS средняя_частота,
    MAX(frequency)       AS макс_частота
FROM words
WHERE language = 'русский'
GROUP BY pos
ORDER BY количество_слов DESC;
```

### Задание 2. Группировка с фильтрацией HAVING

**Задача:** Найти части речи, у которых средняя частота употребления превышает 30.

```sql
SELECT
    pos                  AS часть_речи,
    COUNT(*)             AS кол_во_слов,
    ROUND(AVG(frequency), 2) AS средняя_частота
FROM words
WHERE language = 'русский'
GROUP BY pos
HAVING AVG(frequency) > 30
ORDER BY средняя_частота DESC;
```


### Задание 3. Группировка по нескольким столбцам

**Задача:** Проанализировать распределение частей речи по языкам.

```sql
SELECT
    language             AS язык,
    pos                  AS часть_речи,
    COUNT(*)             AS кол_во,
    SUM(frequency)       AS суммарная_частота
FROM words
GROUP BY language, pos
ORDER BY язык, кол_во DESC;
```


### Задание 4. Группировка с JOIN

**Задача:** Подсчитать среднее количество слов в текстах по жанрам.

```sql
SELECT
    t.genre              AS жанр,
    COUNT(t.text_id)     AS кол_во_текстов,
    ROUND(AVG(t.word_count), 0)  AS среднее_слов,
    MIN(t.word_count)    AS мин_слов,
    MAX(t.word_count)    AS макс_слов
FROM texts t
GROUP BY t.genre
HAVING COUNT(t.text_id) >= 5
ORDER BY среднее_слов DESC;
```


### Задание 5. Вложенные агрегаты и подзапросы

**Задача:** Найти языки, в которых количество уникальных лемм превышает среднее значение по всем языкам.

```sql
SELECT
    language             AS язык,
    COUNT(DISTINCT lemma) AS уникальных_лемм
FROM words
GROUP BY language
HAVING COUNT(DISTINCT lemma) > (
    SELECT AVG(lemma_count)
    FROM (
        SELECT COUNT(DISTINCT lemma) AS lemma_count
        FROM words
        GROUP BY language
    ) AS sub
)
ORDER BY уникальных_лемм DESC;
```


### Задание 6. Группировка с оконными функциями (расширенное)

**Задача:** Для каждой части речи вычислить долю (%) от общего числа слов в языке.

```sql
SELECT
    language             AS язык,
    pos                  AS часть_речи,
    COUNT(*)             AS кол_во,
    ROUND(
        COUNT(*) * 100.0 / SUM(COUNT(*)) OVER (PARTITION BY language),
        2
    )                    AS процент
FROM words
GROUP BY language, pos
ORDER BY язык, процент DESC;
```
