## Теоретическая часть

Строковые функции SQL позволяют выполнять различные операции над текстовыми данными: поиск, замену, форматирование, извлечение подстрок. Для специалиста по компьютерной лингвистике владение этими инструментами необходимо при работе с лингвистическими корпусами, словарями и текстовыми базами данных.

### Основные строковые функции

| Функция               | Синтаксис                    | Описание                       |
|-----------------------|------------------------------|--------------------------------|
| `LENGTH` / `LEN`      | `LENGTH(строка)`             | Возвращает длину строки        |
| `UPPER`               | `UPPER(строка)`              | Переводит в верхний регистр    |
| `LOWER`               | `LOWER(строка)`              | Переводит в нижний регистр     |
| `SUBSTRING`           | `SUBSTRING(строка, нач, дл)` | Извлекает подстроку            |
| `TRIM`                | `TRIM(строка)`               | Удаляет пробелы с обоих концов |
| `LTRIM`               | `LTRIM(строка)`              | Удаляет пробелы слева          |
| `RTRIM`               | `RTRIM(строка)`              | Удаляет пробелы справа         |
| `REPLACE`             | `REPLACE(стр, что, чем)`     | Заменяет подстроку             |
| `CONCAT`              | `CONCAT(стр1, стр2, ...)`    | Объединяет строки              |
| `INSTR` / `CHARINDEX` | `INSTR(строка, подстрока)`   | Позиция вхождения подстроки    |
| `LEFT`                | `LEFT(строка, n)`            | Первые n символов              |
| `RIGHT`               | `RIGHT(строка, n)`           | Последние n символов           |
| `LPAD`                | `LPAD(стр, дл, символ)`      | Дополняет строку слева         |
| `RPAD`                | `RPAD(стр, дл, символ)`      | Дополняет строку справа        |
| `REVERSE`             | `REVERSE(строка)`            | Переворачивает строку          |

## Практические задания

### Задание 1. Функции изменения регистра

**Задача:** Вывести все слова из словаря в верхнем регистре, нижнем регистре и с заглавной буквой.

```sql
SELECT
    word                                          AS исходное_слово,
    UPPER(word)                                   AS верхний_регистр,
    LOWER(word)                                   AS нижний_регистр,
    CONCAT(UPPER(LEFT(word, 1)), LOWER(SUBSTRING(word, 2, LENGTH(word))))
                                                  AS с_заглавной
FROM dictionary;
```

**Ожидаемый результат:**

| исходное_слово | верхний_регистр | нижний_регистр | с_заглавной |
|----------------|-----------------|----------------|-------------|
| лингвистика    | ЛИНГВИСТИКА     | лингвистика    | Лингвистика |
| морфология     | МОРФОЛОГИЯ      | морфология     | Морфология  |
| синтаксис      | СИНТАКСИС       | синтаксис      | Синтаксис   |

### Задание 2. Определение длины строки

**Задача:** Найти длину каждого термина и отсортировать слова по длине (от длинных к коротким).

```sql
SELECT
    word                  AS термин,
    LENGTH(word)          AS количество_символов,
    pos                   AS часть_речи
FROM dictionary
ORDER BY LENGTH(word) DESC;
```

**Дополнительное задание:** Найти слова, длина которых превышает 8 символов.

```sql
SELECT word, LENGTH(word) AS длина
FROM dictionary
WHERE LENGTH(word) > 8
ORDER BY длина DESC;
```

### Задание 3. Извлечение подстрок

**Задача:** Извлечь первые 4 символа каждого слова (префикс) и последние 3 символа (суффикс/окончание).

```sql
SELECT
    word                                              AS слово,
    LEFT(word, 4)                                     AS префикс_4,
    RIGHT(word, 3)                                    AS окончание_3,
    SUBSTRING(word, 2, LENGTH(word) - 2)              AS без_первой_и_последней
FROM dictionary;
```

**Пояснение для лингвистов:** Функция `SUBSTRING(строка, начало, длина)` позволяет моделировать морфемный анализ — выделение корней, суффиксов и окончаний по позиции символа.

### Задание 4. Поиск подстроки

**Задача:** Найти все слова, в которых встречается подстрока «лог», и определить её позицию.

```sql
-- MySQL / MariaDB
SELECT
    word,
    INSTR(word, 'лог')    AS позиция_вхождения
FROM dictionary
WHERE INSTR(word, 'лог') > 0;

-- PostgreSQL / MS SQL Server
SELECT
    word,
    POSITION('лог' IN word)   AS позиция_вхождения
FROM dictionary
WHERE POSITION('лог' IN word) > 0;
```

**Ожидаемый результат:**

| word         | позиция_вхождения |
|--------------|-------------------|
| морфология   | 8                 |
| семасиология | 9                 |

### Задание 5. Замена подстрок

**Задача:** В столбце `origin` заменить сокращение «греч.» на «греческое», «лат.» на «латинское».

```sql
SELECT
    word,
    origin                                           AS исходное_происхождение,
    REPLACE(
        REPLACE(origin, 'греч.', 'греческое'),
        'лат.', 'латинское'
    )                                                AS полное_происхождение
FROM dictionary;
```

### Задание 6. Удаление пробелов (TRIM)

**Задача:** Очистить данные в таблице `corpus_texts` — удалить лишние пробелы из поля `content`.

```sql
-- Просмотр до очистки
SELECT
    id,
    content                AS с_пробелами,
    LENGTH(content)        AS длина_до,
    TRIM(content)          AS без_пробелов,
    LENGTH(TRIM(content))  AS длина_после
FROM corpus_texts;

-- Обновление данных
UPDATE corpus_texts
SET content = TRIM(content)
WHERE content != TRIM(content);
```

### Задание 8. Дополнение строк (LPAD / RPAD)

**Задача:** Сформировать нумерованный список терминов, выровнив номера и слова по ширине.

```sql
SELECT
    CONCAT(
        LPAD(CAST(id AS CHAR), 3, '0'),
        ' | ',
        RPAD(word, 15, ' '),
        ' | ',
        pos
    ) AS форматированная_запись
FROM dictionary
ORDER BY id;
```

**Ожидаемый результат:**

### Задание 9. Реверс строки

**Задача:** Найти палиндромы в специально созданной таблице слов.

```sql
-- Создание тестовой таблицы
CREATE TABLE test_words (
    id   INTEGER PRIMARY KEY,
    word VARCHAR(50)
);

INSERT INTO test_words VALUES
(1, 'кок'), (2, 'дед'), (3, 'лингвист'),
(4, 'топот'), (5, 'язык'), (6, 'казак'),
(7, 'радар'), (8, 'корпус');

-- Поиск палиндромов
SELECT
    word                         AS слово,
    REVERSE(word)                AS перевёрнутое,
    CASE
        WHEN word = REVERSE(word) THEN 'Палиндром'
        ELSE 'Не палиндром'
    END                          AS тип
FROM test_words;
```

### Задание 10. Комплексный запрос

**Задача:** Для каждого термина вывести: нормализованную форму (нижний регистр, без пробелов), длину, первые 3 символа и проверить, содержит ли слово букву «а».

```sql
SELECT
    word                                               AS термин,
    LOWER(TRIM(word))                                  AS нормализованная_форма,
    LENGTH(TRIM(word))                                 AS длина,
    LEFT(LOWER(word), 3)                               AS первые_3_символа,
    CASE
        WHEN INSTR(word, 'а') > 0 THEN 'Содержит "а"'
        ELSE 'Не содержит "а"'
    END                                                AS наличие_буквы_а,
    REPLACE(word, 'а', '*')                            AS с_маркировкой_а
FROM dictionary
ORDER BY LENGTH(word);
```
