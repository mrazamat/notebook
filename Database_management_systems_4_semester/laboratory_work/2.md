## Теоретическая часть

### 1. Синтаксис оператора SELECT с WHERE

```sql
SELECT столбец1, столбец2, ...
FROM таблица
WHERE условие;
```

Команда `WHERE` ограничивает количество возвращаемых строк, оставляя только те записи, для которых условие является **истинным**.

---

### 2. Операторы сравнения

| Оператор | Описание           | Пример                      |
|----------|--------------------|-----------------------------|
| `=`      | Равно              | `WHERE year = 2023`         |
| `<>` или `!=` | Не равно      | `WHERE lang <> 'русский'`   |
| `>`      | Больше             | `WHERE word_count > 1000`   |
| `<`      | Меньше             | `WHERE frequency < 0.5`     |
| `>=`     | Больше или равно   | `WHERE rating >= 4`         |
| `<=`     | Меньше или равно   | `WHERE length <= 10`        |

---

### 3. Логические операторы

- **AND** — оба условия должны быть истинными
- **OR** — хотя бы одно условие должно быть истинным
- **NOT** — инвертирует условие

```sql
-- AND: слова длиннее 5 букв И относящиеся к глаголам
SELECT * FROM words WHERE length > 5 AND part_of_speech = 'глагол';

-- OR: слова из русского ИЛИ английского языка
SELECT * FROM words WHERE language = 'русский' OR language = 'английский';

-- NOT: все слова, кроме существительных
SELECT * FROM words WHERE NOT part_of_speech = 'существительное';
```

---

### 4. Специальные операторы

#### BETWEEN — диапазон значений
```sql
-- Слова с частотой употребления от 100 до 500
SELECT word, frequency FROM corpus WHERE frequency BETWEEN 100 AND 500;
```

#### IN — список значений
```sql
-- Слова, относящиеся к определённым частям речи
SELECT * FROM words WHERE part_of_speech IN ('существительное', 'глагол', 'прилагательное');
```

#### LIKE — поиск по шаблону
```sql
-- Слова, начинающиеся на букву "п"
SELECT * FROM words WHERE word LIKE 'п%';

-- Слова, содержащие "лингв"
SELECT * FROM words WHERE word LIKE '%лингв%';

-- Слова из 5 букв
SELECT * FROM words WHERE word LIKE '_____';
```

> `%` — любое количество любых символов  
> `_` — ровно один любой символ

#### IS NULL / IS NOT NULL — проверка на пустое значение
```sql
-- Слова без указанного перевода
SELECT * FROM words WHERE translation IS NULL;

-- Слова с заполненным полем перевода
SELECT * FROM words WHERE translation IS NOT NULL;
```

### Задания

#### Задание 1. Оператор сравнения `=`
Выбрать все слова русского языка.

```sql
SELECT * FROM words
WHERE language = 'русский';
```

**Ожидаемый результат:** строки с id 1, 3, 4, 6, 8.

---

#### Задание 2. Оператор сравнения `>`
Выбрать слова с частотой употребления более 500.

```sql
SELECT word, frequency FROM words
WHERE frequency > 500;
```

**Ожидаемый результат:** speak (850), read (910), писать (760).

---

#### Задание 3. Оператор `AND`
Выбрать существительные русского языка.

```sql
SELECT word, part_of_speech, language FROM words
WHERE part_of_speech = 'существительное' AND language = 'русский';
```

**Ожидаемый результат:** лингвистика, морфема, синтаксис, фонема.

---

#### Задание 4. Оператор `OR`
Выбрать слова длиной 4 или 5 символов.

```sql
SELECT word, length FROM words
WHERE length = 4 OR length = 5;
```

**Ожидаемый результат:** speak (5), read (4).

---

#### Задание 5. Оператор `BETWEEN`
Выбрать тексты корпуса, написанные с 2019 по 2021 год.

```sql
SELECT title, year FROM texts
WHERE year BETWEEN 2019 AND 2021;
```

**Ожидаемый результат:** English Grammar (2019), Syntax Basics (2021), Введение в лингв. (2020).

---

#### Задание 6. Оператор `IN`
Выбрать слова, которые являются существительными или прилагательными.

```sql
SELECT word, part_of_speech FROM words
WHERE part_of_speech IN ('существительное', 'прилагательное');
```

**Ожидаемый результат:** лингвистика, морфема, синтаксис, фонема, beautiful.

---

#### Задание 7. Оператор `LIKE`
Выбрать слова, содержащие подстроку "ема".

```sql
SELECT word FROM words
WHERE word LIKE '%ема%';
```

**Ожидаемый результат:** морфема, фонема.

---

#### Задание 8*. Комбинирование условий
Выбрать английские глаголы с частотой более 800 и длиной слова меньше 6 символов.

```sql
SELECT word, part_of_speech, frequency, length FROM words
WHERE language = 'английский'
  AND part_of_speech = 'глагол'
  AND frequency > 800
  AND length < 6;
```

**Ожидаемый результат:** speak (850, длина 5), read (910, длина 4).
