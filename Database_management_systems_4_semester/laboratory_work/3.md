## Теоретическая часть

### 1. Назначение ORDER BY

Команда `ORDER BY` используется для сортировки результирующего набора данных, возвращаемого оператором `SELECT`. Без `ORDER BY` порядок строк в результате не гарантирован.

**Общий синтаксис:**

```sql
SELECT столбец1, столбец2, ...
FROM таблица
[WHERE условие]
ORDER BY столбец1 [ASC | DESC], столбец2 [ASC | DESC], ...;
```

### 2. Параметры сортировки

| Параметр | Описание | Пример для текста | Пример для чисел |
|----------|----------|-------------------|------------------|
| `ASC` | По возрастанию (по умолчанию) | А → Я (A → Z) | 1, 2, 3... |
| `DESC` | По убыванию | Я → А (Z → A) | ...3, 2, 1 |

> **Важно:** Если направление сортировки не указано, по умолчанию применяется `ASC`.

### 3. Сортировка по нескольким столбцам

При указании нескольких столбцов данные сортируются последовательно: сначала по первому столбцу, затем — при совпадении значений — по второму и т.д.

```sql
ORDER BY столбец1 ASC, столбец2 DESC;
```

### 4. Сортировка по порядковому номеру столбца

Допускается указывать номер столбца вместо его имени:

```sql
SELECT word, frequency, language
FROM lexicon
ORDER BY 2 DESC;  -- сортировка по столбцу frequency
```

### 5. NULL-значения при сортировке

- В большинстве СУБД `NULL` при `ASC` выводится **последним**, при `DESC` — **первым**.
- В PostgreSQL поведение можно явно задать: `NULLS FIRST` / `NULLS LAST`.

```sql
ORDER BY frequency ASC NULLS LAST;
```

### Задание 6. Сортировка по вычисляемому выражению

**Условие:** Вывести слова и вычислить «индекс значимости» (частота / длина слова). Отсортировать по этому индексу по убыванию.

```sql

       language,
       frequency,
```

### Задание 7* (повышенной сложности). Сортировка с подзапросом

**Условие:** Вывести языки с их суммарной частотой слов в базе, отсортировав от наиболее «представленного» языка к наименее.

```sql
SELECT language,
       COUNT(word_id)     AS word_count,
       SUM(frequency)     AS total_frequency,
       AVG(length)        AS avg_word_length
FROM words
GROUP BY language
ORDER BY total_frequency DESC;
```

**Ожидаемый результат:**

| language    | word_count | total_frequency | avg_word_length |
|-------------|------------|-----------------|-----------------|
| русский     | 11         | 43800           | 6.9             |
| английский  | 4          | 19900           | 7.5             |
