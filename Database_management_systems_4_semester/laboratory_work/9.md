## Теоретическая часть

### 1. Классификация функций SQL

Функции SQL делятся на несколько групп:

#### 1.1 Агрегатные функции
Работают с набором строк и возвращают единственное значение.

| Функция   | Описание                 | Пример            |
|-----------|--------------------------|-------------------|
| `COUNT()` | Подсчёт количества строк | `COUNT(*)`        |
| `SUM()`   | Сумма значений           | `SUM(word_count)` |
| `AVG()`   | Среднее значение         | `AVG(frequency)`  |
| `MAX()`   | Максимальное значение    | `MAX(length)`     |
| `MIN()`   | Минимальное значение     | `MIN(year)`       |

#### 1.2 Строковые функции
Применяются для обработки текстовых данных — особенно важны в компьютерной лингвистике.

| Функция                   | Описание                  | Пример                    |
|---------------------------|---------------------------|---------------------------|
| `LENGTH()` / `LEN()`      | Длина строки              | `LENGTH(word)`            |
| `UPPER()`                 | Перевод в верхний регистр | `UPPER(word)`             |
| `LOWER()`                 | Перевод в нижний регистр  | `LOWER(word)`             |
| `SUBSTRING()`             | Извлечение подстроки      | `SUBSTRING(text, 1, 5)`   |
| `REPLACE()`               | Замена подстроки          | `REPLACE(text, 'а', 'о')` |
| `TRIM()`                  | Удаление пробелов         | `TRIM(word)`              |
| `CONCAT()`                | Конкатенация строк        | `CONCAT(word, ' ', pos)`  |
| `CHARINDEX()` / `INSTR()` | Позиция подстроки         | `INSTR(text, 'лингв')`    |
| `LEFT()` / `RIGHT()`      | Символы с начала/конца    | `LEFT(word, 3)`           |

#### 1.3 Числовые функции

| Функция                | Описание             | Пример               |
|------------------------|----------------------|----------------------|
| `ROUND()`              | Округление           | `ROUND(avg_freq, 2)` |
| `ABS()`                | Модуль числа         | `ABS(difference)`    |
| `FLOOR()`              | Округление вниз      | `FLOOR(3.9)` → 3     |
| `CEIL()` / `CEILING()` | Округление вверх     | `CEIL(3.1)` → 4      |
| `POWER()`              | Возведение в степень | `POWER(x, 2)`        |

#### 1.4 Функции работы с датами

| Функция               | Описание             | Пример                  |
|-----------------------|----------------------|-------------------------|
| `NOW()` / `GETDATE()` | Текущая дата и время | `NOW()`                 |
| `YEAR()`              | Год из даты          | `YEAR(pub_date)`        |
| `MONTH()`             | Месяц из даты        | `MONTH(created_at)`     |
| `DATEDIFF()`          | Разница между датами | `DATEDIFF(day, d1, d2)` |

## Практическая часть

### Структура учебной базы данных

Для выполнения заданий используется база данных **LinguisticCorpus** — корпус текстов для лингвистического анализа.

```sql
-- Таблица текстов
CREATE TABLE texts (
    text_id     INT PRIMARY KEY,
    title       VARCHAR(200),
    content     TEXT,
    language    VARCHAR(50),
    genre       VARCHAR(100),
    author      VARCHAR(150),
    pub_date    DATE,
    word_count  INT
);

-- Таблица слов (словарный запас корпуса)
CREATE TABLE words (
    word_id     INT PRIMARY KEY,
    word_form   VARCHAR(100),
    lemma       VARCHAR(100),
    pos_tag     VARCHAR(20),      -- часть речи
    frequency   FLOAT,
    text_id     INT REFERENCES texts(text_id)
);

-- Таблица авторов
CREATE TABLE authors (
    author_id   INT PRIMARY KEY,
    full_name   VARCHAR(150),
    nationality VARCHAR(100),
    birth_year  INT
);
```

### Задание 1. Агрегатные функции

**Условие:** Проанализировать состав корпуса текстов.

**1.1.** Подсчитать общее количество текстов, среднее, максимальное и минимальное количество слов в тексте.

```sql
SELECT 
    COUNT(*)            AS total_texts,
    AVG(word_count)     AS avg_words,
    MAX(word_count)     AS max_words,
    MIN(word_count)     AS min_words,
    SUM(word_count)     AS total_words
FROM texts;
```

**Ожидаемый результат:**

| total_texts | avg_words | max_words | min_words | total_words |
|-------------|-----------|-----------|-----------|-------------|
| 120         | 1843      | 12500     | 150       | 221160      |

**1.2.** Подсчитать количество текстов по жанрам.

```sql
SELECT 
    genre,
    COUNT(*)        AS text_count,
    SUM(word_count) AS total_words,
    AVG(word_count) AS avg_words
FROM texts
GROUP BY genre
ORDER BY text_count DESC;
```

**1.3.** Найти жанры, в которых более 10 текстов.

```sql
SELECT 
    genre,
    COUNT(*) AS text_count
FROM texts
GROUP BY genre
HAVING COUNT(*) > 10
ORDER BY text_count DESC;
```

> **Обратите внимание:** `HAVING` используется для фильтрации по агрегатным функциям, а `WHERE` — для фильтрации до агрегации.

### Задание 2. Строковые функции

**Условие:** Обработать словарный состав корпуса.

**2.1.** Найти все слова длиннее 10 символов и вывести их в верхнем регистре.

```sql
SELECT 
    word_form,
    UPPER(word_form)        AS word_upper,
    LENGTH(word_form)       AS word_length
FROM words
WHERE LENGTH(word_form) > 10
ORDER BY word_length DESC;
```

**2.2.** Извлечь первые 3 символа каждого слова (префикс) и сгруппировать слова по префиксу.

```sql
SELECT 
    LEFT(lemma, 3)          AS prefix,
    COUNT(*)                AS word_count
FROM words
GROUP BY LEFT(lemma, 3)
HAVING COUNT(*) > 5
ORDER BY word_count DESC;
```

**2.3.** Найти слова, содержащие подстроку «лингв», и показать позицию вхождения.

```sql
SELECT 
    word_form,
    INSTR(word_form, 'лингв')   AS position,
    LENGTH(word_form)           AS total_length
FROM words
WHERE LOWER(word_form) LIKE '%лингв%';
```

**2.4.** Сформировать полное описание слова через конкатенацию.

```sql
SELECT 
    CONCAT(lemma, ' (', pos_tag, ') — частота: ', ROUND(frequency, 4)) AS word_description
FROM words
WHERE frequency > 0.01
ORDER BY frequency DESC
LIMIT 20;
```

**2.5.** Очистить словоформы от лишних пробелов и привести к нижнему регистру.

```sql
UPDATE words
SET word_form = LOWER(TRIM(word_form))
WHERE word_form != LOWER(TRIM(word_form));
```

### Задание 3. Числовые функции

**Условие:** Провести частотный анализ лексики.

**3.1.** Округлить частоты слов и разбить их по диапазонам.

```sql
SELECT 
    word_form,
    frequency,
    ROUND(frequency, 3)             AS freq_rounded,
    ROUND(frequency * 100, 2)       AS freq_percent
FROM words
ORDER BY frequency DESC
LIMIT 30;
```

**3.2.** Вычислить отклонение частоты каждого слова от средней частоты по корпусу.

```sql
SELECT 
    word_form,
    frequency,
    ABS(frequency - (SELECT AVG(frequency) FROM words))     AS deviation,
    ROUND(
        (frequency - (SELECT AVG(frequency) FROM words)) /
        (SELECT AVG(frequency) FROM words) * 100, 2
    )                                                       AS deviation_percent
FROM words
ORDER BY deviation DESC
LIMIT 20;
```

**3.3.** Использовать `FLOOR` и `CEIL` для категоризации слов по диапазонам длины.

```sql
SELECT 
    FLOOR(LENGTH(word_form) / 3) * 3       AS length_group_start,
    CEIL(LENGTH(word_form) / 3.0) * 3      AS length_group_end,
    COUNT(*)                                AS word_count
FROM words
GROUP BY FLOOR(LENGTH(word_form) / 3)
ORDER BY length_group_start;
```

### Задание 4. Функции работы с датами

**Условие:** Проанализировать хронологию текстов в корпусе.

**4.1.** Извлечь год и месяц публикации и сгруппировать тексты по годам.

```sql
SELECT 
    YEAR(pub_date)      AS pub_year,
    COUNT(*)            AS text_count,
    SUM(word_count)     AS total_words
FROM texts
WHERE pub_date IS NOT NULL
GROUP BY YEAR(pub_date)
ORDER BY pub_year;
```

**4.2.** Найти тексты, опубликованные за последние 5 лет.

```sql
SELECT 
    title,
    author,
    pub_date,
    DATEDIFF(NOW(), pub_date)   AS days_since_pub
FROM texts
WHERE pub_date >= DATE_SUB(NOW(), INTERVAL 5 YEAR)
ORDER BY pub_date DESC;
```

**4.3.** Вычислить возраст текста в годах.

```sql
SELECT 
    title,
    pub_date,
    FLOOR(DATEDIFF(NOW(), pub_date) / 365)  AS age_years
FROM texts
ORDER BY age_years DESC;
```

### Задание 5. Комбинирование функций (повышенной сложности)

**Условие:** Создать сводный аналитический отчёт по корпусу.

**5.1.** Составить рейтинг авторов с аналитикой по их текстам.

```sql
SELECT 
    author,
    COUNT(*)                            AS text_count,
    SUM(word_count)                     AS total_words,
    ROUND(AVG(word_count), 0)           AS avg_text_length,
    MAX(word_count)                     AS longest_text,
    MIN(pub_date)                       AS first_publication,
    MAX(pub_date)                       AS last_publication,
    CONCAT(
        MIN(YEAR(pub_date)), '–', 
        MAX(YEAR(pub_date))
    )                                   AS active_years
FROM texts
GROUP BY author
HAVING COUNT(*) >= 3
ORDER BY total_words DESC;
```

**5.2.** Найти наиболее употребимые существительные длиннее 6 символов.

```sql
SELECT 
    lemma,
    UPPER(LEFT(lemma, 1)) || LOWER(SUBSTRING(lemma, 2))    AS lemma_capitalized,
    LENGTH(lemma)                                           AS lemma_length,
    ROUND(AVG(frequency), 5)                               AS avg_frequency,
    COUNT(*)                                               AS occurrences
FROM words
WHERE pos_tag = 'NOUN'
  AND LENGTH(lemma) > 6
GROUP BY lemma
HAVING COUNT(*) > 1
ORDER BY avg_frequency DESC
LIMIT 25;
```

**5.3.** Сформировать статистику по частям речи.

```sql
SELECT 
    pos_tag                                         AS part_of_speech,
    COUNT(*)                                        AS token_count,
    COUNT(DISTINCT lemma)                           AS unique_lemmas,
    ROUND(AVG(LENGTH(word_form)), 2)                AS avg_word_length,
    ROUND(AVG(frequency), 6)                        AS avg_frequency,
    ROUND(COUNT(*) * 100.0 / SUM(COUNT(*)) OVER(), 2) AS percentage
FROM words
GROUP BY pos_tag
ORDER BY token_count DESC;
```
