# Лабораторная работа №1
## Выбор данных с помощью оператора SELECT
### Цель работы
Изучить основы языка SQL и научиться использовать оператор SELECT для выбора данных из таблиц базы данных.

### Теоретическая часть
1. Основы оператора SELECT
Оператор SELECT — основной инструмент для извлечения данных из базы данных. Базовый синтаксис:
```sql
SELECT столбец1, столбец2, ...
FROM имя_таблицы;
```
Для выбора всех столбцов используется символ *:
```sql
SELECT * FROM имя_таблицы;
```
2. Фильтрация данных с помощью WHERE

Условие WHERE позволяет отфильтровать строки по определённым критериям:
```sql
SELECT столбец1, столбец2
FROM имя_таблицы
WHERE условие;
```
Операторы сравнения: =, <> или !=, >, <, >=, <=

Логические операторы: AND, OR, NOT
3. Сортировка результатов

Оператор ORDER BY упорядочивает результаты:
```sql
SELECT * FROM имя_таблицы
ORDER BY столбец1 ASC, столбец2 DESC;
```
ASC — по возрастанию (по умолчанию), DESC — по убыванию.

4.Ограничение количества записей

Для ограничения числа выводимых строк:
```sql
SELECT * FROM имя_таблицы
LIMIT 10;
```
5. Использование псевдонимов

Псевдонимы делают запросы более читаемыми:
```sql
SELECT столбец1 AS псевдоним1, столбец2 AS псевдоним2
FROM имя_таблицы AS псевдоним_таблицы;
```
