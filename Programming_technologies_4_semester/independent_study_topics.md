#### Базы данных и SQL

Начните с основ реляционных БД и SQL-запросов.

Видео охватывает теорию нормальных форм, создание таблиц и базовые команды SELECT/INSERT.

Полный вводный курс по PostgreSQL с установкой и связями таблиц: https://www.youtube.com/watch?v=HVQNxdI6fqY.
