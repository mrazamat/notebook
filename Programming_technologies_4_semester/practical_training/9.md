## Введение

JSON (JavaScript Object Notation) является универсальным форматом для хранения и обмена структурированными данными. В компьютерной лингвистике JSON широко используется для организации текстовых корпусов, хранения морфологической информации, аннотации текстов и работы с лингвистическими базами данных.

- Обрабатывать текстовые данные из JSON
- Сохранять результаты обработки в JSON

## Теоретический материал

### Основы JSON

JSON состоит из следующих типов данных:
- **Строки:** заключены в двойные кавычки `"текст"`
- **Числа:** целые или с плавающей точкой `123`, `45.67`
- **Логические значения:** `true`, `false`
- **Пустое значение:** `null`
- **Массивы:** упорядоченные списки в квадратных скобках `[элемент1, элемент2]`
- **Объекты:** неупорядоченные пары ключ-значение в фигурных скобках `{"ключ": "значение"}`

### Модуль json в Python

Python предоставляет встроенный модуль `json` для работы с JSON-данными:

```python
import json

# Основные функции:
# json.load(file) - загрузить JSON из файла
# json.loads(string) - загрузить JSON из строки
# json.dump(data, file) - сохранить JSON в файл
# json.dumps(data) - преобразовать объект в JSON-строку
```

### Соответствие типов Python и JSON

| Python     | JSON   |
|------------|--------|
| dict       | object |
| list       | array  |
| str        | string |
| int, float | number |
| True       | true   |
| False      | false  |
| None       | null   |

```python
def calculate_text_statistics(json_data):
    """
    Рассчитывает статистику по текстам из JSON.
    
    Args:
        json_data: загруженные данные из JSON
        
    Returns:
        dict: словарь со статистикой
    """
    texts = json_data['texts']
    total_words = sum(text['word_count'] for text in texts)
    avg_words = total_words / len(texts)
    
    return {
        'total_texts': len(texts),
        'total_words': total_words,
        'average_words': round(avg_words, 2),
        'min_words': min(text['word_count'] for text in texts),
        'max_words': max(text['word_count'] for text in texts)
    }

# Использование функции
with open('texts.json', 'r', encoding='utf-8') as file:
    data = json.load(file)

stats = calculate_text_statistics(data)
print("Статистика текстов:")
for key, value in stats.items():
    print(f"{key}: {value}")
```

```python
def count_word_frequency(json_data):
    """
    Подсчитывает частоту каждого слова во всех текстах.
    
    Args:
        json_data: загруженные данные из JSON
        
    Returns:
        dict: словарь с частотой слов
    """
    from collections import Counter
    import re
    
    texts = json_data['texts']
    all_words = []
    
    for text in texts:
        # Получаем слова из содержания текста
        words = re.findall(r'\w+', text['content'].lower())
        all_words.extend(words)
    
    word_freq = Counter(all_words)
    return dict(word_freq)

# Использование функции
word_freq = count_word_frequency(data)
print("Частота слов:")
for word, freq in sorted(word_freq.items(), key=lambda x: x[1], reverse=True):
    print(f"{word}: {freq}")
```

```python
def filter_texts_by_word_count(json_data, min_words, max_words):
    """
    Фильтрует тексты по количеству слов.
    
    Args:
        json_data: загруженные данные из JSON
        min_words: минимальное количество слов
        max_words: максимальное количество слов
        
    Returns:
        list: отфильтрованные тексты
    """
    texts = json_data['texts']
    filtered = [
        text for text in texts 
        if min_words <= text['word_count'] <= max_words
    ]
    return filtered

# Использование функции
with open('texts.json', 'r', encoding='utf-8') as file:

# Найти тексты с 5-10 словами
filtered_texts = filter_texts_by_word_count(data, 5, 10)
print(f"Найдено текстов: {len(filtered_texts)}")
for text in filtered_texts:
    print(f"- {text['title']} ({text['word_count']} слов)")
```

```python
    """
    Добавляет дополнительные метаданные к текстам.
    
    Args:
        json_data: загруженные данные из JSON
        
    Returns:
        dict: данные с дополнительными полями
    """
    import re
    
    for text in json_data['texts']:
        # Добавляем количество символов
        text['char_count'] = len(text['content'])
        
        # Добавляем количество предложений
        sentences = re.split(r'[.!?]+', text['content'])
        text['sentence_count'] = len([s for s in sentences if s.strip()])
        
        # Добавляем средний размер слова
        words = re.findall(r'\w+', text['content'])
        avg_word_length = sum(len(w) for w in words) / len(words) if words else 0
        text['avg_word_length'] = round(avg_word_length, 2)
    
    return json_data
```

```python
def extract_lemmas_from_corpus(json_data):
    """
    Извлекает все леммы из аннотированного корпуса.
    
    Args:
        json_data: загруженные данные корпуса
        
    Returns:
        list: список всех лемм
    """
    lemmas = []
    
    for document in json_data['documents']:
        for sentence in document['sentences']:
            for token in sentence['tokens']:
                lemmas.append(token['lemma'])
    
    return lemmas

# Использование функции
with open('annotated_corpus.json', 'r', encoding='utf-8') as file:
    corpus = json.load(file)

all_lemmas = extract_lemmas_from_corpus(corpus)
print("Все леммы в корпусе:", all_lemmas)
```
