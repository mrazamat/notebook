## Теоретическая часть

### Что такое SQLite?

SQLite — это встроенная реляционная база данных, которая хранится в одном файле на диске. Она не требует установки отдельного сервера и идеально подходит для учебных проектов и небольших приложений.

В Python работа с SQLite обеспечивается стандартным модулем `sqlite3` — дополнительная установка не нужна.

### Зачем использовать SQLite вместо JSON?

| Критерий         | JSON-файл            | SQLite                      |
|------------------|----------------------|-----------------------------|
| Поиск по полю    | Перебор всех записей | SQL-запрос с индексом       |
| Фильтрация       | Вручную в коде       | `WHERE`, `LIKE`, `ORDER BY` |
| Объём данных     | Ограничен памятью    | Миллионы записей            |
| Целостность      | Нет проверок         | Типы, ограничения, ключи    |
| Несколько таблиц | Сложно               | Естественно (`JOIN`)        |

Для словарной базы компьютерного лингвиста SQLite — оптимальный выбор: можно хранить слова, морфологию, примеры употребления и переводы в связанных таблицах.

### Основные понятия

**База данных** — организованное хранилище данных в виде таблиц.

**Таблица** — набор строк (записей) и столбцов (полей).

**Первичный ключ (PRIMARY KEY)** — уникальный идентификатор каждой строки.

**SQL** — язык запросов для работы с реляционными базами данных.

### Схема словарной базы

В этой лабораторной работе мы создадим одну таблицу `words`:

```
words
─────────────────────────────────────────────────
id          INTEGER   PRIMARY KEY AUTOINCREMENT
word        TEXT      NOT NULL UNIQUE
translation TEXT      NOT NULL
pos         TEXT      (часть речи / part of speech)
example     TEXT
created_at  TEXT      (дата добавления)
─────────────────────────────────────────────────
```

### Основные SQL-команды

```sql
-- Создание таблицы
CREATE TABLE IF NOT EXISTS words (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    word        TEXT NOT NULL UNIQUE,
    translation TEXT NOT NULL,
    pos         TEXT,
    example     TEXT,
    created_at  TEXT
);

-- Вставка записи
INSERT INTO words (word, translation, pos, example)
VALUES ('кот', 'cat', 'noun', 'Кот спит на диване.');

-- Выборка всех записей
SELECT * FROM words;

-- Поиск по слову
SELECT * FROM words WHERE word = 'кот';

-- Фильтрация по части речи
SELECT word, translation FROM words WHERE pos = 'noun';

-- Обновление записи
UPDATE words SET translation = 'tomcat' WHERE word = 'кот';

-- Удаление записи
DELETE FROM words WHERE word = 'кот';
```

### Использование `with` для подключения

Конструкция `with` автоматически фиксирует транзакцию при успехе или откатывает при ошибке:

```python
import sqlite3

with sqlite3.connect("vocabulary.db") as conn:
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM words")
    rows = cursor.fetchall()
```

### Защита от SQL-инъекций

Никогда не подставляйте данные пользователя в строку запроса напрямую. Используйте параметры (`?`):

```python
# НЕПРАВИЛЬНО — уязвимость SQL-инъекции
cursor.execute(f"SELECT * FROM words WHERE word = '{user_input}'")

# ПРАВИЛЬНО — параметризованный запрос
cursor.execute("SELECT * FROM words WHERE word = ?", (user_input,))
```

# search.py
import sqlite3

DB_FILE = "vocabulary.db"

def search_by_prefix(prefix):
    """Finds all words starting with the given prefix."""
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("""
            SELECT word, translation, pos
            FROM words
            WHERE word LIKE ?
            ORDER BY word
        """, (prefix + "%",))
        return cursor.fetchall()

def search_by_pos(pos):
    """Returns all words of the specified part of speech."""
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("""
            SELECT word, translation
            FROM words
            WHERE pos = ?
            ORDER BY word
        """, (pos,))
        return cursor.fetchall()

def search_by_translation(keyword):
    """Finds words whose translation contains the keyword."""
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("""
            SELECT word, translation, pos
            FROM words
            WHERE translation LIKE ?
        """, ("%" + keyword + "%",))
        return cursor.fetchall()

def full_text_search(keyword):
    """Searches across word, translation, and example fields."""
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("""
            SELECT word, translation, pos
            FROM words
            WHERE word LIKE ?
               OR translation LIKE ?
               OR example LIKE ?
        """, (f"%{keyword}%", f"%{keyword}%", f"%{keyword}%"))
        return cursor.fetchall()

# stats.py
import sqlite3

DB_FILE = "vocabulary.db"

def get_stats():
    """Prints statistics about the vocabulary database."""
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()

        # Total word count
        cursor.execute("SELECT COUNT(*) FROM words")
        total = cursor.fetchone()[0]

        # Count per part of speech
        cursor.execute("""
            SELECT pos, COUNT(*) as count
            FROM words
            GROUP BY pos
            ORDER BY count DESC
        """)
        pos_stats = cursor.fetchall()

        # Longest word
        cursor.execute("""
            SELECT word, LENGTH(word) as len
            FROM words
            ORDER BY len DESC
            LIMIT 1
        """)
        longest = cursor.fetchone()

        # Words added today
        cursor.execute("""
            SELECT COUNT(*) FROM words
            WHERE created_at = date('now')
        """)
        today_count = cursor.fetchone()[0]

    print(f"Total words in database: {total}")
    print(f"Words added today: {today_count}")
    print(f"\nDistribution by part of speech:")
    for pos, count in pos_stats:
        bar = "█" * count
        print(f"  {(pos or 'unknown'):<15} {count:>3}  {bar}")
    if longest:
        print(f"\nLongest word: '{longest[0]}' ({longest[1]} characters)")

# app.py
import sqlite3
import os

DB_FILE = "vocabulary.db"

def init_db():
    with sqlite3.connect(DB_FILE) as conn:
        conn.execute("""
            CREATE TABLE IF NOT EXISTS words (
                id          INTEGER PRIMARY KEY AUTOINCREMENT,
                word        TEXT NOT NULL UNIQUE,
                translation TEXT NOT NULL,
                pos         TEXT,
                example     TEXT,
                created_at  TEXT DEFAULT (date('now'))
            )
        """)

def add_word(word, translation, pos="", example=""):
    try:
        with sqlite3.connect(DB_FILE) as conn:
            conn.execute(
                "INSERT INTO words (word, translation, pos, example) VALUES (?, ?, ?, ?)",
                (word, translation, pos, example)
            )
        print(f"  ✓ Word '{word}' added.")
    except sqlite3.IntegrityError:
        print(f"  ✗ Word '{word}' already exists.")

def find_word(word):
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("SELECT * FROM words WHERE word = ?", (word,))
        return cursor.fetchone()

def list_all():
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("SELECT word, translation, pos FROM words ORDER BY word")
        return cursor.fetchall()

def delete_word(word):
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("DELETE FROM words WHERE word = ?", (word,))
        return cursor.rowcount > 0

def get_total():
    with sqlite3.connect(DB_FILE) as conn:
        cursor = conn.cursor()
        cursor.execute("SELECT COUNT(*) FROM words")
        return cursor.fetchone()[0]

def show_menu():
    print("\n╔══════════════════════════╗")

    print("╠══════════════════════════╣")

    print("║ 5. Show statistics       ║")
    print("║ 6. Exit                  ║")
    print("╚══════════════════════════╝")

                print(f"  Example:     {entry[4] or '—'}")
                print(f"  Added:       {entry[5] or '—'}")
            else:
                print(f"  Word '{word}' not found.")
