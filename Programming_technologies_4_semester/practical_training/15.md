## Теоретический материал

### Что такое Flask?

Flask — это микрофреймворк для создания веб-приложений на Python. Он позволяет быстро разрабатывать приложения, обрабатывать HTTP-запросы, работать с маршрутами и шаблонами.

**Основные компоненты Flask:**

1. Маршруты (routes) — определяют обработку URL-адресов
2. Шаблоны (templates) — HTML-файлы с поддержкой Jinja2
3. Статические файлы (static) — CSS, JavaScript, изображения
4. Контекстные переменные — передача данных между приложением и шаблонами

### Архитектура веб-сервиса

Веб-сервис для анализа текста состоит из следующих компонентов:

1. **Интерфейс пользователя** — HTML-форма для ввода текста
2. **Обработчик запроса** — Python-функция для получения текста
3. **Модуль анализа** — функции для обработки и анализа текста
4. **Отображение результатов** — HTML-страница с выводом результатов

### Методы анализа текста

В проекте мы реализуем следующие методы анализа:

1. **Подсчёт статистики:**
   - Количество символов
   - Количество слов
   - Количество предложений
   - Среднее количество символов в слове

2. **Анализ лексики:**
   - Уникальные слова
   - Плотность текста (уникальные слова / всего слов)
   - Частотность слов
   - Самые часто встречающиеся слова

3. **Морфологический анализ:**
   - Определение частей речи
   - Выделение лемм (начальных форм)

### Этап 3: Создание основного приложения

Создайте файл `app.py`:

```python
from flask import Flask, render_template, request, redirect, url_for
import re
from collections import Counter

app = Flask(__name__)

def analyze_text(text):
    """
    Функция для анализа текста.
    Возвращает словарь с результатами анализа.
    """
    # Очистка текста
    clean_text = text.strip()
    
    if not clean_text:
        return None
    
    # Подсчёт базовой статистики
    char_count = len(clean_text)
    char_count_no_spaces = len(clean_text.replace(" ", ""))
    
    # Разделение на слова
    words = re.findall(r'\b\w+\b', clean_text.lower())
    word_count = len(words)
    
    # Разделение на предложения
    sentences = re.split(r'[.!?]+', clean_text)
    sentences = [s.strip() for s in sentences if s.strip()]
    sentence_count = len(sentences)
    
    # Расчёт средних показателей
    avg_chars_per_word = char_count_no_spaces / word_count if word_count > 0 else 0
    avg_words_per_sentence = word_count / sentence_count if sentence_count > 0 else 0
    
    # Подсчёт уникальных слов
    unique_words = set(words)
    unique_word_count = len(unique_words)
    lexical_diversity = (unique_word_count / word_count * 100) if word_count > 0 else 0
    
    # Расчёт частоты слов
    word_frequency = Counter(words)
    top_words = word_frequency.most_common(10)
    
    # Формирование результата
    analysis_result = {
        'original_text': text,
        'char_count': char_count,
        'char_count_no_spaces': char_count_no_spaces,
        'word_count': word_count,
        'sentence_count': sentence_count,
        'unique_word_count': unique_word_count,
        'avg_chars_per_word': round(avg_chars_per_word, 2),
        'avg_words_per_sentence': round(avg_words_per_sentence, 2),
        'lexical_diversity': round(lexical_diversity, 2),
        'top_words': top_words
    }
    
    return analysis_result

@app.route('/')
def index():
    """Маршрут для главной страницы"""
    return render_template('index.html')

@app.route('/analyze', methods=['POST'])
def analyze():
    """Маршрут для обработки анализа текста"""
    input_text = request.form.get('text_input', '')
    
    if not input_text.strip():
        return redirect(url_for('index'))
    
    analysis_result = analyze_text(input_text)
    return render_template('results.html', result=analysis_result)

@app.route('/new')
def new_analysis():
    """Маршрут для новой аналитики"""
    return redirect(url_for('index'))

if __name__ == '__main__':
    app.run(debug=True)
```

```python
def check_text_quality(text, word_count, unique_word_count):
    """
    Функция для оценки качества текста.
    Возвращает оценку от 1 до 5.
    """
    lexical_diversity = (unique_word_count / word_count * 100) if word_count > 0 else 0
    
    if lexical_diversity < 30:
        return 1  # Низкое качество
    elif lexical_diversity < 50:
        return 2
    elif lexical_diversity < 70:
        return 3
    elif lexical_diversity < 85:
        return 4
    else:
        return 5  # Высокое качество
```

```python
def analyze_file():
    """Анализ текста из загруженного файла"""
    if 'file' not in request.files:
        return redirect(url_for('index'))
    
    file = request.files['file']
    
    if file.filename == '':
        return redirect(url_for('index'))
    
    if file and file.filename.endswith('.txt'):
        text = file.read().decode('utf-8')
        analysis_result = analyze_text(text)
        return render_template('results.html', result=analysis_result)
    
    return redirect(url_for('index'))
```
