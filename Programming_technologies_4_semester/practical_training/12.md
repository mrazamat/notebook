## Теоретическая часть

### Что такое машинный перевод?

Машинный перевод — это процесс автоматического преобразования текста с одного языка на другой. Существует несколько подходов:

**1. Статистический машинный перевод (SMT)**
- Использует вероятностные модели
- Требует больших параллельных корпусов текстов
- Анализирует часто встречающиеся переводы

**2. Нейронный машинный перевод (NMT)**
- Использует глубокие нейронные сети
- Лучше понимает контекст
- Более современный подход (Google Translate, DeepL)

**3. Словарный перевод (Dictionary-based)**
- Самый простой метод
- Использует словари эквивалентов
- Игнорирует контекст

### Доступные API для перевода

**MyMemory Translation API** (бесплатный, без ключа)
- URL: `https://api.mymemory.translated.net/get`
- Параметры: `q=текст`, `langpair=source|target`
- Пример: `langpair=en|ru` для перевода с английского на русский

**Google Translate** (требует ключ API)
- Платный сервис
- Высокое качество перевода
- Поддерживает много языков

**Yandex Translate** (требует ключ API)
- Хороший перевод русского языка
- Платный сервис

### Задача 1: Создание базового словаря для перевода

**Описание:** Реализуйте простой модуль перевода, основанный на словаре.

**Файл: translation_dictionary.py**

```python
class DictionaryTranslator:
    """Простой переводчик на основе словаря"""
    
    def __init__(self):
        # Словари перевода для разных пар языков
        self.dictionaries = {
            'en_ru': {
                'hello': 'привет',
                'world': 'мир',
                'goodbye': 'до свидания',
                'thank': 'спасибо',
                'yes': 'да',
                'no': 'нет',
                'please': 'пожалуйста',
                'water': 'вода',
                'book': 'книга',
                'computer': 'компьютер'
            },
            'ru_en': {
                'привет': 'hello',
                'мир': 'world',
                'до свидания': 'goodbye',
                'спасибо': 'thank',
                'да': 'yes',
                'нет': 'no',
                'пожалуйста': 'please',
                'вода': 'water',
                'книга': 'book',
                'компьютер': 'computer'
            }
        }
    
    def translate_word(self, word, source_lang, target_lang):
        """Переводит одно слово"""
        language_pair = f"{source_lang}_{target_lang}"
        dictionary = self.dictionaries.get(language_pair, {})
        
        word_lower = word.lower()
        translation = dictionary.get(word_lower, word)
        
        return translation
    
    def translate_text(self, text, source_lang, target_lang):
        """Переводит текст по словам"""
        words = text.split()
        translated_words = []
        
        for word in words:
            translated = self.translate_word(word, source_lang, target_lang)
            translated_words.append(translated)
        
        return ' '.join(translated_words)
    
    def get_supported_languages(self):
        """Возвращает поддерживаемые пары языков"""
        languages = set()
        for pair in self.dictionaries.keys():
            source, target = pair.split('_')
            languages.add((source, target))
        return list(languages)
```

### Задача 2: Работа с MyMemory API

**Описание:** Создайте модуль для работы с бесплатным API перевода.

**Файл: translation_api.py**

```python
import requests
import json
from typing import Dict, Optional

class APITranslator:
    """Переводчик с использованием MyMemory API"""
    
    BASE_URL = "https://api.mymemory.translated.net/get"
    
    # Коды языков для API
    LANGUAGE_CODES = {
        'en': 'en',
        'ru': 'ru',
        'es': 'es',
        'fr': 'fr',
        'de': 'de',
        'zh': 'zh-CN',
        'ja': 'ja',
        'ar': 'ar'
    }
    
    @staticmethod
    def translate_text(text: str, source_lang: str, target_lang: str) -> Dict:
        """
        Переводит текст с помощью MyMemory API
        
        Args:
            text: Текст для перевода
            source_lang: Код исходного языка
            target_lang: Код целевого языка
        
        Returns:
            Словарь с результатом перевода
        """
        try:
            # Проверка поддерживаемых языков
            if source_lang not in APITranslator.LANGUAGE_CODES:
                return {
                    'success': False,
                    'error': f'Язык {source_lang} не поддерживается'
                }
            
            if target_lang not in APITranslator.LANGUAGE_CODES:
                return {
                    'success': False,
                    'error': f'Язык {target_lang} не поддерживается'
                }
            
            # Формирование параметров запроса
            source_code = APITranslator.LANGUAGE_CODES[source_lang]
            target_code = APITranslator.LANGUAGE_CODES[target_lang]
            
            params = {
                'q': text,
                'langpair': f'{source_code}|{target_code}'
            }
            
            # Отправка запроса
            response = requests.get(APITranslator.BASE_URL, params=params)
            
            if response.status_code != 200:
                return {
                    'success': False,
                    'error': f'Ошибка API: статус {response.status_code}'
                }
            
            data = response.json()
            
            # Обработка ответа
            if data['responseStatus'] != 200:
                return {
                    'success': False,
                    'error': 'API вернул ошибку'
                }
            
            translated_text = data['responseData']['translatedText']
            
            return {
                'success': True,
                'original_text': text,
                'translated_text': translated_text,
                'source_language': source_lang,
                'target_language': target_lang
            }
        
        except requests.exceptions.RequestException as e:
            return {
                'success': False,
                'error': f'Ошибка сети: {str(e)}'
            }
        except KeyError as e:
            return {
                'success': False,
                'error': f'Ошибка парсинга ответа: {str(e)}'
            }
    
    @staticmethod
    def get_supported_languages():
        """Возвращает список поддерживаемых языков"""
        return list(APITranslator.LANGUAGE_CODES.keys())
```

### Задача 3: Flask приложение с интерфейсом

**Описание:** Создайте Flask-приложение, которое объединяет оба метода перевода.

**Файл: app.py**

```python
from flask import Flask, render_template, request, jsonify
from translation_api import APITranslator
from translation_dictionary import DictionaryTranslator
import sqlite3
from datetime import datetime
import os

app = Flask(__name__)
app.config['DATABASE'] = 'translation_history.db'

# Инициализация переводчиков
api_translator = APITranslator()
dict_translator = DictionaryTranslator()

# Функции для работы с БД
def init_db():
    """Инициализация базы данных"""
    if not os.path.exists(app.config['DATABASE']):
        conn = sqlite3.connect(app.config['DATABASE'])
        cursor = conn.cursor()
        
        cursor.execute('''
            CREATE TABLE translations (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                original_text TEXT NOT NULL,
                translated_text TEXT NOT NULL,
                source_language TEXT NOT NULL,
                target_language TEXT NOT NULL,
                translation_method TEXT NOT NULL,
                timestamp DATETIME DEFAULT CURRENT_TIMESTAMP
            )
        ''')
        
        conn.commit()
        conn.close()

def save_translation(original_text, translated_text, source_lang, target_lang, method):
    """Сохраняет перевод в историю"""
    try:
        conn = sqlite3.connect(app.config['DATABASE'])
        cursor = conn.cursor()
        
        cursor.execute('''
            INSERT INTO translations 
            (original_text, translated_text, source_language, target_language, translation_method)
            VALUES (?, ?, ?, ?, ?)
        ''', (original_text, translated_text, source_lang, target_lang, method))
        
        conn.commit()
        conn.close()
        return True
    except Exception as e:
        print(f'Ошибка при сохранении: {str(e)}')
        return False

def get_translation_history(limit=10):
    """Получает историю переводов"""
    try:
        conn = sqlite3.connect(app.config['DATABASE'])
        conn.row_factory = sqlite3.Row
        cursor = conn.cursor()
        
        cursor.execute('''
            SELECT * FROM translations 
            ORDER BY timestamp DESC 
            LIMIT ?
        ''', (limit,))
        
        history = [dict(row) for row in cursor.fetchall()]
        conn.close()
        
        return history
    except Exception as e:
        print(f'Ошибка при получении истории: {str(e)}')
        return []

# Маршруты приложения
@app.route('/')
def index():
    """Главная страница"""
    supported_languages = api_translator.get_supported_languages()
    return render_template('index.html', languages=supported_languages)

@app.route('/api/translate', methods=['POST'])
def translate():
    """API маршрут для перевода"""
    try:
        data = request.get_json()
        
        # Валидация входных данных
        if not data or 'text' not in data:
            return jsonify({'success': False, 'error': 'Текст не предоставлен'}), 400
        
        text = data.get('text', '').strip()
        source_lang = data.get('source_language', 'en')
        target_lang = data.get('target_language', 'ru')
        method = data.get('method', 'api')  # 'api' или 'dictionary'
        
        # Проверка длины текста
        if len(text) == 0:
            return jsonify({'success': False, 'error': 'Текст не может быть пустым'}), 400
        
        if len(text) > 500:
            return jsonify({'success': False, 'error': 'Текст слишком длинный (макс 500 символов)'}), 400
        
        # Выбор метода перевода
        if method == 'dictionary':
            translated_text = dict_translator.translate_text(text, source_lang, target_lang)
            save_translation(text, translated_text, source_lang, target_lang, 'dictionary')
            
            return jsonify({
                'success': True,
                'original_text': text,
                'translated_text': translated_text,
                'method': 'dictionary',
                'note': 'Использован словарный метод'
            })
        
        else:  # Использование API
            result = api_translator.translate_text(text, source_lang, target_lang)
            
            if result['success']:
                save_translation(
                    text,
                    result['translated_text'],
                    source_lang,
                    target_lang,
                    'api'
                )
            
            return jsonify(result)
    
    except Exception as e:
        return jsonify({
            'success': False,
            'error': f'Ошибка сервера: {str(e)}'
        }), 500

@app.route('/api/history')
def history():
    """API маршрут для получения истории переводов"""
    try:
        limit = request.args.get('limit', 10, type=int)
        history_data = get_translation_history(limit)
        
        return jsonify({
            'success': True,
            'count': len(history_data),
            'history': history_data
        })
    except Exception as e:
        return jsonify({
            'success': False,
            'error': f'Ошибка: {str(e)}'
        }), 500

@app.route('/api/clear-history', methods=['POST'])
def clear_history():
    """API маршрут для очистки истории"""
    try:
        conn = sqlite3.connect(app.config['DATABASE'])
        cursor = conn.cursor()
        cursor.execute('DELETE FROM translations')
        conn.commit()
        conn.close()
        
        return jsonify({'success': True, 'message': 'История очищена'})
    except Exception as e:
        return jsonify({
            'success': False,
            'error': f'Ошибка: {str(e)}'
        }), 500

@app.route('/api/supported-languages')
def supported_languages():
    """API маршрут для получения поддерживаемых языков"""
    languages = api_translator.get_supported_languages()
    return jsonify({
        'success': True,
        'languages': languages
    })

if __name__ == '__main__':
    init_db()
    app.run(debug=True)
```

                    <textarea id="translated-text" placeholder="Здесь появится перевод..." readonly></textarea>
                </div>
            </div>
            
            <div class="button-group">
                <button class="translate-btn" id="translate-btn">Перевести</button>
                <button class="clear-btn" id="clear-btn">Очистить</button>
            </div>
        </div>
        
        <div class="history-section">
            <h2>История переводов</h2>
            <div class="button-group" style="margin-bottom: 20px;">
                <button class="clear-btn" id="clear-history-btn">Очистить историю</button>
            </div>
            <div class="history-list" id="history-list">
                <div class="no-history">История переводов пуста</div>
            </div>
        </div>
    </div>
    
    <script>
        // Утилиты
        const showMessage = (message, type = 'info') => {
            const messageEl = document.getElementById('status-message');
            messageEl.textContent = message;
            messageEl.className = `status-message ${type}`;
        };
        
        const loadHistory = async () => {
            try {
                const response = await fetch('/api/history?limit=10');
                const data = await response.json();
                
                const historyList = document.getElementById('history-list');
                
                if (!data.success || data.count === 0) {
                    historyList.innerHTML = '<div class="no-history">История переводов пуста</div>';
                    return;
                }
                
                historyList.innerHTML = data.history.map(item => `
                    <div class="history-item">
                        <div class="history-item-text">
                            <strong>Оригинал:</strong> ${escapeHtml(item.original_text)}<br>
                            <strong>Перевод:</strong> ${escapeHtml(item.translated_text)}
                        </div>
                        <div class="history-item-meta">
                            ${item.source_language} → ${item.target_language} | Метод: ${item.translation_method}
                        </div>
                    </div>
                `).join('');
            } catch (error) {
                console.error('Ошибка при загрузке истории:', error);
            }
        };
        
        const escapeHtml = (text) => {
            const div = document.createElement('div');
            div.textContent = text;
            return div.innerHTML;
        };
        
        // Обработчики событий
        document.getElementById('translate-btn').addEventListener('click', async () => {
            const originalText = document.getElementById('original-text').value.trim();
            const sourceLang = document.getElementById('source-language').value;
            const targetLang = document.getElementById('target-language').value;
            const method = document.querySelector('input[name="translation-method"]:checked').value;
            
            if (!originalText) {
                showMessage('Пожалуйста, введите текст для перевода', 'error');
                return;
            }
            
            try {
                showMessage('Переводится... ' + '<span class="loading-spinner"></span>', 'loading');
                
                const response = await fetch('/api/translate', {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json'
                    },
                    body: JSON.stringify({
                        text: originalText,
                        source_language: sourceLang,
                        target_language: targetLang,
                        method: method
                    })
                });
                
                const data = await response.json();
                
                if (data.success) {
                    document.getElementById('translated-text').value = data.translated_text;
                    showMessage('Перевод выполнен успешно!', 'success');
                    loadHistory();
                } else {
                    showMessage(`Ошибка: ${data.error}`, 'error');
                }
            } catch (error) {
                showMessage(`Ошибка сети: ${error.message}`, 'error');
            }
        });
        
        document.getElementById('clear-btn').addEventListener('click', () => {
            document.getElementById('original-text').value = '';
            document.getElementById('translated-text').value = '';
            showMessage('', '');
        });
        
        document.getElementById('clear-history-btn').addEventListener('click', async () => {
            if (!confirm('Вы уверены, что хотите очистить историю?')) {
                return;
            }
            
            try {
                const response = await fetch('/api/clear-history', {
                    method: 'POST'
                });
                
                const data = await response.json();
                
                if (data.success) {
                    loadHistory();
                    showMessage('История очищена', 'success');
                } else {
                    showMessage(`Ошибка: ${data.error}`, 'error');
                }
            } catch (error) {
                showMessage(`Ошибка: ${error.message}`, 'error');
            }
        });
        
        // Инициализация
        document.addEventListener('DOMContentLoaded', loadHistory);
    </script>
</body>
