## Часть 1. Теория

### 1.1 NLP-библиотеки в веб-приложениях

NLP (Natural Language Processing — обработка естественного языка) в веб-приложениях позволяет:
- Анализировать текст, введённый пользователем
- Выполнять морфологический анализ
- Извлекать именованные сущности
- Классифицировать текст
- Выделять ключевые слова

### 1.2 Основные NLP-библиотеки

**NLTK (Natural Language Toolkit)**
- Набор инструментов для обработки естественного языка
- Хорошо подходит для начинающих
- Включает токенизацию, лемматизацию, POS-теговку
- Простая интеграция в Flask

**spaCy**
- Современная и быстрая библиотека
- Специализирована на извлечении информации
- Более производительная, чем NLTK
- Имеет встроенные модели для разных языков

### 1.3 Архитектура Flask-приложения с NLP

Типичное приложение состоит из:

```
app/
├── app.py              # Главный файл приложения
├── requirements.txt    # Зависимости
└── templates/
    └── index.html      # HTML-шаблон
```

Схема взаимодействия:
- Пользователь заполняет форму в браузере
- Flask получает текст через POST-запрос
- Серверная часть обрабатывает текст с помощью NLP-библиотеки
- Результаты анализа возвращаются в браузер
- Результаты отображаются пользователю

### Упражнение 1. Простое Flask-приложение с токенизацией

**Задача:** Создайте приложение, которое разбивает текст на слова и предложения.

**Файл: app.py**

```python
from flask import Flask, render_template, request, jsonify
import nltk
from nltk.tokenize import word_tokenize, sent_tokenize

app = Flask(__name__)

# Загрузка необходимых ресурсов NLTK
nltk.download('punkt')

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/tokenize', methods=['POST'])
def tokenize():
    data = request.get_json()
    text = data.get('text', '')
    
    if not text:
        return jsonify({'error': 'Text is required'}), 400
    
    # Токенизация на предложения
    sentences = sent_tokenize(text)
    
    # Токенизация на слова
    words = word_tokenize(text)
    
    result = {
        'sentences': sentences,
        'sentence_count': len(sentences),
        'words': words,
        'word_count': len(words)
    }
    
    return jsonify(result)

if __name__ == '__main__':
    app.run(debug=True)
```

**Файл: templates/index.html**

```html
<!DOCTYPE html>
<html>
<head>
    <title>Tokenization Analyzer</title>
    <style>
        body { font-family: Arial; margin: 20px; }
        textarea { width: 100%; height: 150px; }
        button { padding: 10px 20px; }
        .result { margin-top: 20px; padding: 10px; border: 1px solid #ccc; }
    </style>
</head>
<body>
    <h1>Text Tokenization Tool</h1>
    <textarea id="textInput" placeholder="Enter your text here..."></textarea>
    <button onclick="analyzeText()">Analyze</button>
    
    <div id="result" class="result" style="display:none;">
        <h3>Results:</h3>
        <p><strong>Sentences:</strong> <span id="sentenceCount"></span></p>
        <div id="sentences"></div>
        <p><strong>Words:</strong> <span id="wordCount"></span></p>
        <div id="words"></div>
    </div>

    <script>
        function analyzeText() {
            const text = document.getElementById('textInput').value;
            
            fetch('/tokenize', {
                method: 'POST',
                headers: {'Content-Type': 'application/json'},
                body: JSON.stringify({text: text})
            })
            .then(response => response.json())
            .then(data => {
                document.getElementById('sentenceCount').textContent = data.sentence_count;
                document.getElementById('sentences').innerHTML = 
                    data.sentences.map(s => '<p>• ' + s + '</p>').join('');
                
                document.getElementById('wordCount').textContent = data.word_count;
                document.getElementById('words').innerHTML = 
                    '<p>' + data.words.join(', ') + '</p>';
                
                document.getElementById('result').style.display = 'block';
            });
        }
    </script>
</body>
</html>
```

**Результат выполнения:**
- Приложение принимает текст от пользователя
- Разбивает текст на предложения и слова
- Отображает количество предложений и слов
- Показывает список всех слов и предложений

### Упражнение 2. Морфологический анализ с POS-теггингом

**Задача:** Создайте приложение для определения частей речи (Part-of-Speech tagging).

**Файл: app.py (добавьте новый маршрут)**

```python
from nltk import pos_tag, word_tokenize
from nltk.corpus import wordnet
from nltk.stem import WordNetLemmatizer

nltk.download('averaged_perceptron_tagger')
nltk.download('wordnet')

lemmatizer = WordNetLemmatizer()

@app.route('/analyze', methods=['POST'])
def analyze():
    data = request.get_json()
    text = data.get('text', '')
    
    if not text:
        return jsonify({'error': 'Text is required'}), 400
    
    # Токенизация
    tokens = word_tokenize(text)
    
    # POS-теггинг
    pos_tags = pos_tag(tokens)
    
    # Лемматизация
    lemmas = []
    for word, pos in pos_tags:
        lemma = lemmatizer.lemmatize(word.lower())
        lemmas.append({
            'word': word,
            'pos': pos,
            'lemma': lemma
        })
    
    result = {
        'tokens': [
            {
                'word': word,
                'pos': pos,
                'lemma': lemmas[i]['lemma']
            }
            for i, (word, pos) in enumerate(pos_tags)
        ]
    }
    
    return jsonify(result)
```

### Упражнение 3. Анализ частотности слов

**Задача:** Создайте приложение для подсчёта частотности слов в тексте.

**Файл: app.py (добавьте новый маршрут)**

```python
from collections import Counter
import string

@app.route('/frequency', methods=['POST'])
def frequency():
    data = request.get_json()
    text = data.get('text', '')
    
    if not text:
        return jsonify({'error': 'Text is required'}), 400
    
    # Очистка текста
    text_cleaned = text.lower()
    
    # Удаление пунктуации
    translator = str.maketrans('', '', string.punctuation)
    text_cleaned = text_cleaned.translate(translator)
    
    # Токенизация
    words = text_cleaned.split()
    
    # Подсчёт частотности
    word_freq = Counter(words)
    
    # Сортировка по частотности
    sorted_freq = sorted(word_freq.items(), key=lambda x: x[1], reverse=True)
    
    result = {
        'frequency': [
            {'word': word, 'count': count}
            for word, count in sorted_freq[:20]  # Топ 20 слов
        ],
        'unique_words': len(word_freq),
        'total_words': len(words)
    }
    
    return jsonify(result)
```

### Упражнение 4. Работа со spaCy

**Задача:** Создайте приложение для извлечения именованных сущностей (Named Entity Recognition — NER).

**Установка модели:**

```bash
python -m spacy download en_core_web_sm
```

**Файл: app.py (добавьте новый маршрут)**

```python
import spacy

# Загрузка модели spaCy
nlp = spacy.load('en_core_web_sm')

@app.route('/entities', methods=['POST'])
def entities():
    data = request.get_json()
    text = data.get('text', '')
    
    if not text:
        return jsonify({'error': 'Text is required'}), 400
    
    # Обработка текста
    doc = nlp(text)
    
    # Извлечение сущностей
    named_entities = [
        {
            'text': ent.text,
            'label': ent.label_,
            'start': ent.start_char,
            'end': ent.end_char
        }
        for ent in doc.ents
    ]
    
    # Извлечение зависимостей
    dependencies = [
        {
            'head': token.head.text,
            'child': token.text,
            'relation': token.dep_
        }
        for token in doc
        if token.head != token
    ]
    
    result = {
        'entities': named_entities,
        'dependencies': dependencies
    }
    
    return jsonify(result)
```

**Файл: templates/index.html (добавьте новую кнопку)**

```html
    <h3>Syntactic Dependencies:</h3>

        <tr style="background-color:#f0f0f0;">
            <th>Head Word</th>

                <tr>
                    <td>${dep.head}</td>
                    <td>${dep.child}</td>

                </tr>
            `).join('');

        });
    }

```
