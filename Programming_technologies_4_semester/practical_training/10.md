* Отслеживать тенденции в корпусах текстов

## Теоретическая часть

### Основные библиотеки для визуализации

#### Matplotlib

Matplotlib – это универсальная библиотека для создания статических, анимированных и интерактивных визуализаций на Python. Основной модуль `pyplot` предоставляет интерфейс, подобный MATLAB.

Основные функции:
* `plt.plot()` – создание линейных графиков
* `plt.bar()` – создание столбчатых диаграмм
* `plt.pie()` – создание круговых диаграмм
* `plt.scatter()` – создание точечных графиков
* `plt.hist()` – создание гистограмм
* `plt.show()` – отображение графика

#### WordCloud

Облако слов (word cloud) – это визуальное представление текста, где размер каждого слова соответствует его частоте появления в тексте. Библиотека `wordcloud` позволяет легко создавать такие облака.

#### Collections.Counter

Встроенный модуль `collections` содержит класс `Counter`, который удобен для подсчета частоты элементов. Это часто используется при предварительной обработке текстовых данных.

### Типы графиков в лингвистике

**Гистограмма (Histogram)** – показывает распределение длины слов или предложений в тексте.

**Столбчатая диаграмма (Bar Chart)** – отображает частоту конкретных слов или элементов.

**Линейный график (Line Plot)** – показывает изменение характеристик текста со временем или по главам.

**Круговая диаграмма (Pie Chart)** – демонстрирует процентное соотношение различных категорий (части речи, стили письма и т.д.).

**Облако слов (Word Cloud)** – визуализирует наиболее частые слова в тексте.

## Практическая часть

### Задача 1: Анализ частоты слов

**Описание:** Создайте программу, которая подсчитывает частоту слов в тексте и визуализирует результаты в виде столбчатой диаграммы.

**Текст для анализа:**

```
The quick brown fox jumps over the lazy dog. 
The dog sleeps under the tree.
The fox is quick and clever.
The tree is very old.
```

**Решение:**

```python
import matplotlib.pyplot as plt
from collections import Counter
import re

text = """The quick brown fox jumps over the lazy dog. 
The dog sleeps under the tree.
The fox is quick and clever.
The tree is very old."""

# Преобразование в нижний регистр и удаление пунктуации
words = re.findall(r'\b\w+\b', text.lower())

# Подсчет частоты слов
word_freq = Counter(words)

# Получение 10 самых частых слов
top_words = dict(word_freq.most_common(10))

# Создание столбчатой диаграммы
plt.figure(figsize=(10, 6))
plt.bar(top_words.keys(), top_words.values(), color='steelblue')
plt.xlabel('Words', fontsize=12)
plt.ylabel('Frequency', fontsize=12)
plt.title('Word Frequency Distribution', fontsize=14, fontweight='bold')
plt.xticks(rotation=45, ha='right')
plt.tight_layout()
plt.show()
```

**Объяснение кода:**

1. Импортируем необходимые библиотеки
2. Используем регулярное выражение для извлечения слов
3. Преобразуем текст в нижний регистр для унификации
4. Используем `Counter` для подсчета частоты каждого слова
5. Извлекаем 10 самых частых слов с помощью `most_common()`
6. Создаем столбчатую диаграмму с помощью `plt.bar()`
7. Устанавливаем подписи осей и заголовок
8. Поворачиваем подписи на оси X для лучшей читаемости

### Задача 2: Распределение длины слов

**Описание:** Создайте гистограмму, которая показывает распределение длины слов в тексте.

**Решение:**

```python
import matplotlib.pyplot as plt
import re

text = """The quick brown fox jumps over the lazy dog. 
The dog sleeps under the tree.
The fox is quick and clever.
The tree is very old."""

# Извлечение слов
words = re.findall(r'\b\w+\b', text.lower())

# Получение длины каждого слова
word_lengths = [len(word) for word in words]

# Создание гистограммы
plt.figure(figsize=(10, 6))
plt.hist(word_lengths, bins=range(min(word_lengths), max(word_lengths)+2), 
         color='coral', edgecolor='black', alpha=0.7)
plt.xlabel('Word Length (characters)', fontsize=12)
plt.ylabel('Frequency', fontsize=12)
plt.title('Distribution of Word Lengths', fontsize=14, fontweight='bold')
plt.xticks(range(min(word_lengths), max(word_lengths)+1))
plt.grid(axis='y', alpha=0.3)
plt.tight_layout()
plt.show()
```

**Объяснение кода:**

1. Извлекаем слова из текста
2. Создаем список длин слов с помощью list comprehension
3. Используем `plt.hist()` с параметром `bins` для определения количества интервалов
4. Параметр `edgecolor='black'` добавляет границы к столбцам
5. `alpha=0.7` делает столбцы слегка прозрачными
6. `grid()` добавляет сетку для лучшей читаемости

### Задача 4: Круговая диаграмма

**Описание:** Создайте круговую диаграмму, которая показывает распределение коротких (1-3 символа), средних (4-6 символов) и длинных (7+ символов) слов.

**Решение:**

```python
import matplotlib.pyplot as plt
import re

text = """The quick brown fox jumps over the lazy dog. 
The dog sleeps under the tree.
The fox is quick and clever.
The tree is very old."""

# Извлечение слов
words = re.findall(r'\b\w+\b', text.lower())

# Категоризация слов по длине
short_words = sum(1 for word in words if len(word) <= 3)
medium_words = sum(1 for word in words if 4 <= len(word) <= 6)
long_words = sum(1 for word in words if len(word) >= 7)

# Данные для круговой диаграммы
sizes = [short_words, medium_words, long_words]
labels = ['Short (1-3)', 'Medium (4-6)', 'Long (7+)']
colors = ['#ff9999', '#66b3ff', '#99ff99']
explode = (0.05, 0.05, 0.05)

# Создание круговой диаграммы
plt.figure(figsize=(8, 6))
plt.pie(sizes, explode=explode, labels=labels, colors=colors,
        autopct='%1.1f%%', shadow=True, startangle=90)
plt.title('Distribution of Word Lengths by Category', 
          fontsize=14, fontweight='bold', pad=20)
plt.axis('equal')
plt.tight_layout()
plt.show()
```

**Объяснение кода:**

1. Подсчитываем количество слов в каждой категории
2. Используем `plt.pie()` для создания круговой диаграммы
3. Параметр `explode` отделяет сегменты от центра
4. Параметр `autopct='%1.1f%%'` показывает процентное значение
5. `shadow=True` добавляет тень под диаграммой
6. `startangle=90` поворачивает диаграмму на 90 градусов

### Задача 5: Сравнение нескольких текстов

**Описание:** Сравните частоту некоторых слов в двух разных текстах, используя группированную столбчатую диаграмму.

**Решение:**

```python
import matplotlib.pyplot as plt
from collections import Counter
import re
import numpy as np

text1 = "the cat is on the mat the cat sleeps"
text2 = "the dog runs in the park the dog plays"

# Функция для получения частоты слов
def get_word_freq(text):
    words = re.findall(r'\b\w+\b', text.lower())
    return Counter(words)

freq1 = get_word_freq(text1)
freq2 = get_word_freq(text2)

# Получение всех уникальных слов
all_words = set(freq1.keys()) | set(freq2.keys())
all_words = sorted(list(all_words))

# Подготовка данных
counts1 = [freq1.get(word, 0) for word in all_words]
counts2 = [freq2.get(word, 0) for word in all_words]

# Создание группированной столбчатой диаграммы
x = np.arange(len(all_words))
width = 0.35

plt.figure(figsize=(12, 6))
plt.bar(x - width/2, counts1, width, label='Text 1', color='steelblue')
plt.bar(x + width/2, counts2, width, label='Text 2', color='coral')

plt.xlabel('Words', fontsize=12)
plt.ylabel('Frequency', fontsize=12)
plt.title('Word Frequency Comparison between Texts', 
          fontsize=14, fontweight='bold')
plt.xticks(x, all_words, rotation=45, ha='right')
plt.legend()
plt.tight_layout()
plt.show()
```

**Объяснение кода:**

1. Создаем функцию `get_word_freq()` для подсчета частоты слов
2. Объединяем все уникальные слова из обоих текстов
3. Используем `np.arange()` для создания позиций на оси X
4. Параметр `width` определяет ширину столбцов
5. Два вызова `plt.bar()` создают две группы столбцов
6. `plt.legend()` добавляет легенду для различения текстов

### Задача 6: Линейный график для анализа тенденций

**Описание:** Создайте линейный график, который показывает изменение средней длины слова по мере развития текста (по главам или абзацам).

**Решение:**

```python
import matplotlib.pyplot as plt
import re

chapters = [
    "The quick brown fox jumps over the lazy dog.",
    "The dog sleeps under the tree.",
    "The fox is quick and clever.",
    "The tree is very old and tall.",
    "Beautiful creatures live in this forest."
]

# Функция для вычисления средней длины слова
def get_avg_word_length(text):
    words = re.findall(r'\b\w+\b', text.lower())
    if len(words) == 0:
        return 0
    return sum(len(word) for word in words) / len(words)

# Подсчет средней длины слова для каждого абзаца
avg_lengths = [get_avg_word_length(chapter) for chapter in chapters]
chapter_numbers = list(range(1, len(chapters) + 1))

# Создание линейного графика
plt.figure(figsize=(10, 6))
plt.plot(chapter_numbers, avg_lengths, marker='o', linestyle='-', 
         color='darkgreen', linewidth=2, markersize=8)
plt.xlabel('Chapter', fontsize=12)
plt.ylabel('Average Word Length', fontsize=12)
plt.title('Change in Average Word Length Across Chapters', 
          fontsize=14, fontweight='bold')
plt.grid(True, alpha=0.3)
plt.xticks(chapter_numbers)
plt.tight_layout()
plt.show()
```

**Объяснение кода:**

1. Определяем несколько "глав" (абзацев) текста
2. Создаем функцию `get_avg_word_length()` для расчета средней длины слова
3. Применяем эту функцию ко всем главам
4. Используем `plt.plot()` с параметрами:
   * `marker='o'` – добавляет точки на линию
   * `linestyle='-'` – стиль линии
   * `linewidth=2` – толщина линии
   * `markersize=8` – размер точек
5. `plt.grid()` добавляет сетку для лучшей читаемости

### Полезные параметры matplotlib

* `figsize=(width, height)` – размер фигуры в дюймах
* `dpi=100` – разрешение в точках на дюйм
* `alpha=0.7` – прозрачность (0 = полностью прозрачно, 1 = полностью непрозрачно)
* `fontsize=12` – размер шрифта
* `fontweight='bold'` – жирный шрифт
* `rotation=45` – поворот текста в градусах
* `grid=True` – включение сетки на графике

### Палитры цветов (colormap)

Для WordCloud можно использовать различные палитры:

* 'viridis' – фиолетово-желтая палитра
* 'plasma' – розово-желтая палитра
* 'inferno' – черно-желтая палитра
* 'cool' – голубая палитра
* 'hot' – красно-черная палитра

1. **Удаляйте стоп-слова:** Слова типа "the", "is", "a" часто не несут информации. Рассмотрите создание списка стоп-слов и их фильтрацию.

2. **Нормализуйте текст:** Всегда приводите текст к нижнему регистру и удаляйте пунктуацию для точного подсчета.

3. **Используйте подходящий размер выборки:** При анализе часто берут не все слова, а только топ-20 или топ-50, чтобы избежать переполненности графика.

4. **Выбирайте правильный тип графика:** 
   * Столбчатая диаграмма – для сравнения дискретных значений
   * Линейный график – для показа тенденций
   * Облако слов – для быстрого визуального обзора
   * Круговая диаграмма – для показа пропорций

5. **Добавляйте подписи и легенды:** Всегда включайте заголовок, подписи осей и легенду для ясности.

6. **Тестируйте на разных текстах:** Убедитесь, что ваша программа работает корректно на различных входных данных.
