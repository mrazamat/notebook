## Введение в проект

В этом практическом уроке вы создадите веб-приложение на базе Flask, которое будет выполнять функцию помощника онлайн-переводчика. Приложение позволит пользователям вводить текст на одном языке и получать его перевод на другой язык.

Основные компетенции, которые вы приобретете:

- Работа с веб-фреймворком Flask
- Интеграция с API переводов (например, MyMemory Translation API)
- Обработка HTTP-запросов и ответов
- Работа с HTML-формами и Jinja2 шаблонами
- Обработка ошибок в веб-приложениях
- Сохранение истории переводов



## Теоретические основы

### Что такое Flask?

Flask — это легкий веб-фреймворк на Python, предназначенный для создания веб-приложений. Он позволяет быстро разработать приложение с минимальными зависимостями.

Основные компоненты Flask:

1. **Маршруты (routes)** — это функции, которые обрабатывают HTTP-запросы
2. **Шаблоны (templates)** — HTML-файлы с переменными, которые заполняются данными
3. **Статические файлы** — CSS, JavaScript, изображения
4. **Контекст приложения** — данные, доступные для всего приложения

### Архитектура приложения "Помощник онлайн-переводчика"

Приложение будет состоять из следующих компонентов:

1. **Backend (Python + Flask)** — обрабатывает запросы, обращается к API переводов, сохраняет историю
2. **Frontend (HTML + Jinja2)** — интерфейс для ввода текста и отображения результатов
3. **API переводов** — внешний сервис для получения переводов (MyMemory API)
4. **База данных** — сохранение истории переводов в JSON или SQLite

### API MyMemory Translation

MyMemory Translation API — это бесплатный сервис для переводов, не требующий регистрации.

Базовый URL: `https://api.mymemory.translated.net/get`

Параметры запроса:

- `q` — текст для перевода
- `langpair` — пара языков в формате `en|ru` (откуда|куда)

Пример запроса:

```
https://api.mymemory.translated.net/get?q=Hello&langpair=en|ru
```

## Этап 2: Создание основного приложения Flask

### Шаг 1: Создание файла app.py

Создайте файл `app.py` с основной логикой приложения:

```python
from flask import Flask, render_template, request, jsonify
import requests
import json
import os
from datetime import datetime

app = Flask(__name__)

TRANSLATIONS_FILE = 'translations_history.json'
MYMEMORY_API_URL = 'https://api.mymemory.translated.net/get'

# Поддерживаемые языки
SUPPORTED_LANGUAGES = {
    'en': 'English',
    'ru': 'Русский',
    'es': 'Español',
    'fr': 'Français',
    'de': 'Deutsch',
    'it': 'Italiano',
    'pt': 'Português',
    'ja': '日本語'
}

def load_translation_history():
    """Загружает историю переводов из файла"""
    if os.path.exists(TRANSLATIONS_FILE):
        with open(TRANSLATIONS_FILE, 'r', encoding='utf-8') as f:
            return json.load(f)
    return []

def save_translation_history(history):
    """Сохраняет историю переводов в файл"""
    with open(TRANSLATIONS_FILE, 'w', encoding='utf-8') as f:
        json.dump(history, f, ensure_ascii=False, indent=2)

def translate_text(text, source_lang, target_lang):
    """Переводит текст с использованием MyMemory API"""
    try:
        langpair = f"{source_lang}|{target_lang}"
        params = {
            'q': text,
            'langpair': langpair
        }
        
        response = requests.get(MYMEMORY_API_URL, params=params)
        response.raise_for_status()
        
        data = response.json()
        
        if data['responseStatus'] == 200:
            translated_text = data['responseData']['translatedText']
            return {
                'success': True,
                'translated_text': translated_text,
                'match': data['responseData'].get('match', 0)
            }
        else:
            return {
                'success': False,
                'error': 'Ошибка перевода. Попробуйте позже.'
            }
    except requests.exceptions.RequestException as e:
        return {
            'success': False,
            'error': f'Ошибка подключения: {str(e)}'
        }

@app.route('/')
def index():
    """Главная страница приложения"""
    languages = SUPPORTED_LANGUAGES
    return render_template('index.html', languages=languages)

@app.route('/translate', methods=['POST'])
def translate():
    """Обрабатывает запрос на перевод"""
    data = request.get_json()
    
    text = data.get('text', '').strip()
    source_lang = data.get('source_lang', 'en')
    target_lang = data.get('target_lang', 'ru')
    
    if not text:
        return jsonify({
            'success': False,
            'error': 'Пожалуйста, введите текст для перевода'
        }), 400
    
    if len(text) > 500:
        return jsonify({
            'success': False,
            'error': 'Текст слишком длинный (максимум 500 символов)'
        }), 400
    
    result = translate_text(text, source_lang, target_lang)
    
    if result['success']:
        history = load_translation_history()
        history_entry = {
            'timestamp': datetime.now().strftime('%Y-%m-%d %H:%M:%S'),
            'source_text': text,
            'translated_text': result['translated_text'],
            'source_lang': source_lang,
            'target_lang': target_lang,
            'match_percentage': result['match'] * 100
        }
        history.insert(0, history_entry)
        
        if len(history) > 100:
            history = history[:100]
        
        save_translation_history(history)
    
    return jsonify(result)

@app.route('/history')
def history():
    """Отображает историю переводов"""
    history_data = load_translation_history()
    languages = SUPPORTED_LANGUAGES
    return render_template('history.html', 
                         history=history_data, 
                         languages=languages)

@app.route('/clear-history', methods=['POST'])
def clear_history():
    """Очищает историю переводов"""
    save_translation_history([])
    return jsonify({'success': True, 'message': 'История очищена'})

@app.route('/api/languages')
def get_languages():
    """Возвращает список поддерживаемых языков"""
    return jsonify(SUPPORTED_LANGUAGES)

if __name__ == '__main__':
    app.run(debug=True, host='127.0.0.1', port=5000)
```

<script>
    document.getElementById('source_text').addEventListener('input', function() {
        const count = this.value.length;
        document.getElementById('char_count').textContent = count;
    });

    document.getElementById('swap_button').addEventListener('click', function() {
        const source = document.getElementById('source_lang').value;
        const target = document.getElementById('target_lang').value;
        document.getElementById('source_lang').value = target;
        document.getElementById('target_lang').value = source;
    });

    document.getElementById('translate_button').addEventListener('click', function() {
        const source_text = document.getElementById('source_text').value;
        const source_lang = document.getElementById('source_lang').value;
        const target_lang = document.getElementById('target_lang').value;

        if (!source_text.trim()) {
            show_message('Пожалуйста, введите текст для перевода', 'error');
            return;
        }

        this.disabled = true;
        this.textContent = 'Переводим...';

        fetch('/translate', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
            },
            body: JSON.stringify({
                text: source_text,
                source_lang: source_lang,
                target_lang: target_lang
            })
        })
        .then(response => response.json())
        .then(data => {
            if (data.success) {
                document.getElementById('translated_text').value = data.translated_text;
                document.getElementById('match_percentage').textContent = Math.round(data.match_percentage);
                show_message('Перевод выполнен успешно!', 'success');
            } else {
                show_message(data.error, 'error');
            }
        })
        .catch(error => {
            show_message('Ошибка при отправке запроса', 'error');
            console.error('Error:', error);
        })
        .finally(() => {
            this.disabled = false;
            this.textContent = 'Перевести';
        });
    });

    document.getElementById('clear_button').addEventListener('click', function() {
        document.getElementById('source_text').value = '';
        document.getElementById('translated_text').value = '';
        document.getElementById('char_count').textContent = '0';
        document.getElementById('match_percentage').textContent = '0';
    });

    function show_message(message, type) {
        const message_box = document.getElementById('message_box');
        message_box.textContent = message;
        message_box.className = 'message-box ' + type;
        
        setTimeout(() => {
            message_box.textContent = '';
            message_box.className = 'message-box';
        }, 3000);
    }
</script>

<script>
    function copyToClipboard(text) {
        navigator.clipboard.writeText(text).then(() => {
            alert('Текст скопирован в буфер обмена');
        });
    }

    document.getElementById('clear_history_button').addEventListener('click', function() {
        if (confirm('Вы уверены, что хотите очистить историю переводов?')) {
            fetch('/clear-history', {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json',
                }
            })
            .then(response => response.json())
            .then(data => {
                location.reload();
            })
            .catch(error => {
                console.error('Error:', error);
                alert('Ошибка при очистке истории');
            });
        }
    });
</script>
