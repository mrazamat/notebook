## Теоретические сведения

### Что такое машинный перевод?

**Машинный перевод (МП)** — это автоматический перевод текста с одного естественного языка на другой с использованием программного обеспечения без участия человека-переводчика.

### Основные подходы к машинному переводу

#### 1. Правилосновный перевод (RBMT — Rule-Based Machine Translation)
Основан на лингвистических правилах и словарях. Система анализирует грамматику исходного языка и генерирует текст на целевом языке согласно заранее определённым правилам.

- **Плюсы:** предсказуемость, прозрачность.
- **Минусы:** трудоёмкость создания правил, плохая обработка идиом.

#### 2. Статистический перевод (SMT — Statistical Machine Translation)
Основан на вероятностных моделях, обученных на больших корпусах параллельных текстов. Модель учится находить наиболее вероятный перевод для каждого фрагмента.

- **Плюсы:** не требует ручного создания правил.
- **Минусы:** требует огромных объёмов данных, плохо обрабатывает контекст.

#### 3. Нейронный перевод (NMT — Neural Machine Translation)
Современный подход на основе нейронных сетей (архитектура Encoder-Decoder, Transformer). Модель обрабатывает весь контекст предложения целиком.

- **Плюсы:** высокое качество, учёт контекста, естественность перевода.
- **Минусы:** требует значительных вычислительных ресурсов.

> **Современные системы** — Google Translate, DeepL, Yandex.Translate — используют именно нейронный подход.

#### Задание 2.1 — Работа с Google Translate API (имитация)

Изучите следующий пример кода на Python, который демонстрирует простую систему словарного перевода. Разберите его, дополните и запустите.

```python
# lab8_translator.py
# Простая система машинного перевода на основе словаря

# Словарь переводов: русский -> английский
dictionary = {
    "кот": "cat",
    "собака": "dog",
    "дом": "house",
    "книга": "book",
    "программирование": "programming",
    "язык": "language",
    "перевод": "translation",
    "компьютер": "computer",
    "слово": "word",
    "текст": "text",
}

def translate_word(word: str) -> str:
    """Переводит одно слово с русского на английский."""
    word = word.lower().strip()
    return dictionary.get(word, f"[{word}: перевод не найден]")

def translate_sentence(sentence: str) -> str:
    """Переводит предложение слово за словом."""
    words = sentence.split()
    translated = [translate_word(w) for w in words]
    return " ".join(translated)

# --- Главная программа ---
if __name__ == "__main__":
    print("=== Лабораторный переводчик (RU -> EN) ===\n")

    test_sentences = [
        "кот дом книга",
        "программирование язык компьютер",
        "перевод текст слово",
    ]

    for sentence in test_sentences:
        result = translate_sentence(sentence)
        print(f"Оригинал : {sentence}")
        print(f"Перевод  : {result}")
        print("-" * 40)
```

# Пример: правило для перевода числа существительных
# Если слово заканчивается на "и" или "ы" — предполагаем множественное число

def detect_plural(word: str) -> bool:
    """Простое определение множественного числа в русском."""
    return word.endswith("и") or word.endswith("ы")

def translate_with_grammar(word: str) -> str:
    """Перевод с учётом числа."""
    plural = detect_plural(word)
    # Убираем окончание для поиска в словаре (упрощённо)
    base = word[:-1] if plural else word
    translation = dictionary.get(base.lower(), f"[{word}]")
    if plural and not translation.startswith("["):
        translation += "s"  # Простое правило английского множественного числа
    return translation
