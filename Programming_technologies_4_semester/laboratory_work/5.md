## Теоретическая часть

### Что такое API перевода?

**API перевода** (Application Programming Interface) — это программный интерфейс, который позволяет разработчикам интегрировать функции машинного перевода в свои приложения. Сервис принимает текст на одном языке и возвращает его перевод на другой язык.

### Популярные API перевода

| Сервис                   | Компания                | Бесплатный лимит              |
|--------------------------|-------------------------|-------------------------------|
| Google Cloud Translation | Google                  | 500 000 символов/месяц        |
| DeepL API                | DeepL                   | 500 000 символов/месяц        |
| LibreTranslate           | Свободный (Open Source) | Без ограничений (self-hosted) |
| MyMemory                 | Translated              | 1 000 символов/запрос         |

### Принцип работы

Взаимодействие с API происходит по следующей схеме:

```
Клиент  →  HTTP-запрос (текст + языки)  →  Сервер API
Клиент  ←  HTTP-ответ (JSON с переводом) ←  Сервер API
```

### Структура HTTP-запроса

Запрос к API, как правило, содержит:

- **URL** — адрес конечной точки (endpoint)
- **Метод** — чаще всего `GET` или `POST`
- **Параметры** — исходный текст, язык источника, язык перевода
- **API-ключ** — для аутентификации (если требуется)

### Пример ответа API (JSON)

```json
{
  "translations": [
    {
      "detected_source_language": "EN",
      "text": "Привет, мир!"
    }
  ]
}
```

### Задание 1. Использование MyMemory API (без регистрации)

MyMemory — бесплатный переводческий API, не требующий ключа для простых запросов.

**Базовый URL:**
```
https://api.mymemory.translated.net/get
```

**Параметры запроса:**

| Параметр   | Описание           | Пример        |
|------------|--------------------|---------------|
| `q`        | Текст для перевода | `Hello world` |
| `langpair` | Пара языков        | `en\|ru`      |

**Пример запроса в браузере:**
```
https://api.mymemory.translated.net/get?q=Hello+world&langpair=en|ru
```

**Пример ответа:**
```json
{
  "responseData": {
    "translatedText": "Привет мир",
    "match": 1
  },
  "responseStatus": 200,
  "responseDetails": ""
}
```

### Задание 2. Написание кода на Python

Создайте файл `translator.py` и реализуйте функцию перевода текста с помощью MyMemory API.

```python
import requests

def translate(text, source_lang, target_lang):
    """
    Переводит текст с помощью MyMemory API.
    
    :param text: текст для перевода
    :param source_lang: язык источника (например, 'en')
    :param target_lang: язык перевода (например, 'ru')
    :return: переведённый текст
    """
    url = "https://api.mymemory.translated.net/get"
    
    params = {
        "q": text,
        "langpair": f"{source_lang}|{target_lang}"
    }
    
    response = requests.get(url, params=params)
    
    if response.status_code == 200:
        data = response.json()
        translated_text = data["responseData"]["translatedText"]
        return translated_text
    else:
        return f"Ошибка: {response.status_code}"


# Основная программа
if __name__ == "__main__":
    # Пример 1: Английский → Русский
    result1 = translate("Good morning!", "en", "ru")
    print(f"Перевод: {result1}")

    # Пример 2: Русский → Немецкий
    result2 = translate("Добрый день", "ru", "de")
    print(f"Перевод: {result2}")

    # Пример 3: Испанский → Английский
    result3 = translate("Buenos días", "es", "en")
    print(f"Перевод: {result3}")
```

### Задание 3. Интерактивный переводчик

Доработайте программу: добавьте интерактивный ввод с консоли.

```python
import requests

def translate(text, source_lang, target_lang):
    url = "https://api.mymemory.translated.net/get"
    params = {
        "q": text,

def main():
    print("=== Переводчик на основе MyMemory API ===")
    print("Введите 'выход' для завершения работы.\n")
    
    while True:
        text = input("Введите текст: ")

    main()
```

### Задание 4. Обработка ошибок

Улучшите программу, добавив обработку возможных ошибок.

```python
import requests

def translate_safe(text, source_lang, target_lang):
    """Перевод текста с обработкой исключений."""
    
    if not text.strip():
        return "Ошибка: пустой текст"
    
    url = "https://api.mymemory.translated.net/get"
    params = {
        "q": text,
        "langpair": f"{source_lang}|{target_lang}"
    }
    
    try:
        response = requests.get(url, params=params, timeout=10)
        response.raise_for_status()
        
        data = response.json()
        
        if data["responseStatus"] == 200:
            return data["responseData"]["translatedText"]
        else:
            return f"Ошибка API: {data.get('responseDetails', 'неизвестная ошибка')}"
    
    except requests.exceptions.Timeout:
        return "Ошибка: превышено время ожидания ответа"
    
    except requests.exceptions.ConnectionError:
        return "Ошибка: нет подключения к интернету"
    
    except requests.exceptions.HTTPError as e:
        return f"HTTP-ошибка: {e}"
    
    except Exception as e:
        return f"Непредвиденная ошибка: {e}"


# Тестирование
print(translate_safe("Hello!", "en", "ru"))
print(translate_safe("", "en", "ru"))           # пустой текст
print(translate_safe("Bonjour", "fr", "ru"))
```
