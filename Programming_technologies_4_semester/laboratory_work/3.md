## Теоретическая часть

### Что такое словарная база?

**Словарная база** — это структурированный набор лингвистических единиц (слов, словосочетаний, лемм) с набором атрибутов для каждой единицы. В компьютерной лингвистике словарные базы используются в задачах морфологического анализа, машинного перевода, проверки правописания и построения корпусов.

### Словарь (dict) в Python

В Python словарь (`dict`) — это встроенная структура данных, которая хранит пары **ключ–значение**. Ключи уникальны и неизменяемы.

```python
# Создание словаря
entry = {
    "слово": "бежать",
    "часть_речи": "глагол",
    "лемма": "бежать",
    "частота": 142
}
```

### Основные операции со словарями

| Операция             | Синтаксис        | Описание                     |
|----------------------|------------------|------------------------------|
| Доступ к значению    | `d[key]`         | Получить значение по ключу   |
| Добавление/изменение | `d[key] = value` | Добавить или обновить запись |
| Удаление             | `del d[key]`     | Удалить запись               |
| Проверка наличия     | `key in d`       | Вернёт True/False            |
| Перебор ключей       | `for k in d`     | Итерация по ключам           |
| Перебор пар          | `d.items()`      | Итерация по (ключ, значение) |
| Список ключей        | `d.keys()`       | Все ключи словаря            |
| Список значений      | `d.values()`     | Все значения словаря         |

### Задание 2. Поиск и получение информации

#### 2.1. Функция поиска слова

```python
def найти_слово(база, слово):
    """Возвращает информацию о слове или сообщение об отсутствии."""
    if слово in база:
        return база[слово]
    else:
        return f"Слово '{слово}' не найдено в базе."
```

#### 2.2. Функция получения всех слов по части речи

```python
def слова_по_части_речи(база, часть_речи):
    """Возвращает список слов заданной части речи."""
    результат = []
    for слово, данные in база.items():

    return результат
```

```python
def частые_слова(база, порог):
```

```python
def отсортировать_по_частоте(база):
    """Возвращает список слов, отсортированных по убыванию частоты."""
```

### Задание 5. Сохранение базы в файл

Реализуйте сохранение словарной базы в текстовый файл формата CSV и её загрузку обратно.

```python
import csv

def сохранить_в_csv(база, имя_файла):
    """Сохраняет словарную базу в CSV-файл."""
    with open(имя_файла, "w", encoding="utf-8", newline="") as f:
        writer = csv.writer(f)
        writer.writerow(["слово", "часть_речи", "лемма", "частота"])
        for слово, данные in база.items():
            writer.writerow([
                слово,
                данные["часть_речи"],
                данные["лемма"],
                данные["частота"]
            ])
    print(f"База сохранена в файл '{имя_файла}'.")


def загрузить_из_csv(имя_файла):
    """Загружает словарную базу из CSV-файла."""
    база = {}
    with open(имя_файла, "r", encoding="utf-8") as f:
        reader = csv.DictReader(f)
        for строка in reader:
            база[строка["слово"]] = {
                "часть_речи": строка["часть_речи"],
                "лемма": строка["лемма"],
                "частота": int(строка["частота"])
            }
    return база
```
