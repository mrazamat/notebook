## Теоретическая часть

### Что такое визуализация текста?

Визуализация текста — это способ графического представления текстовой информации с целью выявления закономерностей, частотности слов, тематических связей и других характеристик, которые сложно увидеть в «сыром» тексте.

Визуализация помогает:
- быстро оценить ключевые темы и слова документа;
- сравнивать несколько текстов между собой;
- выявлять наиболее часто встречающиеся термины;
- представлять результаты анализа в наглядной форме.

### Основные типы визуализации текста

| Тип визуализации             | Описание                                                            |
|------------------------------|---------------------------------------------------------------------|
| **Облако слов (Word Cloud)** | Слова отображаются разным шрифтом — чем чаще слово, тем оно крупнее |
| **Частотная гистограмма**    | Столбчатая диаграмма с топ-N самых частых слов                      |
| **Диаграмма Zipf**           | Демонстрирует закон Ципфа: зависимость частоты слова от его ранга   |
| **Тепловая карта**           | Визуализация матрицы совместной встречаемости слов                  |

### Используемые библиотеки

- **`nltk`** — токенизация, удаление стоп-слов, частотный анализ
- **`collections.Counter`** — подсчёт частоты слов
- **`matplotlib`** — построение диаграмм и графиков
- **`wordcloud`** — генерация облака слов

### Задание 2. Загрузка и предобработка текста

Создайте файл `lab9.py` и напишите функцию предобработки текста.

```python
import re
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

def preprocess(text, language='russian'):
    """
    Предобработка текста:
    - Приведение к нижнему регистру
    - Удаление знаков пунктуации и цифр
    - Токенизация
    - Удаление стоп-слов
    """
    # Привести к нижнему регистру
    text = text.lower()

    # Удалить всё, кроме букв и пробелов
    text = re.sub(r'[^а-яёa-z\s]', '', text)

    # Токенизация
    tokens = word_tokenize(text, language=language)

    # Загрузить стоп-слова
    stop_words = set(stopwords.words(language))

    # Убрать стоп-слова и короткие токены
    tokens = [word for word in tokens if word not in stop_words and len(word) > 2]

    return tokens


# Тестовый текст
sample_text = """
Компьютерная лингвистика — это область науки, которая изучает язык с помощью
компьютерных методов. Она объединяет лингвистику и информатику для создания
систем, способных понимать и обрабатывать естественный язык. Компьютерная
лингвистика применяется в машинном переводе, распознавании речи, анализе текста
и разработке чат-ботов. Современные технологии позволяют анализировать большие
объёмы текстовых данных и извлекать из них полезную информацию.
"""

tokens = preprocess(sample_text)
print("Токены:", tokens[:10])
print("Всего токенов:", len(tokens))
```

### Задание 3. Частотный анализ и гистограмма

Добавьте в файл `lab9.py` код для подсчёта частоты слов и построения гистограммы:

```python
from collections import Counter
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['font.family'] = 'DejaVu Sans'

def plot_frequency(tokens, top_n=10, title='Топ слов по частоте'):
    """
    Строит горизонтальную гистограмму топ-N самых частых слов.
    """
    # Подсчитать частоту
    freq = Counter(tokens)
    most_common = freq.most_common(top_n)

    words = [item[0] for item in most_common]
    counts = [item[1] for item in most_common]

    # Построить график
    plt.figure(figsize=(10, 6))
    plt.barh(words[::-1], counts[::-1], color='steelblue')
    plt.xlabel('Частота')
    plt.title(title)
    plt.tight_layout()
    plt.savefig('frequency_chart.png', dpi=150)
    plt.show()
    print(f"График сохранён: frequency_chart.png")

    return freq


freq = plot_frequency(tokens, top_n=10)
```

### Задание 5. Диаграмма Ципфа

Закон Ципфа гласит: частота слова обратно пропорциональна его рангу. Добавьте код:

```python
import numpy as np

def plot_zipf(freq, title="Закон Ципфа"):
    """
    Строит log-log график зависимости частоты от ранга (закон Ципфа).
    """
    # Отсортировать по убыванию частоты
    sorted_freq = sorted(freq.values(), reverse=True)
    ranks = np.arange(1, len(sorted_freq) + 1)

    plt.figure(figsize=(8, 5))
    plt.loglog(ranks, sorted_freq, marker='o', markersize=3, linestyle='-', color='tomato')
    plt.xlabel('Ранг слова (log)')
    plt.ylabel('Частота (log)')
    plt.title(title)
    plt.grid(True, which='both', linestyle='--', alpha=0.5)
    plt.tight_layout()
    plt.savefig('zipf_chart.png', dpi=150)
    plt.show()
    print("График Ципфа сохранён: zipf_chart.png")


plot_zipf(freq)
```

### Задание 6*. Сравнение двух текстов (дополнительное)

Проведите сравнительный частотный анализ двух текстов на русском языке (например, два абзаца из учебника или статьи).

```python
def compare_texts(text1, text2, label1='Текст 1', label2='Текст 2', top_n=8):
    """
    Строит сгруппированную гистограмму для сравнения двух текстов.
    """
    tokens1 = preprocess(text1)
    tokens2 = preprocess(text2)

    freq1 = Counter(tokens1)
    freq2 = Counter(tokens2)

    # Объединить топ-слова обоих текстов
    all_top = set(
        [w for w, _ in freq1.most_common(top_n)] +
        [w for w, _ in freq2.most_common(top_n)]
    )

    words = sorted(all_top)
    vals1 = [freq1.get(w, 0) for w in words]
    vals2 = [freq2.get(w, 0) for w in words]

    x = np.arange(len(words))
    width = 0.35

    plt.figure(figsize=(12, 6))
    plt.bar(x - width/2, vals1, width, label=label1, color='steelblue')
    plt.bar(x + width/2, vals2, width, label=label2, color='tomato')
    plt.xticks(x, words, rotation=45, ha='right')
    plt.ylabel('Частота')
    plt.title('Сравнение частотности слов')
    plt.legend()
    plt.tight_layout()
    plt.savefig('comparison_chart.png', dpi=150)
    plt.show()


# Вставьте свои тексты
text_a = "Ваш первый текст здесь..."
text_b = "Ваш второй текст здесь..."
compare_texts(text_a, text_b, label1='Текст A', label2='Текст B')
```
