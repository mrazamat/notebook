## Теоретические сведения

### Формат JSON

**JSON** (JavaScript Object Notation) — текстовый формат обмена данными, основанный на синтаксисе JavaScript. Широко используется в веб-приложениях, API и конфигурационных файлах.

**Основные типы данных JSON:**

| Тип    | Пример                          |
|--------|---------------------------------|
| Строка | `"привет"`                      |
| Число  | `42`, `3.14`                    |
| Булево | `true`, `false`                 |
| Null   | `null`                          |
| Массив | `["кот", "пёс", "птица"]`       |
| Объект | `{"слово": "кот", "род": "м."}` |

**Пример JSON-структуры (словарная статья):**

```json
{
  "слово": "лингвистика",
  "часть_речи": "существительное",
  "род": "женский",
  "определение": "Наука о языке и его закономерностях",
  "примеры": [
    "Лингвистика изучает языки мира.",
    "Компьютерная лингвистика — прикладная дисциплина."
  ]
}
```

---

### Формат XML

**XML** (eXtensible Markup Language) — расширяемый язык разметки, предназначенный для хранения и передачи структурированных данных. Активно применяется в лингвистических корпусах, аннотированных текстах и стандартах (например, TEI).

**Основные элементы XML:**

- **Теги** — открывающий `<тег>` и закрывающий `</тег>`
- **Атрибуты** — `<слово род="мужской">`
- **Корневой элемент** — обязательный единственный корень документа
- **Пролог** — `<?xml version="1.0" encoding="UTF-8"?>`

**Пример XML-структуры (словарная статья):**

```xml
<?xml version="1.0" encoding="UTF-8"?>
<словарь>
  <статья id="1">
    <слово>лингвистика</слово>
    <часть_речи>существительное</часть_речи>
    <род>женский</род>
    <определение>Наука о языке и его закономерностях</определение>
    <примеры>
      <пример>Лингвистика изучает языки мира.</пример>
      <пример>Компьютерная лингвистика — прикладная дисциплина.</пример>
    </примеры>
  </статья>
</словарь>
```

---

### Сравнение JSON и XML

| Характеристика      | JSON              | XML                            |
|---------------------|-------------------|--------------------------------|
| Читаемость          | Высокая           | Средняя                        |
| Объём данных        | Меньше            | Больше (теги дублируются)      |
| Поддержка атрибутов | Нет               | Да                             |
| Применение          | API, конфиги, веб | Корпуса, документы, разметка   |
| Парсинг в Python    | Модуль `json`     | Модуль `xml.etree.ElementTree` |

### Задание 1. Работа с JSON в Python

**Задача:** Создать JSON-файл с мини-словарём лингвистических терминов и вывести его содержимое.

```python
import json

# Создание словаря
glossary = {
    "термины": [
        {
            "слово": "морфема",
            "определение": "Минимальная значимая единица языка",
            "примеры": ["корень", "суффикс", "приставка"]
        },
        {
            "слово": "фонема",
            "определение": "Минимальная звуковая единица языка",
            "примеры": ["/а/", "/б/", "/в/"]
        },
        {
            "слово": "лексема",
            "определение": "Единица словарного состава языка",
            "примеры": ["бежать", "стол", "синий"]
        }
    ]
}

# Запись в файл
with open("glossary.json", "w", encoding="utf-8") as f:
    json.dump(glossary, f, ensure_ascii=False, indent=4)

print("Файл успешно создан.")

# Чтение из файла
with open("glossary.json", "r", encoding="utf-8") as f:
    data = json.load(f)

# Вывод содержимого
for term in data["термины"]:
    print(f"\nТермин: {term['слово']}")
    print(f"Определение: {term['определение']}")
    print(f"Примеры: {', '.join(term['примеры'])}")
```

**Ожидаемый вывод:**
```
Файл успешно создан.

Термин: морфема
Определение: Минимальная значимая единица языка
Примеры: корень, суффикс, приставка

Термин: фонема
Определение: Минимальная звуковая единица языка
Примеры: /а/, /б/, /в/

Термин: лексема
Определение: Единица словарного состава языка
Примеры: бежать, стол, синий
```

### Задание 2. Добавление новой записи в JSON

**Задача:** Прочитать существующий JSON-файл и добавить новый термин.

```python
import json

# Чтение существующего файла
with open("glossary.json", "r", encoding="utf-8") as f:
    data = json.load(f)

# Новый термин

print(f"Добавлен термин: {new_term['слово']}")
print(f"Всего терминов в словаре: {len(data['термины'])}")
```

### Задание 3. Работа с XML в Python

**Задача:** Создать XML-файл с аннотированным предложением и разобрать его.

```python
import xml.etree.ElementTree as ET

# Создание структуры XML
root = ET.Element("корпус")
предложение = ET.SubElement(root, "предложение", id="1")

# Добавление токенов с морфологической разметкой
токены = [
    {"форма": "Студент", "лемма": "студент", "pos": "NOUN", "падеж": "Nom"},
    {"форма": "изучает", "лемма": "изучать", "pos": "VERB", "время": "Present"},
    {"форма": "лингвистику", "лемма": "лингвистика", "pos": "NOUN", "падеж": "Acc"},
]

for i, токен in enumerate(токены, start=1):
    элемент = ET.SubElement(предложение, "токен", id=str(i))
    for ключ, значение in токен.items():
        дочерний = ET.SubElement(элемент, ключ)
        дочерний.text = значение

# Запись в файл
tree = ET.ElementTree(root)
ET.indent(tree, space="  ")  # красивое форматирование (Python 3.9+)
tree.write("corpus.xml", encoding="unicode", xml_declaration=True)

print("XML-файл создан.")

# Чтение и разбор файла
tree = ET.parse("corpus.xml")
root = tree.getroot()

print("\nАннотированное предложение:")
for предл in root.findall("предложение"):
    for токен in предл.findall("токен"):
        форма = токен.find("форма").text
        лемма = токен.find("лемма").text
        pos   = токен.find("pos").text
        print(f"  [{форма}] → лемма: {лемма}, POS: {pos}")
```

**Ожидаемый вывод:**
```
XML-файл создан.

Аннотированное предложение:
  [Студент] → лемма: студент, POS: NOUN
  [изучает] → лемма: изучать, POS: VERB
  [лингвистику] → лемма: лингвистика, POS: NOUN
```

### Задание 4. Поиск и фильтрация в XML

**Задача:** Найти все существительные (NOUN) в созданном корпусе.

```python
import xml.etree.ElementTree as ET

tree = ET.parse("corpus.xml")
root = tree.getroot()

print("Существительные в корпусе:")
for токен in root.iter("токен"):
    pos_elem = токен.find("pos")
    if pos_elem is not None and pos_elem.text == "NOUN":
        форма = токен.find("форма").text
        лемма = токен.find("лемма").text
        падеж_elem = токен.find("падеж")
        падеж = падеж_elem.text if падеж_elem is not None else "—"
        print(f"  Форма: {форма}, Лемма: {лемма}, Падеж: {падеж}")
```

**Ожидаемый вывод:**
```
Существительные в корпусе:
  Форма: Студент, Лемма: студент, Падеж: Nom
  Форма: лингвистику, Лемма: лингвистика, Падеж: Acc
```
