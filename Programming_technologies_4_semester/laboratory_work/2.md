## Теоретическая часть

### Что такое база данных?

**База данных (БД)** — организованная коллекция структурированной информации, хранящейся в компьютерной системе. Для управления базами данных используется **СУБД** (Система управления базами данных).

### SQLite

**SQLite** — это лёгкая встраиваемая реляционная СУБД, которая не требует отдельного сервера. Вся база данных хранится в одном файле на диске. Идеально подходит для учебных проектов и небольших приложений.

Основные понятия:
- **Таблица** — структура, в которой хранятся данные (строки и столбцы).
- **Запись (строка)** — один элемент данных в таблице.
- **Поле (столбец)** — отдельная характеристика записи.
- **SQL** — язык структурированных запросов для работы с базой данных.

### Основные SQL-команды

| Команда        | Описание              |
|----------------|-----------------------|
| `CREATE TABLE` | Создание таблицы      |
| `INSERT INTO`  | Добавление записи     |
| `SELECT`       | Чтение/выборка данных |
| `UPDATE`       | Обновление записи     |
| `DELETE`       | Удаление записи       |

### Шаг 2. Создание базы данных и таблицы

```python
import sqlite3
from datetime import datetime

# Подключение к базе данных (файл создаётся автоматически)
connection = sqlite3.connect("texts.db")
cursor = connection.cursor()

# Создание таблицы для хранения текстов
cursor.execute("""
    CREATE TABLE IF NOT EXISTS texts (
        id        INTEGER PRIMARY KEY AUTOINCREMENT,
        title     TEXT NOT NULL,
        content   TEXT NOT NULL,
        author    TEXT,
        created_at TEXT
    )
""")

connection.commit()
print("База данных и таблица успешно созданы.")
```

**Пояснение к полям таблицы:**

| Поле         | Тип     | Описание                                            |
|--------------|---------|-----------------------------------------------------|
| `id`         | INTEGER | Уникальный номер записи (заполняется автоматически) |
| `title`      | TEXT    | Заголовок текста                                    |
| `content`    | TEXT    | Содержимое текста                                   |
| `author`     | TEXT    | Автор текста                                        |
| `created_at` | TEXT    | Дата и время добавления                             |

def add_text(title, content, author):
    """Добавляет текст в базу данных."""

        INSERT INTO texts (title, content, author, created_at)

def read_all_texts():
    """Читает и выводит все тексты из базы данных."""
    cursor.execute("SELECT id, title, author, created_at FROM texts")
    rows = cursor.fetchall()

        print("База данных пуста.")
        return

def search_texts(keyword):
    """Ищет тексты по ключевому слову в заголовке или содержании."""
    cursor.execute("""
        SELECT id, title, author
        FROM texts
        WHERE title LIKE ? OR content LIKE ?
    """, (f"%{keyword}%", f"%{keyword}%"))
    rows = cursor.fetchall()

    print(f'\n=== Результаты поиска по слову "{keyword}" ===')
    if rows:
        for row in rows:
            print(f"[{row[0]}] {row[1]} — {row[2]}")

### Полный код программы

```python
import sqlite3
from datetime import datetime

# ── Подключение к БД ──────────────────────────────────────────
connection = sqlite3.connect("texts.db")
cursor = connection.cursor()

# ── Создание таблицы ──────────────────────────────────────────
cursor.execute("""
    CREATE TABLE IF NOT EXISTS texts (
        id         INTEGER PRIMARY KEY AUTOINCREMENT,
        title      TEXT NOT NULL,
        content    TEXT NOT NULL,
        author     TEXT,
        created_at TEXT
    )
""")
connection.commit()


# ── Функции ───────────────────────────────────────────────────
def add_text(title, content, author):
    created_at = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    cursor.execute(
        "INSERT INTO texts (title, content, author, created_at) VALUES (?, ?, ?, ?)",
        (title, content, author, created_at)
    )
    connection.commit()
    print(f'Добавлен: "{title}"')


def read_all_texts():
    cursor.execute("SELECT id, title, author, created_at FROM texts")
    rows = cursor.fetchall()
    print("\n=== Все тексты ===")
    for row in rows:
        print(f"[{row[0]}] {row[1]} | Автор: {row[2]} | {row[3]}")


def read_text_by_id(text_id):
    cursor.execute("SELECT * FROM texts WHERE id = ?", (text_id,))
    row = cursor.fetchone()
    if row:
        print(f"\n[#{row[0]}] {row[1]}\n{row[2]}")
    else:
        print(f"Текст #{text_id} не найден.")


def search_texts(keyword):
    cursor.execute(
        "SELECT id, title, author FROM texts WHERE title LIKE ? OR content LIKE ?",
        (f"%{keyword}%", f"%{keyword}%")
    )
    rows = cursor.fetchall()
    print(f'\n=== Поиск: "{keyword}" ===')
    for row in rows:
        print(f"[{row[0]}] {row[1]} — {row[2]}")


# ── Демонстрация ──────────────────────────────────────────────
add_text("Введение в лингвистику",
         "Лингвистика — наука о языке и речи.",
         "Иванов А.А.")

add_text("Что такое морфология",
         "Морфология изучает структуру слова и его формы.",
         "Петрова М.С.")

add_text("Синтаксис простыми словами",
         "Синтаксис изучает правила построения предложений.",
         "Иванов А.А.")

read_all_texts()
read_text_by_id(1)
search_texts("лингвистика")

# ── Закрытие соединения ───────────────────────────────────────
connection.close()
```
