## Теоретическая часть

**Flask** — лёгкий веб-фреймворк для Python, позволяющий быстро создавать веб-приложения. Он обрабатывает HTTP-запросы, управляет маршрутами и возвращает ответы пользователю.

Основные понятия:

| Понятие                 | Описание                                        |
|-------------------------|-------------------------------------------------|
| **Маршрут (route)**     | URL-адрес, с которым связана функция-обработчик |
| **GET-запрос**          | Запрос на получение данных (открытие страницы)  |
| **POST-запрос**         | Запрос на отправку данных (форма с текстом)     |
| **Шаблон (template)**   | HTML-файл с подстановкой данных через Jinja2    |
| **`render_template()`** | Функция Flask для отображения HTML-шаблона      |
| **`request.form`**      | Словарь с данными, отправленными из HTML-формы  |

Лингвистические метрики, вычисляемые в приложении:

| Метрика                | Способ вычисления                 |
|------------------------|-----------------------------------|
| Символы (с пробелами)  | `len(text)`                       |
| Символы (без пробелов) | `len(text.replace(' ', ''))`      |
| Количество слов        | `re.findall(r'\b\w+\b', text)`    |
| Количество предложений | Разбивка по `.`, `!`, `?`         |
| Средняя длина слова    | Сумма длин слов / количество слов |
| Топ-5 частотных слов   | `collections.Counter`             |

## Задание

Разработать веб-приложение на Flask, которое:

1. Отображает главную страницу с формой ввода текста.
2. Принимает текст через POST-запрос и выполняет его анализ.
3. Вычисляет и отображает:
   - количество символов (с пробелами и без);
   - количество слов;
   - количество предложений;
   - среднюю длину слова;
   - топ-5 наиболее частотных слов.
4. Отображает результаты на той же странице.

### Шаг 3. Создание файла `app.py`

Создайте файл `app.py` и добавьте в него следующий код:

```python
from flask import Flask, render_template, request
from collections import Counter
import re

app = Flask(__name__)


def analyze(text):
    """Выполняет лингвистический анализ переданного текста."""

    # Количество символов
    chars_all = len(text)
    chars_no_space = len(text.replace(' ', ''))

    # Список слов (только буквы, включая кириллицу)
    words = re.findall(r'[а-яёА-ЯЁa-zA-Z]+', text)
    word_count = len(words)

    # Количество предложений
    sentences = re.split(r'[.!?]+', text)
    sentence_count = len([s for s in sentences if s.strip()])

    # Средняя длина слова
    if word_count > 0:
        avg_word_len = round(sum(len(w) for w in words) / word_count, 2)
    else:
        avg_word_len = 0

    # Топ-5 частотных слов (без учёта регистра)
    words_lower = [w.lower() for w in words]
    top_words = Counter(words_lower).most_common(5)

    return {
        'chars_all': chars_all,
        'chars_no_space': chars_no_space,
        'word_count': word_count,
        'sentence_count': sentence_count,
        'avg_word_len': avg_word_len,
        'top_words': top_words,
    }


@app.route('/', methods=['GET', 'POST'])
def index():
    results = None
    text = ''

    if request.method == 'POST':
        text = request.form.get('text', '').strip()
        if text:
            results = analyze(text)

    return render_template('index.html', results=results, text=text)


if __name__ == '__main__':
    app.run(debug=True)
```
