**Предмет:** Технологии программирования  
**Специальность:** Компьютерная лингвистика  


---

## Цель работы

Разработать простое веб-приложение, выполняющее базовый лингвистический анализ текста: подсчёт слов, предложений, символов, определение наиболее частотных слов и отображение результатов в браузере.

---

## Теоретическая часть

**Веб-приложение** — программа, работающая в браузере и взаимодействующая с пользователем через интерфейс на основе HTML, CSS и JavaScript.

Основные технологии, применяемые в работе:

- **HTML (HyperText Markup Language)** — язык разметки для создания структуры страницы.
- **CSS (Cascading Style Sheets)** — язык стилей для оформления интерфейса.
- **JavaScript** — язык программирования для реализации логики приложения на стороне клиента.

Базовые лингвистические метрики текста:

| Метрика                | Описание                                      |
|------------------------|-----------------------------------------------|
| Количество символов    | Общее число символов в тексте                 |
| Количество слов        | Число токенов, разделённых пробелами          |
| Количество предложений | Число фраз, заканчивающихся на `.`, `!`, `?`  |
| Частотность слов       | Сколько раз каждое слово встречается в тексте |
| Средняя длина слова    | Среднее число символов на одно слово          |

---

## Задание

Разработать одностраничное лингвистическое веб-приложение, которое:

1. Принимает произвольный текст от пользователя через поле ввода.
2. Вычисляет и отображает следующие характеристики:
   - количество символов (с пробелами и без);
   - количество слов;
   - количество предложений;
   - среднюю длину слова;
   - топ-5 наиболее частотных слов.
3. Имеет кнопку **«Анализировать»** и кнопку **«Очистить»**.
4. Корректно отображается в современном браузере.

### Шаг 4. Написание JavaScript-логики

Вставьте следующий код внутрь тега `<script>`:

```javascript
function analyzeText() {
  const text = document.getElementById('inputText').value.trim();

  if (!text) {
    alert('Пожалуйста, введите текст для анализа.');
    return;
  }

  // 1. Подсчёт символов
  const charsAll = text.length;
  const charsNoSpace = text.replace(/\s/g, '').length;

  // 2. Подсчёт слов
  const words = text.match(/\b[\wа-яёА-ЯЁ]+\b/gi) || [];
  const wordCount = words.length;

  // 3. Подсчёт предложений
  const sentences = text.split(/[.!?]+/).filter(s => s.trim().length > 0);
  const sentenceCount = sentences.length;

  // 4. Средняя длина слова
  const totalLetters = words.reduce((sum, w) => sum + w.length, 0);
  const avgWordLen = wordCount > 0
    ? (totalLetters / wordCount).toFixed(2)
    : 0;

  // 5. Топ-5 частотных слов
  const freq = {};
  words.forEach(w => {
    const lower = w.toLowerCase();
    freq[lower] = (freq[lower] || 0) + 1;
  });

  const topWords = Object.entries(freq)
    .sort((a, b) => b[1] - a[1])
    .slice(0, 5);

  // Вывод результатов
  document.getElementById('charsAll').textContent = charsAll;
  document.getElementById('charsNoSpace').textContent = charsNoSpace;
  document.getElementById('wordCount').textContent = wordCount;
  document.getElementById('sentenceCount').textContent = sentenceCount;
  document.getElementById('avgWordLen').textContent = avgWordLen;

  const topList = document.getElementById('topWords');
  topList.innerHTML = '';
  topWords.forEach(([word, count]) => {
    const li = document.createElement('li');
    li.textContent = `«${word}» — ${count} раз(а)`;
    topList.appendChild(li);
  });

  document.getElementById('results').classList.remove('hidden');
}

function clearAll() {
  document.getElementById('inputText').value = '';
  document.getElementById('results').classList.add('hidden');
}
```
