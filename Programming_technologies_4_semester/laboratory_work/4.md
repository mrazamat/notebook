## Теоретическая часть

### Что такое Flask?

**Flask** — это лёгкий веб-фреймворк для Python, предназначенный для быстрой разработки веб-приложений. Он относится к категории **микрофреймворков**, так как не включает встроенных инструментов для работы с базами данных или валидации форм — всё это подключается при необходимости через расширения.

Flask основан на двух библиотеках:
- **Werkzeug** — инструментарий для работы с WSGI (интерфейс между веб-сервером и Python-приложением)
- **Jinja2** — шаблонизатор для генерации HTML-страниц

### Основные концепции Flask

| Понятие                  | Описание                                                       |
|--------------------------|----------------------------------------------------------------|
| **Маршрут (route)**      | URL-адрес, по которому доступна страница                       |
| **Представление (view)** | Функция Python, которая обрабатывает запрос и возвращает ответ |
| **Шаблон (template)**    | HTML-файл с динамическими вставками данных через Jinja2        |
| **Запрос (request)**     | Объект, содержащий данные входящего HTTP-запроса               |
| **Ответ (response)**     | Данные, которые Flask возвращает клиенту (браузеру)            |

### Структура простого Flask-приложения

```
my_project/
├── app.py              # Основной файл приложения
├── templates/          # Папка с HTML-шаблонами
│   ├── index.html
│   └── result.html
└── static/             # Статические файлы (CSS, JS, изображения)
    └── style.css
```

### Задание 2. Создание простого веб-приложения «Привет, мир!»

Создайте файл `app.py` со следующим содержимым:

```python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return '<h1>Привет, мир!</h1>'

if __name__ == '__main__':
    app.run(debug=True)
```

Запустите приложение:

```bash
python app.py
```

Откройте браузер и перейдите по адресу: `http://127.0.0.1:5000`

**Ожидаемый результат:** На странице отображается надпись «Привет, мир!»

### Задание 4. Использование шаблонов Jinja2

**Шаг 1.** Создайте папку `templates` и файл `templates/index.html`:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
</head>
<body>
    <h1>{{ heading }}</h1>
    <p>Добро пожаловать, {{ username }}!</p>

    <ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
    </ul>
</body>
</html>
```

**Шаг 2.** Обновите `app.py`:

```python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    data = {
        'title': 'Главная страница',
        'heading': 'Список языков программирования',
        'username': 'Студент',
        'items': ['Python', 'JavaScript', 'Java', 'C++']
    }
    return render_template('index.html', **data)

if __name__ == '__main__':
    app.run(debug=True)
```

**Ожидаемый результат:** Страница отображает заголовок, приветствие и маркированный список языков программирования.

### Задание 5. Обработка HTML-формы

**Шаг 1.** Создайте файл `templates/form.html`:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Форма</title>
</head>
<body>
    <h2>Введите слово для анализа</h2>
    <form method="POST" action="/analyze">
        <input type="text" name="word" placeholder="Введите слово" required>
        <button type="submit">Анализировать</button>
    </form>

    {% if result %}
        <hr>
        <h3>Результат:</h3>
        <p>Слово: <strong>{{ result.word }}</strong></p>
        <p>Длина: {{ result.length }} символов</p>
        <p>Символы в верхнем регистре: {{ result.upper }}</p>
    {% endif %}
</body>
</html>
```

**Шаг 2.** Добавьте маршруты в `app.py`:

```python
from flask import Flask, render_template, request

app = Flask(__name__)

@app.route('/analyze', methods=['GET', 'POST'])
def analyze():
    result = None
    if request.method == 'POST':
        word = request.form.get('word', '')
        result = {
            'word': word,
            'length': len(word),
            'upper': word.upper()
        }
    return render_template('form.html', result=result)

if __name__ == '__main__':
    app.run(debug=True)
```

Перейдите по адресу `http://127.0.0.1:5000/analyze`, введите любое слово и нажмите кнопку.

**Ожидаемый результат:** После отправки формы на той же странице появляется информация о введённом слове: его длина и вариант в верхнем регистре.
