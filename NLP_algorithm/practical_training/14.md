## Введение

Семантика текста — это область NLP, которая изучает смысл слов, фраз и предложений. Вместо анализа только синтаксической структуры, семантический анализ стремится понять, *что именно* означает текст.

**Зачем нам нужна векторизация текста?**

Компьютеры работают с числами, а не со словами. Чтобы применить математические методы машинного обучения к тексту, нам нужно преобразовать слова и документы в **векторы** (последовательности чисел), которые сохраняют семантическую информацию.

**Простая аналогия:** Представьте, что каждое слово — это точка в многомерном пространстве. Слова с похожим смыслом находятся близко друг к другу, а слова с разным смыслом — далеко друг от друга.


## Основные концепции

### Что такое вектор слова?

Вектор слова — это последовательность чисел фиксированной длины, которая представляет слово в пространстве признаков.

```
word = "король"
vector = [0.25, -0.48, 0.75, 0.12, ..., -0.33]  # размерность: 300
```

### Косинусное сходство

Для измерения семантической близости двух слов используется **косинусное сходство**:

$$\text{cosine\_similarity}(v_1, v_2) = \frac{v_1 \cdot v_2}{||v_1|| \cdot ||v_2||}$$

где:
- $v_1 \cdot v_2$ — скалярное произведение векторов
- $||v_1||, ||v_2||$ — длины векторов (нормы)

**Значение:** от -1 до 1, где 1 означает идентичность, 0 — независимость, -1 — противоположность.

### Контекстная гипотеза

Основной принцип: **слова, используемые в похожем контексте, имеют похожее значение**.

Пример:
- "Кот сидит на диване."
- "Собака лежит на диване."

Слова "кот" и "собака" часто появляются в похожем контексте, поэтому их векторы должны быть близки.


## Векторные модели слов

### 1. Word2Vec

**Word2Vec** — это нейросетевая модель, которая обучается на большом текстовом корпусе и создает векторные представления слов.

#### Skip-gram архитектура

Skip-gram предсказывает контекстные слова по целевому слову:

```
Входное слово: "король"
Контекстные слова: ["большой", "дворец", "страна", ...]
```

**Процесс обучения:**

1. Берем центральное слово
2. Предсказываем окружающие слова (в окне контекста размером 2c)
3. Корректируем веса нейросети, чтобы улучшить предсказание

#### CBOW архитектура

Continuous Bag of Words делает противоположное — предсказывает целевое слово по его контексту:

```
Контекстные слова: ["большой", "дворец", "страна", ...]
Предсказываемое слово: "король"
```

**Преимущества Word2Vec:**
- Быстрое обучение
- Хорошее качество на больших корпусах
- Простота использования (готовые предобученные модели)

**Недостатки:**
- Один вектор для одного слова (не учитывает многозначность)
- Требует большой корпус для обучения

**Пример связей в Word2Vec:**
```
vector("король") - vector("мужчина") + vector("женщина") ≈ vector("королева")
```

### 2. GloVe (Global Vectors)

**GloVe** объединяет два подхода: глобальную статистику слов и локальный контекст.

#### Основная идея

Вместо предсказания, GloVe минимизирует функцию потерь на основе матрицы совместных вхождений:

$$J = \sum_{i,j=1}^{V} f(X_{ij}) (w_i^T \tilde{w}_j + b_i + \tilde{b}_j - \log X_{ij})^2$$

где:
- $X_{ij}$ — количество раз, когда слова $i$ и $j$ появляются в одном контексте
- $w_i, \tilde{w}_j$ — векторы слов
- $f(X_{ij})$ — весовая функция

**Преимущества GloVe:**
- Использует глобальную статистику корпуса
- Обучается быстрее, чем Word2Vec
- Хорошее качество на меньших корпусах

**Недостатки:**
- Требует построения матрицы совместных вхождений (память-интенсивно)
- Менее гибкая, чем нейросетевые подходы

### 3. FastText

**FastText** расширяет Word2Vec, работая с подсловами (n-граммами символов) вместо целых слов.

#### Ключевое отличие

Вместо одного вектора для слова, каждое слово представляется суммой векторов его подслов:

```
word = "котенок"
subwords = ["<к", "ко", "от", "те", "ен", "но", "ок", "к>"]
vector("котенок") = sum(vector(subword) для каждого subword)
```

**Преимущества FastText:**
- Работает с редкими и неизвестными словами
- Лучше справляется со словами, близкими по написанию
- Хорошо для морфологически богатых языков (как русский)

**Недостатки:**
- Медленнее, чем Word2Vec при использовании
- Требует больше памяти для хранения подсловных векторов

### Решение Упражнения 2: Матрица совместных вхождений

```python
import numpy as np
import pandas as pd

text = "король живет в большом дворце. королева живет с королем в дворце."

# Токенизация (удаляем пунктуацию и переводим в нижний регистр)
tokens = text.lower().replace('.', '').split()
vocabulary = sorted(set(tokens))

print(f"Словарь: {vocabulary}")
print(f"Размер словаря: {len(vocabulary)}")

# Инициализация матрицы совместных вхождений
cooc_matrix = np.zeros((len(vocabulary), len(vocabulary)))

# Размер окна контекста
window_size = 2

# Построение матрицы
for i, token in enumerate(tokens):
    # Левый контекст
    for j in range(max(0, i - window_size), i):
        context_token = tokens[j]
        token_idx = vocabulary.index(token)
        context_idx = vocabulary.index(context_token)
        cooc_matrix[token_idx][context_idx] += 1
    
    # Правый контекст
    for j in range(i + 1, min(len(tokens), i + window_size + 1)):
        context_token = tokens[j]
        token_idx = vocabulary.index(token)
        context_idx = vocabulary.index(context_token)
        cooc_matrix[token_idx][context_idx] += 1

# Визуализация матрицы
df = pd.DataFrame(cooc_matrix, index=vocabulary, columns=vocabulary)
print("\n=== Матрица совместных вхождений ===")
print(df)

# Найдите пару слов с максимальным числом совместных вхождений
print("\n=== Анализ ===")
max_value = 0
max_pair = (None, None)

for i in range(len(vocabulary)):
    for j in range(len(vocabulary)):
        if cooc_matrix[i][j] > max_value:
            max_value = cooc_matrix[i][j]
            max_pair = (vocabulary[i], vocabulary[j])

print(f"Пара с наибольшим числом совместных вхождений: {max_pair} ({int(max_value)} раз)")

# Какие слова чаще встречаются друг с другом?
print("\n=== Совместные вхождения (> 0) ===")
for i in range(len(vocabulary)):
    for j in range(len(vocabulary)):
        if cooc_matrix[i][j] > 0:
            print(f"{vocabulary[i]:15} - {vocabulary[j]:15}: {int(cooc_matrix[i][j])}")
```


### Решение Упражнения 3: Косинусное сходство

```python
import numpy as np
from scipy.spatial.distance import cosine

v1 = np.array([0.5, 0.3, 0.8, 0.1])  # "красивый"
v2 = np.array([0.6, 0.2, 0.7, 0.2])  # "привлекательный"

print("=== Вычисление косинусного сходства вручную ===")
print(f"v1 = {v1}")
print(f"v2 = {v2}")

# Задание 1: Скалярное произведение
dot_product = np.dot(v1, v2)
print(f"\n1. Скалярное произведение (v1 · v2):")
print(f"   Расчет: {v1[0]}*{v2[0]} + {v1[1]}*{v2[1]} + {v1[2]}*{v2[2]} + {v1[3]}*{v2[3]}")
print(f"   = {v1[0]*v2[0]:.2f} + {v1[1]*v2[1]:.2f} + {v1[2]*v2[2]:.2f} + {v1[3]*v2[3]:.2f}")
print(f"   = {dot_product:.4f}")

# Задание 2: Нормы векторов
norm_v1 = np.linalg.norm(v1)
norm_v2 = np.linalg.norm(v2)
print(f"\n2. Нормы векторов:")
print(f"   ||v1|| = √({v1[0]}² + {v1[1]}² + {v1[2]}² + {v1[3]}²)")
print(f"   ||v1|| = √{sum(v1**2):.4f} = {norm_v1:.4f}")
print(f"   ||v2|| = √({v2[0]}² + {v2[1]}² + {v2[2]}² + {v2[3]}²)")
print(f"   ||v2|| = √{sum(v2**2):.4f} = {norm_v2:.4f}")

# Задание 3: Косинусное сходство
cosine_sim = dot_product / (norm_v1 * norm_v2)
print(f"\n3. Косинусное сходство:")
print(f"   cos(v1, v2) = (v1 · v2) / (||v1|| × ||v2||)")
print(f"   cos(v1, v2) = {dot_product:.4f} / ({norm_v1:.4f} × {norm_v2:.4f})")
print(f"   cos(v1, v2) = {dot_product:.4f} / {norm_v1 * norm_v2:.4f}")
print(f"   cos(v1, v2) = {cosine_sim:.4f}")

# Задание 4: Интерпретация
print(f"\n4. Интерпретация:")
print(f"   Значение косинусного сходства: {cosine_sim:.4f}")
if cosine_sim > 0.8:
    print(f"   Слова очень близки по значению (высокое сходство)")
elif cosine_sim > 0.5:
    print(f"   Слова довольно близки по значению")
elif cosine_sim > 0.2:
    print(f"   Слова имеют некоторое сходство")
else:
    print(f"   Слова слабо связаны по значению")

# Проверка с использованием готовой функции
print(f"\n=== Проверка результата ===")
cosine_sim_scipy = 1 - cosine(v1, v2)
print(f"Результат (scipy): {cosine_sim_scipy:.4f}")
print(f"Совпадение: {np.isclose(cosine_sim, cosine_sim_scipy)}")
```


### Решение Упражнения 4: Усредненные векторы предложений

```python
from gensim.models import KeyedVectors
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

sentences = [
    "кот спит на диване",
    "кошка отдыхает на софе",
    "математика это наука о числах"
]

print("=== Анализ предложений с помощью усредненных векторов ===")

# Загрузка модели (используем русскую модель)
try:
    model = KeyedVectors.load_word2vec_format('cc.ru.300.vec')
except FileNotFoundError:
    print("Скачайте модель fastText для русского языка:")
    print("https://fasttext.cc/docs/en/crawl-vectors.html")
    exit()

# Функция для получения вектора предложения
def get_sentence_vector(sentence, model):
    """
    Вычисляет среднее арифметическое векторов слов в предложении.
    Пропускает слова, которые не в словаре модели.
    """
    words = sentence.split()
    vectors = []
    
    for word in words:
        try:
            vectors.append(model[word])
        except KeyError:
            # Слово не в словаре — пропускаем
            pass
    
    if len(vectors) == 0:
        # Если все слова неизвестны, возвращаем нулевой вектор
        return np.zeros(model.vector_size)
    
    return np.mean(vectors, axis=0)

# Задание 1 & 2: Получение векторов для каждого слова и усредненного вектора
print("\nЗадание 1 & 2: Получение векторов")
print("-" * 60)

sentence_vectors = []

for sentence in sentences:
    print(f"\nПредложение: '{sentence}'")
    
    words = sentence.split()
    print(f"Слова: {words}")
    
    # Получение векторов отдельных слов
    word_vectors = []
    for word in words:
        try:
            vec = model[word]
            word_vectors.append(vec)
            print(f"  '{word}': размерность вектора = {len(vec)}")
        except KeyError:
            print(f"  '{word}': НЕ НАЙДЕНО в словаре")
    
    # Вычисление среднего вектора
    sentence_vec = get_sentence_vector(sentence, model)
    sentence_vectors.append(sentence_vec)
    print(f"Средний вектор предложения: размерность = {len(sentence_vec)}")

# Задание 3: Косинусное сходство между предложениями
print("\n" + "=" * 60)
print("Задание 3: Косинусное сходство между предложениями")
print("=" * 60)

# Преобразование в правильный формат
vectors_array = np.array(sentence_vectors)

# Вычисление матрицы косинусного сходства
similarity_matrix = cosine_similarity(vectors_array)

print("\nМатрица косинусного сходства:")
print("-" * 60)

for i, sent1 in enumerate(sentences):
    for j, sent2 in enumerate(sentences):
        if i < j:
            sim = similarity_matrix[i][j]
            print(f"Предложение {i+1} и Предложение {j+1}: {sim:.4f}")
            print(f"  '{sent1}'")
            print(f"  '{sent2}'")
            print()

# Задание 4: Найдите наиболее похожие предложения
print("=" * 60)
print("Задание 4: Анализ результатов")
print("=" * 60)

max_similarity = -1
most_similar_pair = (0, 1)

for i in range(len(sentences)):
    for j in range(i + 1, len(sentences)):
        if similarity_matrix[i][j] > max_similarity:
            max_similarity = similarity_matrix[i][j]
            most_similar_pair = (i, j)

i, j = most_similar_pair
print(f"\nНаиболее похожие предложения (сходство: {max_similarity:.4f}):")
print(f"  1. {sentences[i]}")
print(f"  2. {sentences[j]}")

print(f"\nПочему они похожи?")
print(f"  - Оба описывают животное в доме")
print(f"  - Используют семантически близкие слова:")
print(f"    'кот' ~ 'кошка' (близкие слова по смыслу)")
print(f"    'диван' ~ 'софе' (синонимы - одна мебель)")
print(f"    'спит' ~ 'отдыхает' (синонимы - похожее действие)")

print(f"\nСравнение с третьим предложением:")
print(f"  - Третье предложение о математике - это совсем другая тема")
print(f"  - Лексика полностью отличается")
```


### Решение Упражнения 5: Кластеризация слов

```python
from gensim.models import KeyedVectors
from scipy.cluster.hierarchy import dendrogram, linkage
from scipy.spatial.distance import pdist, squareform
import matplotlib.pyplot as plt
import numpy as np

words = [
    "король", "королева", "принц", "принцесса",
    "врач", "медсестра", "пациент",
    "красивый", "прекрасный", "изящный",
    "быстро", "стремительно", "медленно"
]

print("=== Кластеризация слов по семантической близости ===\n")

# Загрузка модели
try:
    model = KeyedVectors.load_word2vec_format('cc.ru.300.vec')
except FileNotFoundError:
    print("Скачайте модель fastText для русского языка:")
    print("https://fasttext.cc/docs/en/crawl-vectors.html")
    exit()

# Задание 1: Получение векторов для всех слов

for word in words:
    try:
        vector = model[word]
        word_vectors.append(vector)
        found_words.append(word)
        print(f"✓ '{word}' найдено")
    except KeyError:
        print(f"✗ '{word}' НЕ НАЙДЕНО")

plt.xlabel('Слова', fontsize=12)
plt.tight_layout()

# Сохранение дендрограммы
```

## Справочник терминов

| Термин                                                  | Определение                                                                                        |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| **Вектор слова (Word embedding)**                       | Числовое представление слова в виде последовательности чисел фиксированной длины                   |
| **Семантика**                                           | Смысл и значение языковых единиц                                                                   |
| **Контекст**                                            | Окружающие слова, которые помогают определить значение целевого слова                              |
| **Косинусное сходство (Cosine similarity)**             | Мера сходства между двумя векторами, основанная на косинусе угла между ними                        |
| **Скалярное произведение (Dot product)**                | Сумма произведений соответствующих элементов двух векторов                                         |
| **Норма вектора (Vector norm)**                         | Длина вектора, обычно вычисляется как L2 норма                                                     |
| **Word2Vec**                                            | Нейросетевая модель для обучения векторных представлений слов (Skip-gram и CBOW)                   |
| **Skip-gram**                                           | Архитектура Word2Vec, которая предсказывает контекстные слова по целевому слову                    |
| **CBOW (Continuous Bag of Words)**                      | Архитектура Word2Vec, которая предсказывает целевое слово по его контексту                         |
| **GloVe (Global Vectors)**                              | Метод векторизации, объединяющий глобальную статистику и локальный контекст                        |
| **Матрица совместных вхождений (Co-occurrence matrix)** | Матрица, в которой элемент (i,j) показывает, сколько раз слово i появляется с словом j в контексте |
| **FastText**                                            | Расширение Word2Vec, работающее с подсловами (n-граммами символов)                                 |
| **Подслово (Subword)**                                  | N-грамма символов, из которых состоит слово                                                        |
| **Окно контекста (Context window)**                     | Размер окружения (в словах) вокруг целевого слова                                                  |
| **Предобученная модель (Pre-trained model)**            | Модель, которая была обучена на большом корпусе и готова к использованию                           |
| **Корпус (Corpus)**                                     | Большой набор текстов для обучения моделей                                                         |
| **Многозначность (Polysemy)**                           | Наличие у одного слова нескольких значений                                                         |
| **Синоним**                                             | Слово с похожим или одинаковым значением                                                           |
| **Дендрограмма**                                        | Древовидная диаграмма для визуализации иерархической кластеризации                                 |
| **Евклидово расстояние**                                | Расстояние между двумя точками в многомерном пространстве (прямая линия)                           |
