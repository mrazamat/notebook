# Стилометрия: Статистические и лингвистические методы определения стиля текста

## Введение

Стилометрия — это наука о количественном анализе стиля письма. Она позволяет определять авторство текстов, классифицировать документы по жанрам, выявлять закономерности в использовании языка и автоматически находить плагиат. Стилометрия активно применяется в литературоведении, криминалистике, лингвистике и обработке естественного языка (NLP).

### Основная идея

Каждый автор имеет уникальные привычки в использовании языка — как осознанные, так и подсознательные. Эти привычки формируют **лингвистический отпечаток** (stylistic fingerprint), который можно выявить и анализировать с помощью статистических методов.

## Основные концепции стилометрии

### 1. Признаки стиля (stylometric features)

Признаки стиля — это измеримые характеристики текста, которые отражают стилистические особенности автора. Они делятся на несколько категорий:

#### Поверхностные признаки (Surface features)
- **Длина слов**: среднее количество букв в слове
- **Длина предложений**: среднее количество слов в предложении
- **Длина абзацев**: среднее количество предложений в абзаце
- **Длина текста**: общее количество слов, символов или предложений

#### Лексические признаки (Lexical features)
- **Словарный запас**: количество уникальных слов (type-token ratio)
- **Частотность слов**: распределение наиболее частых слов
- **Редкие слова**: доля слов, встречающихся только один раз
- **Части речи**: процент глаголов, прилагательных, предлогов и т.д.

#### Синтаксические признаки (Syntactic features)
- **Пунктуация**: частота использования определённых знаков препинания
- **Структура предложений**: распределение простых и сложных предложений
- **Использование союзов**: частота связывающих слов
- **Порядок слов**: синтаксические паттерны

#### Специфичные для автора признаки (Idiolectal features)
- **Личные слова**: частота использования местоимений (я, мы, он, она)
- **Функциональные слова**: предлоги (в, на, из, к), союзы (и, но, или)
- **Словосочетания**: n-граммы, которые часто используются автором

### 2. Функциональные слова (Function words)

Функциональные слова — это служебные слова, которые несут грамматическую функцию и выступают как стабильные маркеры стиля:

**Примеры функциональных слов на русском языке:**
- Предлоги: в, на, из, к, по, с, у, про, без, для
- Союзы: и, но, или, потому что, хотя, если, что, чтобы
- Местоимения: я, ты, он, она, оно, мы, вы, они, что, кто, который
- Вспомогательные глаголы: быть, есть, иметь

**Почему они важны?**

Авторы редко осознанно выбирают функциональные слова, поэтому они отражают глубокие стилистические привычки. Плагиаторы часто замещают смысловые слова синонимами, но функциональные слова сохраняют своё употребление.

## Статистические методы анализа

### 1. Частотный анализ (Frequency analysis)

Частотный анализ заключается в подсчёте частоты появления определённых элементов (слова, буквы, n-граммы) и создании их распределения.

**Основная формула:**

$$f(w) = \frac{\text{количество появлений слова } w}{\text{общее количество слов в тексте}}$$

**Пример на русском:**
```
Текст: "Любить читать. Читать интересно. Все любят читать."
- "читать" встречается 3 раза из 8 слов → f(читать) = 3/8 = 0.375
- "любить" встречается 2 раза → f(любить) = 2/8 = 0.25
```

### 2. N-граммы (N-grams)

N-грамма — это последовательность из n элементов (слов, букв, символов). N-граммы отражают локальные паттерны в тексте и помогают выявить авторские привычки.

**Виды n-грамм:**

| Тип | Пример | Описание |
|-----|--------|---------|
| Unigramm (1-gram) | "и", "что", "но" | Отдельные слова |
| Bigram (2-gram) | "и что", "но если" | Пары слов |
| Trigram (3-gram) | "что если бы", "и то что" | Тройки слов |

**Пример на русском:**
```
Текст: "Я люблю читать книги"
Unigramm: "Я", "люблю", "читать", "книги"
Bigram: "Я люблю", "люблю читать", "читать книги"
Trigram: "Я люблю читать", "люблю читать книги"
```

### 3. Type-Token Ratio (TTR)

TTR измеряет разнообразие словарного запаса автора.

$$TTR = \frac{\text{количество уникальных слов (types)}}{\text{общее количество слов (tokens)}}$$

**Интерпретация:**
- **Высокий TTR (0.7-1.0)**: богатый словарный запас, разнообразный стиль
- **Средний TTR (0.4-0.7)**: нормальный словарный запас
- **Низкий TTR (0.1-0.4)**: ограниченный словарный запас, повторяющийся стиль

**Пример:**
```
Текст: "Кот ловит мышь. Мышь бежит от кота."
Types (уникальные слова): кот, ловит, мышь, бежит, от (5 слов)
Tokens (общее количество слов): 8 слов
TTR = 5/8 = 0.625 (хороший уровень разнообразия)
```

### 4. Энтропия Шеннона (Shannon Entropy)

Энтропия измеряет неопределённость или разнообразие распределения элементов текста. Высокая энтропия указывает на более разнообразный текст.

$$H = -\sum_{i=1}^{n} p_i \log_2(p_i)$$

где $p_i$ — вероятность (относительная частота) элемента $i$.

**Интерпретация:**
- Максимальная энтропия: все элементы равновероятны
- Минимальная энтропия: только один элемент используется

## Методы определения авторства

### 1. Дельта-метод Бурроуза (Burrows' Delta)

Дельта-метод определяет, насколько текст похож на текст известного автора, путём сравнения z-оценок (z-scores) частот функциональных слов.

**Шаги алгоритма:**

1. Выбрать 100-200 наиболее частых слов во всех текстах
2. Для каждого текста вычислить z-оценку для каждого слова:
   $$z_i = \frac{f_i - \mu}{\sigma}$$
   где $f_i$ — частота слова i, $\mu$ — средняя частота, $\sigma$ — стандартное отклонение

3. Вычислить разницу z-оценок между текстом неизвестного авторства и текстами известных авторов
4. Найти минимальную разницу — это предполагаемый автор

**Пример вычисления:**
```
Слово "и": частота = 0.12
Средняя частота по корпусу = 0.10
Стандартное отклонение = 0.02
z-оценка = (0.12 - 0.10) / 0.02 = 1.0
```

### 2. Машинное обучение (Machine Learning)

Для определения авторства можно использовать классические алгоритмы машинного обучения.

**Основные этапы:**

1. **Извлечение признаков**: для каждого текста создать вектор признаков (длина слов, TTR, частоты функциональных слов и т.д.)
2. **Обучение классификатора**: использовать алгоритмы (Naive Bayes, SVM, Random Forest) на текстах с известным авторством
3. **Классификация**: предсказать авторство неизвестного текста

**Пример подхода:**
```
Текст автора А: [5.2, 0.65, 0.14, 0.08, ...] → Класс A
Текст автора Б: [6.1, 0.58, 0.11, 0.12, ...] → Класс B
Неизвестный текст: [5.1, 0.64, 0.15, 0.09, ...] → Класс A (похож на А)
```

## Практические примеры на Python

### Пример 1: Базовые статистики текста

```python
import re
from collections import Counter
import math

def analyze_text(text):
    """
    Анализирует текст и возвращает основные статистики.
    """
    # Приведение текста к нижнему регистру и разбиение на слова
    words = re.findall(r'\b\w+\b', text.lower())
    
    # Базовые метрики
    total_words = len(words)
    unique_words = len(set(words))
    sentences = len(re.split(r'[.!?]+', text))
    
    # Подсчёт букв
    letters = sum(1 for c in text if c.isalpha())
    
    # Вычисление статистик
    avg_word_length = letters / total_words if total_words > 0 else 0
    avg_sentence_length = total_words / sentences if sentences > 0 else 0
    ttr = unique_words / total_words if total_words > 0 else 0
    
    return {
        'total_words': total_words,
        'unique_words': unique_words,
        'avg_word_length': round(avg_word_length, 2),
        'avg_sentence_length': round(avg_sentence_length, 2),
        'ttr': round(ttr, 3),
        'sentences': sentences
    }

# Пример использования
text = "Люблю читать книги. Книги открывают новые миры. Чтение развивает ум."
stats = analyze_text(text)
print(stats)
# Результат:
# {
#   'total_words': 12,
#   'unique_words': 9,
#   'avg_word_length': 5.33,
#   'avg_sentence_length': 4.0,
#   'ttr': 0.75,
#   'sentences': 3
# }
```

### Пример 2: Частотный анализ и n-граммы

```python
def extract_ngrams(text, n=2):
    """
    Извлекает n-граммы из текста.
    """
    words = re.findall(r'\b\w+\b', text.lower())
    ngrams = [tuple(words[i:i+n]) for i in range(len(words) - n + 1)]
    return Counter(ngrams)

def get_word_frequencies(text, top_n=10):
    """
    Возвращает частоты наиболее частых слов.
    """
    words = re.findall(r'\b\w+\b', text.lower())
    freq = Counter(words)
    return freq.most_common(top_n)

# Пример использования
text = "Кот ловит мышь. Мышь бежит. Кот быстрый. Мышь маленькая."

# N-граммы
bigrams = extract_ngrams(text, n=2)
print("Bigrams:", bigrams.most_common(5))
# Результат: [(('кот',), ('ловит',)), (('мышь',), ('бежит',)), ...]

# Частоты слов
freq = get_word_frequencies(text, top_n=5)
print("Top words:", freq)
# Результат: [('кот', 2), ('мышь', 2), ('ловит', 1), ('бежит', 1), ('быстрый', 1)]
```

### Пример 3: Вычисление энтропии Шеннона

```python
def calculate_entropy(text):
    """
    Вычисляет энтропию Шеннона для распределения слов.
    """
    words = re.findall(r'\b\w+\b', text.lower())
    freq = Counter(words)
    total = sum(freq.values())
    
    entropy = 0
    for count in freq.values():
        probability = count / total
        entropy -= probability * math.log2(probability)
    
    return round(entropy, 3)

# Пример использования
text1 = "и и и и и"  # Низкая энтропия (одно слово)
text2 = "кот мышь книга дом город" # Высокая энтропия (разные слова)

print(f"Энтропия текста 1: {calculate_entropy(text1)}")  # 0.0
print(f"Энтропия текста 2: {calculate_entropy(text2)}")  # ~2.32
```

### Пример 4: Type-Token Ratio

```python
def calculate_ttr(text):
    """
    Вычисляет Type-Token Ratio.
    """
    words = re.findall(r'\b\w+\b', text.lower())
    unique_words = len(set(words))
    total_words = len(words)
    
    ttr = unique_words / total_words if total_words > 0 else 0
    return round(ttr, 3)

# Пример использования
text_low = "Кот кот кот мышь мышь"  # TTR = 0.4
text_high = "Кот прыгает быстро вверх вниз"  # TTR = 1.0

print(f"TTR (низкий): {calculate_ttr(text_low)}")
print(f"TTR (высокий): {calculate_ttr(text_high)}")
```

## Применение стилометрии

### 1. Определение авторства
- Выявление истинного автора текста (анонимные произведения, письма)
- Проверка работ студентов на оригинальность
- Криминалистический анализ текстов (записки, письма)

### 2. Выявление плагиата
- Сравнение подозрительного текста с известными источниками
- Выявление скрытого переписывания (paraphrasing)
- Обнаружение текстов, написанных с использованием ИИ

### 3. Классификация по жанрам
- Определение жанра текста (научный, художественный, публицистический)
- Классификация литературных стилей (барокко, классицизм, романтизм)
- Категоризация по типам (новости, блоги, статьи)

### 4. Психолингвистический анализ
- Выявление психического состояния автора
- Определение уровня образования и возраста
- Анализ личностных характеристик

## Вызовы и ограничения

### Проблемы при анализе
- **Коротких текстов**: меньше данных для анализа
- **Многоязычных текстов**: разные языки имеют разные паттерны
- **Намеренной подделки**: опытный подделыватель может имитировать стиль
- **Изменения стиля**: авторы развиваются и меняют свой стиль со временем
- **Жанровых различий**: один автор может писать в разных стилях

### Технические сложности
- Выбор релевантных признаков
- Переобучение (overfitting) моделей на малых датасетах
- Обработка морфологии и синтаксиса русского языка
- Требование большого корпуса для обучения
